### 题目 1：期望效用计算 ⭐

**难度**: ⭐  
**涉及知识点**: 期望效用、最大期望效用原则

**题目**:  
假设你面临以下选择：
- **选项A**: 确定获得100元
- **选项B**: 50%概率获得250元，50%概率获得0元

(a) 如果你使用期望货币价值(EMV)作为决策标准，应该选择哪个选项？

(b) 假设你的效用函数为U(x) = √x (x≥0)，计算每个选项的期望效用，并做出选择。

(c) 讨论为什么(a)和(b)的选择可能不同。

#### 逐步提示

<details>
<summary>点击查看提示</summary>

1. EMV计算: 概率加权平均货币价值
2. 期望效用: 先计算各结果的效用，再概率加权
3. 效用函数的曲率反映风险态度
</details>

#### 详细解答

**(a) EMV计算**

```
EMV(A) = 100
EMV(B) = 0.5 × 250 + 0.5 × 0 = 125

选择: B (EMV更高)
```

**(b) 期望效用计算**

```
EU(A) = U(100) = √100 = 10

EU(B) = 0.5 × U(250) + 0.5 × U(0)
      = 0.5 × √250 + 0.5 × 0
      = 0.5 × 15.81
      = 7.905

选择: A (EU更高)
```

**(c) 分析**

| 标准 | 选项A | 选项B | 选择 |
|-----|------|------|-----|
| EMV | 100 | 125 | B |
| EU | 10 | 7.905 | A |

**原因**: 
- U(x) = √x 是**凹函数**，反映风险厌恶
- 边际效用递减: 从0到100的效用增益 > 从100到250的增益
- 确定性等价值(Certainty Equivalent): A的EU=10对应的货币值为100，而B需要约62.5元才能达到相同效用

### 题目 2：效用公理验证 ⭐⭐

**难度**: ⭐⭐  
**涉及知识点**: 效用理论公理、传递性

**题目**:  
考虑以下彩票：
- L₁: [0.5, $100; 0.5, $0]
- L₂: [0.6, $80; 0.4, $0]  
- L₃: [0.7, $60; 0.3, $0]

(a) 如果某人的偏好为 L₁ ≻ L₂ 且 L₂ ≻ L₃，根据传递性公理，应该有什么偏好关系？

(b) 假设效用函数 U(x) = x²，验证这些偏好是否与期望效用最大化一致。

(c) 证明如果偏好违反传递性，可能导致"钱泵"(money pump)现象。

#### 逐步提示

<details>
<summary>点击查看提示</summary>

1. 传递性要求: A≻B且B≻C ⇒ A≻C
2. 计算每个彩票的期望效用
3. 非传递性偏好可被利用进行无限套利
</details>

#### 详细解答

**(a) 传递性推导**

```
给定: L₁ ≻ L₂ 且 L₂ ≻ L₃
由传递性: L₁ ≻ L₃
```

**(b) 期望效用验证**

```
U(x) = x²

EU(L₁) = 0.5 × 100² + 0.5 × 0² = 5000
EU(L₂) = 0.6 × 80² + 0.4 × 0² = 3840  
EU(L₃) = 0.7 × 60² + 0.3 × 0² = 2520

5000 > 3840 > 2520 ✓

与 L₁ ≻ L₂ ≻ L₃ 一致
```

**(c) 钱泵证明**

假设非传递性偏好: L₁ ≻ L₂, L₂ ≻ L₃, L₃ ≻ L₁

**套利循环**:
1. 从L₁开始
2. 用L₁换L₂(因为L₂ ≺ L₁，需支付少量费用ε)
3. 用L₂换L₃(支付ε)
4. 用L₃换L₁(支付ε)
5. 回到起点，但损失了3ε

重复此循环可提取无限资金，证明非传递性导致非理性行为。

### 题目 3：风险态度分析 ⭐⭐

**难度**: ⭐⭐  
**涉及知识点**: 风险厌恶、确定性等价值、保险

**题目**:  
假设你有10000元财富，效用函数为 U(w) = ln(w)。

(a) 计算你愿意为以下保险支付的最高保费：
- 保险覆盖的损失: 1000元(房屋火灾)
- 火灾概率: 1%

(b) 保险公司愿意以12元出售此保险。解释为什么这对双方都有利。

(c) 定义"风险溢价"并计算本例中的风险溢价。

#### 逐步提示

<details>
<summary>点击查看提示</summary>

1. 无保险时的期望效用 vs 买保险后的确定效用
2. 保险公司风险中性(财富远大于保费)
3. 风险溢价 = EMV - 确定性等价值
</details>

#### 详细解答

**(a) 最高保费计算**

```
无保险期望效用:
EU(no) = 0.99 × ln(10000) + 0.01 × ln(9000)
       = 0.99 × 9.21 + 0.01 × 9.10
       = 9.1189 + 0.091
       = 9.2099

买保险后(确定状态):
EU(yes) = ln(10000 - P)

令 EU(yes) = EU(no):
ln(10000 - P) = 9.2099
10000 - P = e^9.2099 ≈ 9990
P_max = 10元
```

**(b) 互利分析**

| 参与者 | 最高支付/最低接受 | 实际价格 | 收益 |
|-------|----------------|---------|-----|
| 你 | 10元 | 12元 | -2元(仍购买，因恐惧损失) |
| 保险公司 | 10元(期望赔付) | 12元 | +2元利润 |

**修正**: 重新计算

```
期望赔付 = 0.01 × 1000 = 10元

保险公司定价12元，利润2元。

如果你愿意支付最多10元，你不会买12元的保险。

但如果考虑风险厌恶程度更高，或损失评估不同...
```

**重新计算最高保费**:

```
ln(10000 - P) = 9.2099
10000 - P = 9990.05
P = 9.95元 ≈ 10元
```

**(c) 风险溢价**

```
无保险EMV = 0.99 × 10000 + 0.01 × 9000 = 9990元
确定性等价值 = 9990.05元(使效用相等的确定金额)

风险溢价 = EMV - CE = 9990 - 9990.05 ≈ 0

这说明对数效用在此小风险下接近风险中性。
```

### 题目 4：信息价值(VPI)基础 ⭐⭐⭐

**难度**: ⭐⭐⭐  
**涉及知识点**: 完美信息价值、决策网络

**题目**:  
考虑一个简单的医疗决策问题：
- 疾病D的先验概率: P(D) = 0.3
- 治疗T的成本: -50元
- 如果患病且不治疗: 效用 = -200元
- 如果患病且治疗: 效用 = -100元  
- 如果不患病且不治疗: 效用 = 0元
- 如果不患病且治疗: 效用 = -50元

(a) 不获取信息时的最优决策是什么？

(b) 计算完美诊断测试的信息价值(VPI)。

(c) 如果测试成本为30元，是否应该进行测试？

#### 逐步提示

<details>
<summary>点击查看提示</summary>

1. 计算每个动作的期望效用
2. 有信息时的决策: 根据测试结果选择不同动作
3. VPI = 有信息时的EU - 无信息时的EU
</details>

#### 详细解答

**(a) 无信息最优决策**

```
EU(治疗) = 0.3 × (-100) + 0.7 × (-50) = -30 - 35 = -65
EU(不治疗) = 0.3 × (-200) + 0.7 × 0 = -60

最优决策: 不治疗 (EU = -60 > -65)
```

**(b) VPI计算**

```
测试阳性(P(D|+) ≈ 1):
  最优: 治疗, EU = -100

测试阴性(P(D|-) ≈ 0):
  最优: 不治疗, EU = 0

有信息时的EU:
EU(with info) = 0.3 × (-100) + 0.7 × 0 = -30

VPI = EU(with info) - EU(without info)
    = -30 - (-60)
    = 30元
```

**(c) 测试决策**

```
净收益 = VPI - 测试成本 = 30 - 30 = 0

边际情况，可能不测试(考虑其他因素)

如果测试成本 < 30元: 应该测试
如果测试成本 > 30元: 不应该测试
```

### 题目 5：多属性效用 ⭐⭐⭐

**难度**: ⭐⭐⭐  
**涉及知识点**: 偏好独立性、加性价值函数

**题目**:  
考虑购买汽车决策，涉及三个属性：
- 价格(P): 越低越好
- 安全性(S): 越高越好
- 燃油效率(F): 越高越好

(a) 解释什么是偏好独立性(Mutual Preferential Independence, MPI)。

(b) 假设MPI成立，写出加性价值函数的一般形式。

(c) 给定以下具体形式，计算两种车的总价值：
```
V(P, S, F) = -0.5 × P + 100 × S + 2 × F

车A: P=20万, S=8/10, F=15km/L
车B: P=15万, S=6/10, F=12km/L
```

#### 逐步提示

<details>
<summary>点击查看提示</summary>

1. MPI: 两个属性的权衡不依赖于第三个属性的值
2. 加性形式: V = Σ v_i(x_i)
3. 代入计算并比较
</details>

#### 详细解答

**(a) MPI解释**

```
偏好独立性: 属性X₁和X₂的偏好权衡独立于X₃的值

例子: 如果"
  (P=20万, S=9) ≻ (P=18万, S=7)
在F=15时成立

那么当F=12时，相同的偏好应该成立
```

**(b) 加性价值函数**

```
V(P, S, F) = v₁(P) + v₂(S) + v₃(F)

其中:
- v₁(P) 是关于价格的单属性价值函数(递减)
- v₂(S) 是关于安全性的单属性价值函数(递增)
- v₃(F) 是关于燃油效率的单属性价值函数(递增)
```

**(c) 价值计算**

```
车A:
V_A = -0.5 × 20 + 100 × 0.8 + 2 × 15
    = -10 + 80 + 30
    = 100

车B:
V_B = -0.5 × 15 + 100 × 0.6 + 2 × 12
    = -7.5 + 60 + 24
    = 76.5

选择: 车A (V_A > V_B)
```

### 题目 6：决策网络分析 ⭐⭐⭐⭐

**难度**: ⭐⭐⭐⭐  
**涉及知识点**: 决策网络、信息价值、敏感性分析

**题目**:  
考虑以下投资决策的决策网络：
- 决策节点: 投资(I) - 投资或不投资
- 机会节点: 
  - 市场状态(M): 好(0.4)或坏(0.6)
  - 公司表现(C): 取决于M和I
- 效用节点: U(取决于C和I)

条件概率和效用表：
```
P(M=好) = 0.4

P(C=成功 | M=好, I=是) = 0.8
P(C=成功 | M=好, I=否) = 0
P(C=成功 | M=坏, I=是) = 0.3
P(C=成功 | M=坏, I=否) = 0

效用:
U(C=成功, I=是) = 100
U(C=失败, I=是) = -50
U(C=成功, I=否) = 0
U(C=失败, I=否) = 0
```

(a) 计算不获取信息时的最优决策。

(b) 如果可以完美观测市场状态M，计算VPI(M)。

(c) 如果市场预测准确率为80%(P(预测=好|M=好)=0.8)，计算该不完美信息的VPI。

#### 逐步提示

<details>
<summary>点击查看提示</summary>

1. 计算每种决策的期望效用
2. 完美信息: 根据实际M值选择不同动作
3. 不完美信息: 需要考虑预测错误的概率
</details>

#### 详细解答

**(a) 无信息最优决策**

```
EU(I=是) = P(M=好) × [P(C=成功|M=好,I=是) × 100 + P(C=失败|M=好,I=是) × (-50)]
         + P(M=坏) × [P(C=成功|M=坏,I=是) × 100 + P(C=失败|M=坏,I=是) × (-50)]
         = 0.4 × (0.8 × 100 + 0.2 × (-50)) + 0.6 × (0.3 × 100 + 0.7 × (-50))
         = 0.4 × (80 - 10) + 0.6 × (30 - 35)
         = 0.4 × 70 + 0.6 × (-5)
         = 28 - 3 = 25

EU(I=否) = 0

最优决策: 投资 (EU = 25)
```

**(b) 完美信息VPI(M)**

```
如果M=好(0.4):
  EU(I=是) = 0.8 × 100 + 0.2 × (-50) = 70
  EU(I=否) = 0
  最优: 投资, EU = 70

如果M=坏(0.6):
  EU(I=是) = 0.3 × 100 + 0.7 × (-50) = -5
  EU(I=否) = 0
  最优: 不投资, EU = 0

EU(with info) = 0.4 × 70 + 0.6 × 0 = 28

VPI(M) = 28 - 25 = 3
```

**(c) 不完美信息VPI(预测)**

```
首先计算后验:
P(预测=好) = P(预测=好|M=好)P(M=好) + P(预测=好|M=坏)P(M=坏)
         = 0.8 × 0.4 + 0.2 × 0.6 = 0.32 + 0.12 = 0.44

P(M=好|预测=好) = P(预测=好|M=好)P(M=好) / P(预测=好)
              = 0.8 × 0.4 / 0.44 ≈ 0.727

P(M=好|预测=坏) = 0.2 × 0.4 / 0.56 ≈ 0.143

如果预测=好(0.44):
  EU(I=是) = 0.727 × 70 + 0.273 × (-5) ≈ 50.9 - 1.4 = 49.5
  EU(I=否) = 0
  最优: 投资

如果预测=坏(0.56):
  EU(I=是) = 0.143 × 70 + 0.857 × (-5) ≈ 10 - 4.3 = 5.7
  EU(I=否) = 0
  最优: 投资

EU(with prediction) = 0.44 × 49.5 + 0.56 × 5.7 ≈ 21.8 + 3.2 = 25

VPI(预测) = 25 - 25 = 0

解释: 不完美预测在此情况下没有价值，因为即使预测坏，投资仍是更优选择。
```

### 题目 7：人类决策偏误 ⭐⭐⭐⭐

**难度**: ⭐⭐⭐⭐  
**涉及知识点**: 阿莱悖论、前景理论、框架效应

**题目**:  
考虑以下阿莱悖论的经典设置：

**问题1 - 选择A或B:**
- A: 100%获得3000元
- B: 80%获得4000元, 20%获得0元

**问题2 - 选择C或D:**
- C: 20%获得4000元, 80%获得0元  
- D: 25%获得3000元, 75%获得0元

(a) 证明如果选择A和D，或选择B和C，都与期望效用理论一致。

(b) 解释为什么大多数人选择A和C构成了悖论。

(c) 讨论前景理论如何解释这一模式。

#### 逐步提示

<details>
<summary>点击查看提示</summary>

1. 设U(0)=0，检验偏好不等式
2. A≻B和C≻D导致矛盾的不等式
3. 前景理论: 确定性效应、损失厌恶
</details>

#### 详细解答

**(a) 一致性证明**

```
设U(0) = 0, U(3000) = u₃, U(4000) = u₄

如果选择A和D:
  A≻B: u₃ > 0.8 × u₄
  D≻C: 0.25 × u₃ > 0.2 × u₄ ⇒ u₃ > 0.8 × u₄
  一致!

如果选择B和C:
  B≻A: 0.8 × u₄ > u₃
  C≻D: 0.2 × u₄ > 0.25 × u₃ ⇒ 0.8 × u₄ > u₃
  一致!
```

**(b) 悖论解释**

```
大多数人选择A和C:
  A≻B: u₃ > 0.8 × u₄  ... (1)
  C≻D: 0.2 × u₄ > 0.25 × u₃ ⇒ 0.8 × u₄ > u₃ ... (2)

(1)和(2)矛盾!不存在满足两者的效用函数。
```

**(c) 前景理论解释**

```
1. 确定性效应(Certainty Effect):
   - 人们过度加权确定的结果
   - A中的100%获得比B中的80%更具吸引力

2. 参照点依赖:
   - 问题1: 在获得域，风险厌恶
   - 问题2: 在获得域，但都是概率性的

3. 概率加权:
   - 小概率过度加权
   - 中等概率(20% vs 25%)的差异被放大

4. 框架效应:
   - 如果问题2重新表述为"从4000元开始，有80%概率失去它"...
   - 选择模式可能改变
```

### 题目 8：多属性决策网络 ⭐⭐⭐⭐

**难度**: ⭐⭐⭐⭐  
**涉及知识点**: 决策网络、多属性效用、随机占优

**题目**:  
考虑一个产品开发决策，涉及两个属性:
- 利润(P): 单位:百万元
- 环境影响(E): 单位:吨碳排放(负效用)

有两个设计方案:

**方案A**:
- P ~ Uniform(5, 15)
- E = 20 (确定)

**方案B**:
- P ~ Uniform(8, 12)  
- E ~ Uniform(15, 25)

(a) 方案A是否在利润上随机占优于方案B？

(b) 假设效用函数 U(P, E) = P - 0.3E，计算每个方案的期望效用。

(c) 如果环境影响限制为E ≤ 22，分析这会如何改变决策。

#### 逐步提示

<details>
<summary>点击查看提示</summary>

1. 随机占优: 比较累积分布函数
2. 期望效用: 计算E[U]
3. 约束可能排除某些方案或改变最优选择
</details>

#### 详细解答

**(a) 随机占优分析**

```
方案A利润: Uniform(5, 15)
方案B利润: Uniform(8, 12)

累积分布:
- F_A(x) = (x-5)/10 for 5≤x≤15
- F_B(x) = (x-8)/4 for 8≤x≤12

检查: F_A(x) ≥ F_B(x) 对所有x?

在x=10:
  F_A(10) = 0.5
  F_B(10) = 0.5

在x=9:
  F_A(9) = 0.4
  F_B(9) = 0.25

F_A(9) > F_B(9)，不满足F_A ≥ F_B

结论: A不在利润上随机占优于B
(实际上B在利润上占优，因为B的利润总是≥8，而A可能<8)
```

**(b) 期望效用计算**

```
E[P_A] = (5+15)/2 = 10
E[E_A] = 20
EU(A) = 10 - 0.3 × 20 = 10 - 6 = 4

E[P_B] = (8+12)/2 = 10
E[E_B] = (15+25)/2 = 20
EU(B) = 10 - 0.3 × 20 = 4

相同期望效用!
```

**(c) 约束分析**

```
约束E ≤ 22:

方案A: E = 20 ≤ 22 ✓ 可行
方案B: E ~ Uniform(15, 25)，P(E>22) = 3/10 = 0.3

需要条件期望:
EU(B | E ≤ 22) = E[P_B] - 0.3 × E[E_B | E ≤ 22]

E[E_B | E ≤ 22] = (15+22)/2 = 18.5

EU(B | feasible) = 10 - 0.3 × 18.5 = 10 - 5.55 = 4.45

EU(A) = 4

现在B更优(4.45 > 4)，因为约束排除了B的高排放场景。
```

### 题目 9：顺从与开关游戏 ⭐⭐⭐⭐⭐

**难度**: ⭐⭐⭐⭐⭐  
**涉及知识点**: 偏好不确定性、顺从、博弈论

**题目**:  
考虑16.7.2节的开关游戏。机器人R可以选择：
1. **行动a**: 预订酒店，对H的价值V ~ Uniform(-40, 60)
2. **关闭o**: 价值0
3. **顺从d**: 询问H，H可关闭R或允许继续

如果H允许继续，R知道V > 0，后验V ~ Uniform(0, 60)。

(a) 计算EU(a)、EU(o)和EU(d)。

(b) 证明当R对V完全确定时，顺从没有价值。

(c) 扩展到两阶段决策: R可以现在行动、关闭或顺从；如果顺从且H允许，R又可以行动、关闭或再次顺从。分析最优策略。

#### 逐步提示

<details>
<summary>点击查看提示</summary>

1. 均匀分布的期望 = (a+b)/2
2. 完全确定意味着没有信息增益
3. 多阶段需要考虑信息的累积价值
</details>

#### 详细解答

**(a) 期望效用计算**

```
EU(a) = E[V] = (-40 + 60)/2 = 10

EU(o) = 0

EU(d):
  P(H关闭) = P(V < 0) = 40/100 = 0.4
  P(H允许) = P(V ≥ 0) = 60/100 = 0.6
  
  如果H允许，E[V | V≥0] = (0+60)/2 = 30
  
  EU(d) = 0.4 × 0 + 0.6 × 30 = 18

最优选择: 顺从 (EU = 18 > 10 > 0)
```

**(b) 完全确定时顺从无价值**

```
假设R确定V = v:

如果v < 0:
  EU(a) = v < 0
  EU(o) = 0
  EU(d) = 0 (H会关闭)
  最优: 关闭

如果v ≥ 0:
  EU(a) = v
  EU(o) = 0  
  EU(d) = v (H允许，R获得v)
  最优: 行动(与顺从相同价值)

在完全确定时，顺从从不严格优于直接行动。
```

**(c) 两阶段决策**

```
如果第一次顺从后H允许:
  后验V ~ Uniform(0, 60)
  
  第二次选择:
    EU(a) = 30
    EU(o) = 0
    EU(d) = 0.4 × 0 + 0.6 × 45 = 27 (因为后验更新为Uniform(0,60)再截断)
    
    注意: 第二次顺从的后验是Uniform(0,60)如果再次允许
    
  最优: 直接行动 (30 > 27)

因此最优策略:
  第一阶段: 顺从
  如果允许: 第二阶段直接行动(不再顺从)
  
原因是第一次顺从后不确定性已大幅降低(从100范围到60范围)，
进一步顺从的信息价值递减。
```

### 题目 10：优化者诅咒 ⭐⭐⭐⭐⭐

**难度**: ⭐⭐⭐⭐⭐  
**涉及知识点**: 乐观者诅咒、贝叶斯校正、决策后失望

**题目**:  
考虑一个有n个候选药物的决策问题。每个药物i的真实效用U_i = 0，但估计值Û_i ~ N(0, 1)独立。

(a) 证明选择最大估计值的期望大于0。

(b) 计算n=10时的期望"失望"(选择的最大估计减去实际0)。

(c) 讨论如何用贝叶斯方法校正这一偏误。

#### 详细解答

**(a) 最大估计的期望**

```
设X₁, ..., Xₙ ~ N(0, 1) i.i.d.
令X_max = max(X₁, ..., Xₙ)

E[X_max] > 0 对于n ≥ 2:

因为分布对称，P(X_max > 0) > P(X_max < 0)
(至少一个X_i > 0的概率 > 全部X_i < 0的概率)

具体地:
P(X_max ≤ x) = Φ(x)ⁿ
f_{X_max}(x) = n × Φ(x)^{n-1} × φ(x)

E[X_max] = ∫ x × n × Φ(x)^{n-1} × φ(x) dx > 0
```

**(b) n=10时的期望失望**

```
对于标准正态，最大值的期望近似:
E[X_max] ≈ Φ^{-1}(1 - 1/n) + 偏差校正

对于n=10:
E[X_max] ≈ 1.54

期望失望 ≈ 1.54个标准差

这意味着如果我们选择估计最高的药物，
期望其实际效用比估计低约1.54。
```

**(c) 贝叶斯校正**

```
1. 建立层级模型:
   U_i ~ N(μ, σ²)  // 先验
   Û_i | U_i ~ N(U_i, τ²)  // 测量误差

2. 给定Û_i，后验:
   U_i | Û_i ~ N(后验均值, 后验方差)
   
   后验均值 = (τ²μ + σ²Û_i) / (σ² + τ²)
   
   这是Û_i向先验均值μ的收缩

3. 决策:
   选择argmax E[U_i | Û_i]而非argmax Û_i
   
   对于对称先验μ=0:
   E[U_i | Û_i] = w × Û_i, w < 1
   
   这意味着: 估计越高，收缩比例越大

4. 实际应用:
   - 估计先验参数μ, σ², τ²
   - 使用经验贝叶斯
   - 或在组织层面建立预测效用的历史数据库
```

**1. 根据期望效用理论，理性智能体应该：**

A. 最大化期望货币价值  
B. 最大化期望效用  
C. 最小化最大可能损失  
D. 选择概率最高的结果  

<details>
<summary>答案</summary>

**答案: B**

期望效用理论的核心原则是最大化期望效用，而非货币价值。这允许模型考虑风险态度。

</details>

**2. 如果效用函数是凹函数(如U(x)=√x)，智能体表现出：**

A. 风险中性  
B. 风险厌恶  
C. 风险寻求  
D. 无法确定  

<details>
<summary>答案</summary>

**答案: B**

凹效用函数反映边际效用递减，导致确定性结果比具有相同期望值的彩票更受偏好——风险厌恶的特征。

</details>

**3. 信息价值(VPI)的性质包括：**

A. 总是正的  
B. 具有可加性  
C. 可能是负的  
D. 与信息成本无关  

<details>
<summary>答案</summary>

**答案: A (非负性)**

VPI ≥ 0，因为在最坏情况下可以忽略信息。VPI不具有可加性，而是次序独立的。

</details>

**4. 阿莱悖论表明：**

D. 金钱和效用成正比  

<details>
<summary>答案</summary>

**答案: C**

阿莱悖论展示了人们在某些选择模式中表现出与期望效用理论不一致的行为，特别是涉及确定性效应时。

</details>

**5. 多属性效用理论中的"偏好独立性"是指：**

A. 所有属性相互独立  
B. 两个属性的权衡不依赖于其他属性  
C. 偏好与属性值无关  
D. 属性之间的相关性为零  

<details>
<summary>答案</summary>

**答案: B**

偏好独立性(MPI)是指对两个属性的偏好权衡不受第三个属性值的影响，这是加性价值函数存在的条件。

</details>

**7. 随机占优意味着：**

A. 一个选项在所有结果上都更好  
B. 对所有单调效用函数，占优选项的期望效用更高  
C. 两个选项具有相同的期望价值  
D. 占优选项的风险更小  

<details>
<summary>答案</summary>

**答案: B**

随机占优的定义是：如果A随机占优于B，则对所有单调不减的效用函数，A的期望效用至少与B一样高。

</details>

C. 必须等待确定性  
D. 应该随机选择  

<details>
<summary>答案</summary>

**答案: B**

偏好不确定性可以建模为随机变量(如LikesDurian)，然后应用标准的决策理论框架。

**9. 在"开关游戏"中，机器人顺从人类的原因是：**

A. 人类总是正确的  
B. 顺从提供了关于人类偏好的信息  
C. 机器人被编程为顺从  
D. 避免法律责任  

<details>
<summary>答案</summary>

**答案: B**

当机器人对人类的偏好不确定时，人类的决策提供了信息。信息的期望价值非负，使顺从成为理性选择。

**10. 优化者诅咒(Optimizer's Curse)是指：**

A. 选择最优选项总是最好的  
B. 选择最大估计值会导致高估  
C. 优化算法总是收敛  
D. 最优选项的实际效用最高  

<details>
<summary>答案</summary>

**答案: B**

优化者诅咒是指当我们从多个选项中选择估计效用最高的时，由于估计误差，期望实际效用会低于估计值——即过度乐观。

</details>

## 考点解析

### 核心概念对照表

| 概念 | 关键理解 | 常见错误 |
|-----|---------|---------|
| 期望效用 | 效用概率加权平均 | 与EMV混淆 |
| 风险厌恶 | 凹效用函数 | 认为风险厌恶总是理性的 |
| VPI | 信息改善决策的期望价值 | 认为VPI可相加 |
| MPI | 属性权衡独立性 | 与统计独立性混淆 |
| 随机占优 | 一阶随机占优定义 | 只比较期望值 |
| 阿莱悖论 | 确定性效应 | 认为人们总是非理性 |
| 优化者诅咒 | 选择偏差 | 忽视后验收缩 |

### 公式速查

```
期望效用: EU(a) = Σ P(result=s') × U(s')

信息价值: VPI(E) = Σ P(E=e) × EU(best|e) - EU(best)

加性价值函数: V(x₁,...,xₙ) = Σ vᵢ(xᵢ)

风险溢价: RP = EMV - CE (确定性等价值)

贝叶斯校正: E[U|Û] = (τ²μ + σ²Û) / (σ² + τ²)
```
