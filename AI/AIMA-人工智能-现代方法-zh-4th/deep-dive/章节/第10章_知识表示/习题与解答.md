### 练习10.1 [难度: ★☆☆] 类别层次结构

**涉及知识点**: 类别、子类、继承

**题目**:
用一阶逻辑表示以下类别层次结构和属性：

**类别结构**：
- Animals（动物）包含 Mammals（哺乳动物）和 Birds（鸟类）
- Mammals 包含 Dogs（狗）和 Cats（猫）
- Birds 包含 Eagles（鹰）和 Penguins（企鹅）

**属性**：
- 所有动物都有生命
- 哺乳动物都有毛发
- 鸟类都有羽毛
- 狗会吠叫
- 猫会抓老鼠
- 鹰会飞
- 企鹅不会飞但会游泳

**要求**：
1. 用Subset表示子类关系
2. 用Member表示成员关系
3. 写出至少5条描述属性的语句

---

**详细解答**:

**1. 子类关系**：

```
Mammals ⊂ Animals
Birds ⊂ Animals
Dogs ⊂ Mammals
Cats ⊂ Mammals
Eagles ⊂ Birds
Penguins ⊂ Birds
```

**2. 属性描述**：

```
% 所有动物都有生命
∀x x ∈ Animals ⇒ Living(x)

% 哺乳动物都有毛发
∀x x ∈ Mammals ⇒ HasHair(x)

% 鸟类都有羽毛
∀x x ∈ Birds ⇒ HasFeathers(x)

% 狗会吠叫
∀x x ∈ Dogs ⇒ CanBark(x)

% 猫会抓老鼠
∀x x ∈ Cats ⇒ CanCatchMice(x)

% 鹰会飞
∀x x ∈ Eagles ⇒ CanFly(x)

% 企鹅不会飞但会游泳
∀x x ∈ Penguins ⇒ ¬CanFly(x) ∧ CanSwim(x)
```

**3. 继承推导示例**：

已知：Buddy ∈ Dogs，可推导：
- Buddy ∈ Mammals （子类继承）
- Buddy ∈ Animals （传递继承）
- Living(Buddy) （属性继承）
- HasHair(Buddy) （属性继承）

---

**考点解析**:
- 类别层次通过子类关系组织
- 属性从父类别继承到子类别
- 子类可以覆盖父类的属性（如Penguin覆盖CanFly）

### 练习10.2 [难度: ★★☆] 划分与不相交

**涉及知识点**: 划分(Partition)、不相交(Disjoint)

**题目**:
定义划分概念，并判断{Mammals, Birds, Reptiles, Amphibians, Fish}是否构成Vertebrates的划分。

---

**详细解答**:

**形式化定义**：

```
% 不相交
Disjoint(s) ⇔ ∀c₁,c₂ ∈ s (c₁ ≠ c₂ ⇒ c₁ ∩ c₂ = ∅)

% 完全分解
ExhaustiveDecomposition(s, c) ⇔ 
    ∀x x ∈ c ⇔ ∃cᵢ ∈ s, x ∈ cᵢ

% 划分
Partition(s, c) ⇔ Disjoint(s) ∧ ExhaustiveDecomposition(s, c)
```

**应用**：

设s = {Mammals, Birds, Reptiles, Amphibians, Fish}

**验证**：
- 不相交：五类动物互不相交 ✓
- 完全分解：所有脊椎动物都属于其中一类 ✓

**结论**：这五类构成Vertebrates的划分。

**详细解答**:

```
5. PartOf(Piston, Car)        [3,4假言推理] ✓
```

### 练习10.5 [难度: ★★☆] 物质与对象

**涉及知识点**: 物质(Stuffs)、固有属性

**题目**:
解释为什么Butter是物质而PoundOfButter不是物质。

---

**详细解答**:

**物质的特征**：
```
Stuff(s) ∧ PartOf(p, s) ⇒ Stuff(p) ∧ SameStuff(p, s)
```
任意部分仍是同种物质。

**PoundOfButter的问题**：
```
b ∈ PoundOfButter ⇔ b ∈ Butter ∧ Weight(b) = Pound(1)
```

如果PartOf(p, b)且Weight(p) = Pound(0.5)：
- p ∈ Butter ✓（物质保持）
- p ∉ PoundOfButter（重量不是1磅）

因此PoundOfButter是基于外在属性的对象类别，不是物质。

### 练习10.6 [难度: ★★★] 语义网络分析

**涉及知识点**: 语义网络、继承、缺省值

**题目**:
分析给定的类别层次，回答Bird和Penguin的CanFly属性。

**类别定义**：
- Bird定义缺省值CanFly = true
- Penguin是Bird的子类，定义CanFly = false

---

**详细解答**:

**Bird查询CanFly**：
- 在Bird自己的属性中查找
- 返回缺省值：true

**Penguin查询CanFly**：
- 首先在Penguin自己的属性中查找
- 找到特定值：false
- 停止搜索（覆盖Bird的缺省值）

**结论**：
- 继承算法优先使用更具体（更靠近对象）的值
- 这允许子类覆盖父类的缺省属性

### 练习10.7 [难度: ★★★] 模态逻辑基础

**涉及知识点**: 模态逻辑、知识算子

**题目**:
解释∃x KₐP(x)与Kₐ∃x P(x)的区别。

---

**详细解答**:

```
∃x KₐP(x): 
"存在一个特定的x，使得a知道P(x)"
- a知道"那个人是间谍"（a知道是谁）
- 存在量词在模态算子内

Kₐ∃x P(x):
"a知道存在某个x满足P(x)"
- a知道"存在间谍"（但a不知道是谁）
- 存在量词在模态算子外
```

**关键区别**：
- ∃x KₐP(x) 强于 Kₐ∃x P(x)
- 前者蕴含后者，反之不成立
- 这是量词的模态范围问题

### 练习10.8 [难度: ★★★] 描述逻辑包容

**涉及知识点**: 描述逻辑、包容检测

**题目**:
判断Mother是否是Person的子集。

**定义**：
- Person = Thing
- Mother = And(Female, Parent)
- Parent = And(Person, AtLeast(1, hasChild))

---

**详细解答**:

```
Mother = And(Female, Parent)
       = And(Female, And(Person, AtLeast(1, hasChild)))
       = And(Female, Person, AtLeast(1, hasChild))

Person = Thing
```

**判断**：
- Mother的组件包含Person
- And的所有组件必须满足
- 因此Mother的实例必须满足Person

**结论**：Mother ⊂ Person ✓

### 练习10.9 [难度: ★★★] 缺省逻辑

**涉及知识点**: 缺省逻辑、尼克松菱形

**题目**:
分析尼克松菱形的两个扩展。

**背景**：
- Nixon是Quaker又是Republican
- Quaker缺省是Pacifist
- Republican缺省不是Pacifist

---

**详细解答**:

**扩展1**（Nixon是和平主义者）：
- 应用贵格会缺省规则 → Pacifist(Nixon)
- 共和党规则失效（论证不一致）

**扩展2**（Nixon不是和平主义者）：
- 应用共和党缺省规则 → ¬Pacifist(Nixon)
- 贵格会规则失效

**问题**：两个扩展都有效但矛盾！

**解决方案**：优先限定(Prioritized Circumscription)
- 声明某一规则优先
- 例如：政治属性优先于宗教属性
- 结果：Nixon不是和平主义者

## 挑战练习题

### 练习10.10 [难度: ★★★★] 真值维护系统

**涉及知识点**: JTMS、论证、信念修正

**题目**:
设计汽车故障诊断的TMS场景。

**知识**：
1. 引擎不启动 ∧ 电瓶没电 ⇒ 需要充电
2. 灯不亮 ∧ 电瓶没电 ⇒ 需要充电
3. 电瓶没电 ⇒ 引擎不启动

**观测**：引擎不启动，灯不亮

---

**详细解答**:

**论证网络**：
```
观测A: 引擎不启动
观测B: 灯不亮
假设H₁: 电瓶没电

推导：
- 从A和H₁：引擎不启动 [论证: {A, H₁, 规则3}]
- 从B和H₁：需要充电 [论证: {B, H₁, 规则2}]
- 从A, B, H₁：需要充电 [论证: {A, B, H₁, 规则1}]
```

**新证据：喇叭响**：
- 如果喇叭响与电瓶状态相关
- H₁可能失去部分支持
- 需要权衡证据强度决定是否收回假设

### 练习10.11 [难度: ★★★★] 时间间隔推理

**涉及知识点**: Allen代数、时间关系

**题目**:
用Allen代数描述会议时间关系。

**会议**：
- A: 9:00-11:00
- B: 10:00-12:00
- C: 11:30-12:30

---

**详细解答**:

**时间关系**：
```
A与B: Overlap(A, B)
      9:00---A---11:00
            10:00---B---12:00

A与C: Before(A, C)
      A结束11:00 < C开始11:30

B与C: Overlap(B, C)
      B: 10:00-12:00
            C: 11:30-12:30
```

**A和C不重叠证明**：
```
Before(i, j) ⇔ End(i) < Begin(j)
End(A) = 11:00
Begin(C) = 11:30
11:00 < 11:30
因此 Before(A, C) 为真
```
