### 习题20.1 【难度：★☆☆】
**涉及知识点：** 贝叶斯学习基本概念、MAP与最大似然

**题目：**
假设有一袋糖果，可能是以下三种类型之一：
- $h_1$：100%樱桃味
- $h_2$：50%樱桃味 + 50%酸橙味  
- $h_3$：100%酸橙味

先验概率为 $P(h_1) = 0.2, P(h_2) = 0.5, P(h_3) = 0.3$。现在你连续观测到3颗糖果，都是酸橙味。

(1) 计算每个假设的后验概率
(2) 确定MAP假设
(3) 预测下一颗糖果是酸橙味的概率（贝叶斯预测）

**逐步提示：**
1. 使用贝叶斯公式计算后验概率
2. 似然计算使用伯努利分布
3. 贝叶斯预测是对所有假设加权求和

**详细解答：**

**(1) 计算后验概率**

观测数据 $d$ = 3颗酸橙味糖果

似然计算（独立同分布假设）：
- $P(d|h_1) = 0^3 = 0$（$h_1$不可能产生酸橙味）
- $P(d|h_2) = 0.5^3 = 0.125$
- $P(d|h_3) = 1^3 = 1$

先验×似然：
- $h_1$：$0.2 \times 0 = 0$
- $h_2$：$0.5 \times 0.125 = 0.0625$
- $h_3$：$0.3 \times 1 = 0.3$

归一化常数 $\alpha = 0 + 0.0625 + 0.3 = 0.3625$

后验概率：
- $P(h_1|d) = 0 / 0.3625 = 0$
- $P(h_2|d) = 0.0625 / 0.3625 \approx 0.172$
- $P(h_3|d) = 0.3 / 0.3625 \approx 0.828$

**(2) MAP假设**

MAP假设是后验概率最大的假设：
$$h_{MAP} = h_3 \text{（全酸橙味糖果袋）}$$

**(3) 贝叶斯预测**

$$P(D_4 = \text{lime}|d) = \sum_i P(D_4 = \text{lime}|h_i) P(h_i|d)$$

$$= 0 \times 0 + 0.5 \times 0.172 + 1 \times 0.828$$

$$\approx 0 + 0.086 + 0.828 = 0.914$$

**考点解析：**
本题考查贝叶斯学习的核心概念。MAP只使用单个最佳假设，而贝叶斯预测整合所有假设的信息，通常更稳健。随着观测数据增加，贝叶斯预测会趋向真实值。

### 习题20.2 【难度：★★☆】
**涉及知识点：** 最大似然估计、离散分布

**题目：**
假设一枚硬币正面朝上的概率为 $\theta$。你抛掷硬币10次，观察到7次正面，3次反面。

(1) 写出似然函数 $L(\theta)$
(2) 求最大似然估计 $\hat{\theta}_{ML}$
(3) 如果采用贝叶斯方法，使用Beta(2,2)作为先验，求后验分布及后验均值预测

**逐步提示：**
1. 似然函数是二项分布的概率质量函数
2. 对数似然求导并令为0
3. Beta分布是二项分布的共轭先验

**详细解答：**

**(1) 似然函数**

$$L(\theta) = P(\text{数据}|\theta) = \binom{10}{7}\theta^7(1-\theta)^3 \propto \theta^7(1-\theta)^3$$

**(2) 最大似然估计**

对数似然：
$$\ln L(\theta) = 7\ln\theta + 3\ln(1-\theta) + \text{const}$$

求导并令为0：
$$\frac{d}{d\theta}\ln L = \frac{7}{\theta} - \frac{3}{1-\theta} = 0$$

$$\frac{7}{\theta} = \frac{3}{1-\theta}$$

$$7(1-\theta) = 3\theta$$

$$7 = 10\theta$$

$$\hat{\theta}_{ML} = \frac{7}{10} = 0.7$$

**(3) 贝叶斯后验**

先验：$\theta \sim \text{Beta}(2, 2)$，即 $P(\theta) \propto \theta^{2-1}(1-\theta)^{2-1} = \theta(1-\theta)$

观测：7正3反

后验：
$$P(\theta|\text{数据}) \propto P(\text{数据}|\theta)P(\theta) \propto \theta^7(1-\theta)^3 \cdot \theta(1-\theta) = \theta^8(1-\theta)^4$$

因此后验分布为 $\text{Beta}(9, 4)$

后验均值预测：
$$E[\theta|\text{数据}] = \frac{9}{9+4} = \frac{9}{13} \approx 0.692$$

**考点解析：**
本题展示了MLE和贝叶斯估计的区别。MLE只使用数据信息，而贝叶斯方法结合先验和数据。在小样本时，先验影响大；大样本时，两者收敛。

### 习题20.3 【难度：★★☆】
**涉及知识点：** 朴素贝叶斯、分类器

**题目：**
考虑一个垃圾邮件分类的朴素贝叶斯模型，使用以下简化特征：
- 是否包含"免费"（Free）
- 是否包含"点击"（Click）

训练数据：
| 邮件 | 免费 | 点击 | 类别 |
|:---:|:---:|:---:|:---:|
| 1 | 是 | 是 | 垃圾 |
| 2 | 是 | 否 | 垃圾 |
| 3 | 否 | 是 | 垃圾 |
| 4 | 否 | 否 | 正常 |
| 5 | 否 | 否 | 正常 |
| 6 | 否 | 是 | 正常 |

(1) 计算朴素贝叶斯模型的所有参数（使用最大似然估计）
(2) 对新邮件"免费=是，点击=否"进行分类

**逐步提示：**
1. 计算先验概率 P(类别)
2. 计算条件概率 P(特征|类别)
3. 使用贝叶斯规则计算后验概率

**详细解答：**

**(1) 参数估计**

**先验概率：**
- $P(\text{垃圾}) = 3/6 = 0.5$
- $P(\text{正常}) = 3/6 = 0.5$

**条件概率（垃圾邮件）：**
- $P(\text{免费}=\text{是}|\text{垃圾}) = 2/3 \approx 0.667$
- $P(\text{免费}=\text{否}|\text{垃圾}) = 1/3 \approx 0.333$
- $P(\text{点击}=\text{是}|\text{垃圾}) = 2/3 \approx 0.667$
- $P(\text{点击}=\text{否}|\text{垃圾}) = 1/3 \approx 0.333$

**条件概率（正常邮件）：**
- $P(\text{免费}=\text{是}|\text{正常}) = 0/3 = 0$
- $P(\text{免费}=\text{否}|\text{正常}) = 3/3 = 1$
- $P(\text{点击}=\text{是}|\text{正常}) = 1/3 \approx 0.333$
- $P(\text{点击}=\text{否}|\text{正常}) = 2/3 \approx 0.667$

**(2) 分类新邮件**

新邮件特征：免费=是，点击=否

**计算后验概率（未归一化）：**

垃圾邮件：
$$P(\text{垃圾}) \cdot P(\text{免费}=\text{是}|\text{垃圾}) \cdot P(\text{点击}=\text{否}|\text{垃圾})$$
$$= 0.5 \times 0.667 \times 0.333 \approx 0.111$$

正常邮件：
$$P(\text{正常}) \cdot P(\text{免费}=\text{是}|\text{正常}) \cdot P(\text{点击}=\text{否}|\text{正常})$$
$$= 0.5 \times 0 \times 0.667 = 0$$

**结论：** 分类为**垃圾邮件**。

**零概率问题：**
注意 $P(\text{免费}=\text{是}|\text{正常}) = 0$ 导致正常邮件概率为0。实际应用中应使用拉普拉斯平滑避免此问题。

**考点解析：**
朴素贝叶斯是简单而有效的分类器。条件独立性假设使其参数数量线性增长，但这也可能导致概率估计过度自信。

### 习题20.4 【难度：★★☆】
**涉及知识点：** 高斯分布、最大似然估计

**题目：**
给定以下5个数据点，假设它们来自一个高斯分布：$2, 4, 4, 6, 4$

(1) 计算均值 $\mu$ 和方差 $\sigma^2$ 的最大似然估计
(2) 如果采用贝叶斯方法，使用正态-逆卡方先验，解释后验分布的形式
(3) 使用(1)的结果，计算新数据点 $x=5$ 的似然

**逐步提示：**
1. MLE均值是样本均值，MLE方差是样本方差（除以N而非N-1）
2. 高斯分布的概率密度函数

**详细解答：**

**(1) 最大似然估计**

样本均值：
$$\hat{\mu} = \frac{2 + 4 + 4 + 6 + 4}{5} = \frac{20}{5} = 4$$

样本方差（MLE使用N而非N-1）：
$$\hat{\sigma}^2 = \frac{(2-4)^2 + (4-4)^2 + (4-4)^2 + (6-4)^2 + (4-4)^2}{5}$$
$$= \frac{4 + 0 + 0 + 4 + 0}{5} = \frac{8}{5} = 1.6$$

**(2) 贝叶斯后验**

高斯分布参数的共轭先验是正态-逆卡方分布（或正态-逆Gamma分布）。

观测数据后，后验分布仍为同族分布，参数更新为：
- 后验均值是先验均值和样本均值的加权平均
- 后验方差受先验方差和数据离散程度共同影响

随着数据量增加，后验会越来越集中在MLE估计附近。

**(3) 数据点似然**

高斯分布概率密度函数：
$$P(x|\mu,\sigma) = \frac{1}{\sigma\sqrt{2\pi}}\exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$$

代入 $x=5, \mu=4, \sigma^2=1.6$：

$$P(5) = \frac{1}{\sqrt{1.6}\sqrt{2\pi}}\exp\left(-\frac{(5-4)^2}{2 \times 1.6}\right)$$

$$= \frac{1}{\sqrt{3.2\pi}}\exp\left(-\frac{1}{3.2}\right)$$

$$\approx \frac{1}{3.17} \times 0.732 \approx 0.231$$

**考点解析：**
本题考查连续分布的参数估计。注意MLE方差与无偏估计的区别（除以N vs 除以N-1）。高斯分布是统计学习中最基础也最重要的分布。

### 习题20.5 【难度：★★☆】
**涉及知识点：** 生成模型 vs 判别模型

**题目：**
解释以下概念并比较：

(1) 生成模型与判别模型的定义和区别
(2) 朴素贝叶斯属于哪类模型？逻辑斯谛回归属于哪类模型？
(3) 在什么情况下生成模型可能优于判别模型？

**详细解答：**

**(1) 定义与区别**

**生成模型（Generative Model）：**
- 建模：$P(X, Y) = P(Y)P(X|Y)$
- 学习类别的联合概率分布
- 可以生成属于某类别的新样本
- 例子：朴素贝叶斯、高斯判别分析、隐马尔可夫模型

**判别模型（Discriminative Model）：**
- 建模：$P(Y|X)$ 或直接学习决策边界
- 专注于区分不同类别
- 不能生成新样本
- 例子：逻辑斯谛回归、SVM、决策树、神经网络

**核心区别：**
- 生成模型"理解"每个类别的特征分布
- 判别模型"专注"于分类边界

**(2) 模型分类**

- **朴素贝叶斯**：生成模型（建模 $P(X|Y)P(Y)$）
- **逻辑斯谛回归**：判别模型（直接建模 $P(Y|X)$）

**(3) 生成模型优势场景**

生成模型在以下情况可能更好：
- **小样本学习**：生成模型利用先验分布，在数据少时更鲁棒
- **半监督学习**：容易利用未标记数据
- **缺失数据处理**：可以自然地处理缺失特征
- **需要生成样本**：如图像生成
- **多任务学习**：学到的特征表示可用于多个相关任务

**实证结果：**
Ng和Jordan(2002)的实验表明，在15个小数据集上，朴素贝叶斯（生成）在14个数据集上使用少量数据时优于逻辑斯谛回归（判别），但使用全部数据时逻辑斯谛回归在9个数据集上更好。

**考点解析：**
理解生成模型和判别模型的区别对于选择合适算法很重要。实践中，判别模型在大数据集上通常表现更好，但生成模型有其独特优势。

### 习题20.6 【难度：★★★】
**涉及知识点：** EM算法、混合高斯模型

**题目：**
考虑一个包含两个高斯分量的混合模型，数据点：0, 2, 5, 7

初始参数：
- 分量1：$w_1=0.5, \mu_1=1, \sigma_1^2=1$
- 分量2：$w_2=0.5, \mu_2=6, \sigma_2^2=1$

执行一轮EM算法：

(1) E步：计算每个数据点属于每个分量的责任值(responsibility)
(2) M步：更新参数

**逐步提示：**
1. E步使用贝叶斯规则计算后验概率
2. M步使用加权平均更新参数

**详细解答：**

**(1) E步：计算责任值**

高斯密度函数：$\mathcal{N}(x|\mu,\sigma^2) = \frac{1}{\sqrt{2\pi\sigma^2}}\exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$

对每个数据点计算：
$$\gamma_{ij} = P(C=i|x_j) = \frac{w_i \mathcal{N}(x_j|\mu_i,\sigma_i^2)}{\sum_k w_k \mathcal{N}(x_j|\mu_k,\sigma_k^2)}$$

数据点 $x=0$：
- $\mathcal{N}(0|1,1) = \frac{1}{\sqrt{2\pi}}e^{-1/2} \approx 0.242$
- $\mathcal{N}(0|6,1) = \frac{1}{\sqrt{2\pi}}e^{-18} \approx 0$
- $\gamma_{1,0} \approx 1, \gamma_{2,0} \approx 0$

数据点 $x=2$：
- $\mathcal{N}(2|1,1) = \frac{1}{\sqrt{2\pi}}e^{-1/2} \approx 0.242$
- $\mathcal{N}(2|6,1) = \frac{1}{\sqrt{2\pi}}e^{-8} \approx 0$
- $\gamma_{1,2} \approx 1, \gamma_{2,2} \approx 0$

数据点 $x=5$：
- $\mathcal{N}(5|1,1) = \frac{1}{\sqrt{2\pi}}e^{-8} \approx 0$
- $\mathcal{N}(5|6,1) = \frac{1}{\sqrt{2\pi}}e^{-1/2} \approx 0.242$
- $\gamma_{1,5} \approx 0, \gamma_{2,5} \approx 1$

数据点 $x=7$：
- $\mathcal{N}(7|1,1) \approx 0$
- $\mathcal{N}(7|6,1) = \frac{1}{\sqrt{2\pi}}e^{-1/2} \approx 0.242$
- $\gamma_{1,7} \approx 0, \gamma_{2,7} \approx 1$

**(2) M步：更新参数**

有效点数：
- $n_1 = 1 + 1 + 0 + 0 = 2$
- $n_2 = 0 + 0 + 1 + 1 = 2$

新权重：
- $w_1 = 2/4 = 0.5$
- $w_2 = 2/4 = 0.5$

新均值：
- $\mu_1 = (1\times0 + 1\times2 + 0\times5 + 0\times7)/2 = 1$
- $\mu_2 = (0\times0 + 0\times2 + 1\times5 + 1\times7)/2 = 6$

新方差：
- $\sigma_1^2 = (1(0-1)^2 + 1(2-1)^2)/2 = (1+1)/2 = 1$
- $\sigma_2^2 = (1(5-6)^2 + 1(7-6)^2)/2 = (1+1)/2 = 1$

**结论：** 本轮EM后参数基本不变（已接近收敛）。

**考点解析：**
EM算法是处理隐变量学习的标准方法。E步"猜测"隐变量，M步基于猜测更新参数。算法保证收敛到局部最优。

### 习题20.7 【难度：★★★】
**涉及知识点：** 隐变量学习、贝叶斯网络

**题目：**
考虑一个带有隐变量H的贝叶斯网络，结构为：$H \rightarrow A, H \rightarrow B$，其中A和B是布尔可观测变量，H是布尔隐变量。

观测数据（10个样例）：
| A | B | 计数 |
|:---:|:---:|:---:|
| T | T | 4 |
| T | F | 1 |
| F | T | 2 |
| F | F | 3 |

(1) 如果H可观测，描述如何学习参数
(2) 使用EM算法的思想，描述如何处理H不可观测的情况

**详细解答：**

**(1) H可观测时的参数学习**

网络参数：
- $P(H)$：隐变量的先验
- $P(A|H), P(A|\neg H)$：给定H时A的条件概率
- $P(B|H), P(B|\neg H)$：给定H时B的条件概率

**学习方法**：最大似然估计
- $P(H) = \frac{\text{H=T的样例数}}{\text{总样例数}}$
- $P(A|H) = \frac{\text{H=T且A=T的样例数}}{\text{H=T的样例数}}$
- 其他参数类似计算

由于是完全数据，每个参数可以独立估计。

**(2) H不可观测时的EM算法**

**初始化**：随机设置参数值

**E步**：对每个观测到的 $(A, B)$ 组合，计算
$$P(H|A,B) = \frac{P(A|H)P(B|H)P(H)}{P(A,B)}$$

将每个观测样例"分配"给H=T和H=F，权重为后验概率。

**M步**：将"完整"数据（带权重）当作真实数据，重新估计参数：

$$P(H) = \frac{\sum_{\text{所有样例}} P(H|A,B)}{\text{总样例数}}$$

$$P(A|H) = \frac{\sum_{A=T} P(H|A,B)}{\sum_{\text{所有样例}} P(H|A,B)}$$

其他参数类似计算。

**迭代**：重复E步和M步直到收敛。

**考点解析：**
EM算法是贝叶斯网络参数学习的核心方法。通过隐变量的概率分布（软分配），将不完全数据学习转化为完全数据学习问题。

### 习题20.8 【难度：★★★】
**涉及知识点：** 贝叶斯线性回归

**题目：**
考虑简化的贝叶斯线性回归：$y = \theta x + \epsilon$，其中 $\epsilon \sim \mathcal{N}(0, \sigma^2)$，$\sigma^2=1$已知。

先验：$\theta \sim \mathcal{N}(0, 1)$

观测数据：$(1, 2), (2, 3)$

(1) 写出似然函数
(2) 计算后验分布 $P(\theta|data)$
(3) 预测 $x=3$ 时的y值分布

**逐步提示：**
1. 高斯先验与高斯似然，后验也是高斯
2. 利用共轭先验的性质

**详细解答：**

**(1) 似然函数**

$$P(y|x,\theta) = \mathcal{N}(y; \theta x, \sigma^2) = \frac{1}{\sqrt{2\pi}}\exp\left(-\frac{(y-\theta x)^2}{2}\right)$$

对两个数据点：
- $(1, 2)$：$P(2|1,\theta) \propto \exp\left(-\frac{(2-\theta)^2}{2}\right)$
- $(2, 3)$：$P(3|2,\theta) \propto \exp\left(-\frac{(3-2\theta)^2}{2}\right)$

联合似然：
$$P(data|\theta) \propto \exp\left(-\frac{(2-\theta)^2 + (3-2\theta)^2}{2}\right)$$

**(2) 后验分布**

先验：$P(\theta) \propto \exp\left(-\frac{\theta^2}{2}\right)$（均值为0，方差为1）

后验：
$$P(\theta|data) \propto P(data|\theta)P(\theta)$$

$$\propto \exp\left(-\frac{(2-\theta)^2 + (3-2\theta)^2 + \theta^2}{2}\right)$$

展开指数：
$$(2-\theta)^2 + (3-2\theta)^2 + \theta^2$$
$$= 4 - 4\theta + \theta^2 + 9 - 12\theta + 4\theta^2 + \theta^2$$
$$= 6\theta^2 - 16\theta + 13$$

配方：
$$= 6\left(\theta^2 - \frac{8}{3}\theta\right) + 13$$
$$= 6\left(\theta - \frac{4}{3}\right)^2 - 6 \times \frac{16}{9} + 13$$
$$= 6\left(\theta - \frac{4}{3}\right)^2 + \text{const}$$

因此后验为 $\mathcal{N}(\mu_N, \sigma_N^2)$，其中：
- $\mu_N = 4/3 \approx 1.333$
- $\sigma_N^2 = 1/6 \approx 0.167$

**(3) 预测分布**

对 $x=3$，预测分布为：
$$P(y|x=3, data) = \int P(y|x=3,\theta)P(\theta|data)d\theta$$

这是两个高斯的卷积：
- $y|\theta \sim \mathcal{N}(3\theta, 1)$
- $\theta \sim \mathcal{N}(4/3, 1/6)$

因此：
- 均值：$E[y] = 3 \times E[\theta] = 3 \times 4/3 = 4$
- 方差：$Var(y) = 9 \times Var(\theta) + \sigma^2 = 9 \times 1/6 + 1 = 2.5$

预测分布：$y|x=3 \sim \mathcal{N}(4, 2.5)$

**考点解析：**
贝叶斯线性回归提供了预测的置信度。与点估计不同，预测分布包含了参数不确定性的信息，在新数据点远离观测数据时，不确定性会自动增加。

### 习题20.9 【难度：★★★】
**涉及知识点：** 贝叶斯网络结构学习

**题目：**
解释贝叶斯网络结构学习中的以下问题：

(1) 为什么单纯的最大似然结构学习会导致学习到一个完全连通的网络？

(2) 描述两种避免过拟合的方法

(3) 简述基于约束的方法（如PC算法）和基于评分的方法的区别

**详细解答：**

**(1) 最大似然的过拟合问题**

增加边的数量永远不会降低似然：
- 给定父节点集 $\Pi_i$，条件概率表(CPT)可以完美拟合观测到的条件频率
- 添加更多父节点只会使CPT更灵活
- 极端情况下，一个节点的所有其他节点作为父节点，可以使条件概率完美拟合数据

因此，最大化似然会选择最复杂的（完全连通）结构，导致严重过拟合。

**(2) 避免过拟合的方法**

**方法一：MDL/MAP方法**
- 对模型复杂度进行惩罚
- 评分函数：$Score(G) = \log P(Data|G) - \text{Penalty}(G)$
- 惩罚项与网络参数数量成正比

**方法二：贝叶斯方法**
- 对网络结构赋予先验概率
- 评分函数：$P(G|Data) \propto P(Data|G)P(G)$
- 使用MCMC等方法在结构空间采样

**方法三：约束方法**
- 使用条件独立性检验
- 只有当数据强烈拒绝独立性假设时才添加边

**(3) 两类方法的区别**

| 特性 | 基于约束的方法 | 基于评分的方法 |
|:---:|:---:|:---:|
| **策略** | 检验条件独立性 | 搜索评分最优的结构 |
| **输出** | 一个或多个等价类 | 单个最佳结构或后验分布 |
| **统计基础** | 假设检验 | 贝叶斯或信息论 |
| **优势** | 计算效率；理论基础清晰 | 可以整合先验知识；评分可比较不同结构 |
| **代表算法** | PC算法, IC算法 | K2算法, 爬山算法, MCMC |

**考点解析：**
结构学习是贝叶斯网络中最具挑战性的问题之一。理解过拟合问题和不同学习方法的权衡对于实际应用至关重要。

### 习题20.10 【难度：★★★★】
**涉及知识点：** 隐马尔可夫模型、EM算法

**题目：**
考虑一个两状态的隐马尔可夫模型，用于建模DNA序列中的CpG岛。
状态：$S_1$（CpG岛内），$S_2$（CpG岛外）
观测：A, C, G, T

假设我们有以下观测序列："CGC"，使用EM算法思想：

(1) 描述HMM的参数
(2) 如果初始参数给定，描述E步如何计算状态序列的期望
(3) 描述M步如何更新转移概率和发射概率

**详细解答：**

**(1) HMM参数**

**初始概率**：
- $\pi_1 = P(X_1 = S_1)$
- $\pi_2 = P(X_1 = S_2)$

**转移概率**（$a_{ij} = P(X_{t+1}=S_j|X_t=S_i)$）：
- $a_{11}, a_{12}, a_{21}, a_{22}$

**发射概率**（$b_i(k) = P(O_t=k|X_t=S_i)$）：
- $b_1(A), b_1(C), b_1(G), b_1(T)$（CpG岛内各碱基概率）
- $b_2(A), b_2(C), b_2(G), b_2(T)$（CpG岛外各碱基概率）

**(2) E步：前向-后向算法**

**前向概率**：
$$\alpha_t(i) = P(O_1, ..., O_t, X_t=S_i)$$

**后向概率**：
$$\beta_t(i) = P(O_{t+1}, ..., O_T|X_t=S_i)$$

**状态占用概率**：
$$\gamma_t(i) = P(X_t=S_i|O) = \frac{\alpha_t(i)\beta_t(i)}{\sum_j \alpha_t(j)\beta_t(j)}$$

**状态转移概率**：
$$\xi_t(i,j) = P(X_t=S_i, X_{t+1}=S_j|O)$$

$$= \frac{\alpha_t(i)a_{ij}b_j(O_{t+1})\beta_{t+1}(j)}{P(O)}$$

**(3) M步：参数更新**

**转移概率**：
$$\hat{a}_{ij} = \frac{\sum_{t=1}^{T-1}\xi_t(i,j)}{\sum_{t=1}^{T-1}\gamma_t(i)}$$

**发射概率**：
$$\hat{b}_i(k) = \frac{\sum_{t=1}^{T} \mathbb{1}(O_t=k)\gamma_t(i)}{\sum_{t=1}^{T}\gamma_t(i)}$$

**解释**：
- 转移概率：从状态i到j的期望转移次数 / 在状态i的期望停留次数
- 发射概率：在状态i发射符号k的期望次数 / 在状态i的期望停留次数

**考点解析：**
Baum-Welch算法（HMM的EM算法）是生物信息学等领域的核心技术。前向-后向算法高效计算隐变量的后验分布，使EM迭代成为可能。

### 习题20.11 【难度：★★★★★】
**涉及知识点：** 贝叶斯模型平均、模型选择

**题目：**
讨论以下高级主题：

(1) 解释贝叶斯模型平均(Bayesian Model Averaging, BMA)与MAP模型选择的区别

(2) 为什么在实际应用中BMA往往难以实现？

(3) 描述一种近似BMA的方法

**详细解答：**

**(1) BMA vs MAP**

**MAP模型选择**：
- 选择单个"最佳"模型 $M^* = \arg\max_M P(M|Data)$
- 所有预测基于 $M^*$
- 忽略模型不确定性

**贝叶斯模型平均**：
- 对所有可能的模型进行加权平均
- 预测：$P(Y|Data) = \sum_M P(Y|M,Data)P(M|Data)$
- 权重是模型的后验概率
- 显式考虑模型不确定性

**数学对比**：
- MAP：$P(Y|Data) \approx P(Y|M_{MAP},Data)$
- BMA：$P(Y|Data) = \sum_M P(Y|M,Data)P(M|Data)$

**(2) BMA的实现困难**

**计算困难**：
- 模型空间通常是超指数级的
- 对所有模型求和/积分通常不可行

**先验指定**：
- 需要为每个模型指定先验概率
- 模型先验的选择往往主观

**模型间比较**：
- 不同模型可能有不同参数空间
- 边缘似然计算困难

**(3) 近似方法**

**MCMC方法**：
- 在模型空间上进行MCMC采样
- 只采样高后验概率的模型
- 用采样频率近似模型后验

**Occam's Razor近似**：
- 只考虑模型的一小部分（如得分最高的k个）
- 或只考虑某个邻域内的模型

**贝叶斯信息准则(BIC)近似**：
$$P(M|Data) \approx \exp(-BIC_M/2)$$

**stacking（堆叠）**：
- 使用交叉验证学习模型权重
- 不是严格贝叶斯，但实践效果好

**考点解析：**
模型平均是处理模型不确定性的原则性方法。虽然在实践中面临计算挑战，但理解其思想对于统计建模的严谨性很重要。近似方法使BMA在大规模问题上成为可能。

## 公式速查表

### 贝叶斯学习
$$P(h|d) = \frac{P(d|h)P(h)}{P(d)}$$

### MAP与ML
$$h_{MAP} = \arg\max_h P(d|h)P(h)$$
$$h_{ML} = \arg\max_h P(d|h)$$

### Beta分布
$$Beta(\theta;a,b) = \alpha\theta^{a-1}(1-\theta)^{b-1}$$

### 高斯分布
$$\mathcal{N}(x|\mu,\sigma^2) = \frac{1}{\sqrt{2\pi\sigma^2}}\exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$$

### EM算法
**E步**：$q(Z) = P(Z|X,\theta^{(t)})$

**M步**：$\theta^{(t+1)} = \arg\max_\theta E_q[\log P(X,Z|\theta)]$

### 混合高斯E步
$$\gamma_{ij} = \frac{w_i \mathcal{N}(x_j|\mu_i,\Sigma_i)}{\sum_k w_k \mathcal{N}(x_j|\mu_k,\Sigma_k)}$$

### 混合高斯M步
$$\mu_i = \frac{\sum_j \gamma_{ij}x_j}{\sum_j \gamma_{ij}}$$

$$\Sigma_i = \frac{\sum_j \gamma_{ij}(x_j-\mu_i)(x_j-\mu_i)^T}{\sum_j \gamma_{ij}}$$

$$w_i = \frac{\sum_j \gamma_{ij}}{N}$$
