# 25.4 图像分类

## 背景与动机

### 什么是图像分类？

**定义**：给定一张图像，预测图像所属的类别。

**两种主要场景**：

| 场景 | 特点 | 示例 |
|------|------|------|
| 物体图像 | 图像主体为单个物体，背景不重要 | 服装、家具分类 |
| 场景图像 | 图像包含多个物体和复杂背景 | 草原、客厅分类 |

### 为什么困难？

同一类别的物体外观差异巨大，同一物体在不同条件下看起来也不同：

```
同一物体（猫）的不同外观：
┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐
│  黑猫   │  │  橘猫   │  │  白猫   │  │  花猫   │
│  (颜色) │  │  (颜色) │  │  (颜色) │  │  (颜色) │
└─────────┘  └─────────┘  └─────────┘  └─────────┘
```

## 外观变化因素

### 六大变化因素

```
┌─────────────────────────────────────────────────────┐
│                   外观变化因素                        │
├─────────────┬───────────────────────────────────────┤
│  1. 照明     │ 改变图像亮度和颜色                      │
├─────────────┼───────────────────────────────────────┤
│  2. 透视收缩 │ 斜视角度导致图案扭曲                    │
├─────────────┼───────────────────────────────────────┤
│  3. 视向     │ 不同观察方向物体形状不同                │
├─────────────┼───────────────────────────────────────┤
│  4. 遮挡     │ 物体部分被其他物体或自身遮挡            │
├─────────────┼───────────────────────────────────────┤
│  5. 变形     │ 物体改变自身形状（如人移动）            │
├─────────────┼───────────────────────────────────────┤
│  6. 背景干扰 │ 物体周围环境的干扰                      │
└─────────────┴───────────────────────────────────────┘
```

## 基于CNN的图像分类

### 为什么CNN有效？

**核心洞察**：图像具有两个重要性质

**性质1：局部模式包含丰富信息**
- 数字0、6、8、9中有环
- 数字4和8中有交叉
- 数字1、2、3、5、7中有尾端

**性质2：局部模式的空间关系关键**
- 数字1：两个尾端，一个在另一个上方
- 数字6：一个尾端在环的上方

### CNN的分层特征学习

```
输入图像 → 第1层 → 第2层 → 第3层 → ... → 分类结果
            ↓       ↓       ↓
          边缘    纹理    部件
         (线条)  (图案)  (眼睛)
         
第4层 → 第5层 → ...
  ↓       ↓
 物体    场景
 (脸)   (教室)
```

### 卷积层的工作原理

**卷积 + ReLU = 局部模式检测器**

```
卷积核（水平条检测器）：
┌────┬────┬────┐
│ -1 │ -1 │ -1 │   顶部暗
├────┼────┼────┤
│  0 │  0 │  0 │   中间过渡
├────┼────┼────┤
│  1 │  1 │  1 │   底部亮
└────┴────┴────┘

输入图像区域：        卷积结果：
┌────┬────┬────┐     ┌────┐
│ 10 │ 10 │ 10 │     │    │
├────┼────┼────┤  →  │大正│  ← 检测到水平条
│128 │128 │128 │     │ 值 │
├────┼────┼────┤     └────┘
│200 │200 │200 │
└────┴────┴────┘
```

### 感受野扩大

```
第1层：3×3窗口        第2层：5×5窗口        第3层：7×7窗口
┌───┐               ┌─────┐             ┌───────┐
│ ● │               │ ● ● │             │ ● ● ● │
│●●●│               │●●●●●│             │●●●●●●●│
│ ● │               │●●●●●│             │●●●●●●●│
└───┘               │ ● ● │             │●●●●●●●│
  ↑                 └─────┘             │ ● ● ● │
小模式                               └───────┘
(边缘)                  ↑                    ↑
                      中等模式              大模式
                    (纹理元素)            (物体部件)
```

## ImageNet竞赛与进展

### ImageNet数据集

**规模**：
- 1400万+ 训练图像
- 超过3万个细分类别
- 包含189个狗的子类别

**评估指标**：
- **Top-1准确率**：单次猜测正确率
- **Top-5准确率**：5次猜测中至少一次正确

### 历史进展

```
年份    Top-5准确率      里程碑
────────────────────────────────────────
2010年     70%        传统方法
2012年     85%        AlexNet (CNN革命)
2015年     96%        ResNet (超过人类)
2019年     98%        Top-1达87%
```

### 成功因素

1. **大数据**：ImageNet提供充足训练数据
2. **公开竞赛**：公平比较，获胜模型开源
3. **GPU计算**：大规模并行计算能力
4. **学习特征**：从数据自动学习，非手工设计

## 实用训练技巧

### 数据增强

**思想**：复制训练样本并稍加修改

```

┌──────┐         ┌──────┐  ┌──────┐  ┌──────┐
│  🐱  │   →     │🐱    │  │  🐱  │  │  🐱  │
│      │         │      │  │      │  │      │
└──────┘         └──────┘  └──────┘  └──────┘
                (平移)    (旋转)    (缩放)
                
                ┌──────┐  ┌──────┐
                │  🐱  │  │  🐱  │
                │      │  │      │
                └──────┘  └──────┘
```

**测试时增强**（TTA）：
1. 将测试图像多次变换
2. 对每个变换图像运行分类器
3. 投票决定最终类别

## 上下文的作用

### 什么是上下文？

**定义**：物体周围的环境信息，可能帮助或干扰识别。

**示例**：
- 猫躺在狗床上 → 分类器仍能识别"猫"
- 猫玩具、猫粮盘 → 可能帮助判断"猫"

### 上下文的双面性

```
┌─────────────────────────────────────────┐
│            上下文的作用                  │
├─────────────────┬───────────────────────┤
│     有帮助       │        有干扰          │
├─────────────────┼───────────────────────┤
│ 沙滩上的船 → 船  │ 水中的狗 → 可能误分类  │
│ 雪地里的狗 → 狗  │ 车里的猫 → 可能误分类  │
│ 盘子里的食物→食物│ 玩具熊 → 可能误分类为熊│
└─────────────────┴───────────────────────┘
```

## 常见陷阱

### 陷阱1：数据集偏差

**问题**：分类器可能学习背景而非物体

```
训练数据：                测试结果：
所有"游泳"图片都有泳池 → 新场景中河流游泳者
                         可能被误分类
```

**解决方案**：
- 多样化训练数据
- 使用领域随机化
- 关注物体而非背景

### 陷阱2：对抗样本

```
原始图像：    加微小扰动：    分类结果：
┌──────┐     ┌──────┐       "长臂猿"
│  🐼  │  +  不可见噪声  =   (99.3%置信度)
└──────┘     └──────┘
```

### 陷阱3：类别不平衡

**问题**：某些类别样本极少

**解决方案**：
- 过采样少数类
- 类别权重调整
- 使用Focal Loss

### 陷阱4：过度自信

**问题**：模型对错误预测也给出高置信度

**解决方案**：
- 温度缩放校准
- 使用集成方法
- 输出"未知"选项

## 关键公式

| 概念 | 公式 | 说明 |
|------|------|------|
| Softmax | $P(y_i) = \frac{e^{z_i}}{\sum_j e^{z_j}}$ | 类别概率 |
| 交叉熵损失 | $L = -\sum_i y_i \log(\hat{y}_i)$ | 分类损失 |
| Top-K准确率 | $\frac{\text{正确类别在K个预测中的次数}}{\text{总样本数}}$ | 评估指标 |

## 小结

1. **图像分类的挑战**：
   - 照明、透视、视向、遮挡、变形等多种变化因素
   - 需要学习对这些变化鲁棒的特征

2. **CNN的优势**：
   - 分层学习：边缘 → 纹理 → 部件 → 物体
   - 局部连接：利用图像的空间局部性
   - 权值共享：减少参数，提高泛化

3. **成功要素**：
   - 大数据（ImageNet）
   - 深网络架构
   - 数据增强
   - GPU计算

4. **注意事项**：
   - 避免学习数据集偏差
   - 注意模型的置信度校准
   - 理解上下文的作用
