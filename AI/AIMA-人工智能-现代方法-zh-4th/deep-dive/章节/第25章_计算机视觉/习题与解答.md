### 习题 25.1 图像形成

**题目**：在针孔照相机模型中，一个高度为2米的物体位于距离针孔10米处，像平面距离针孔5厘米。求该物体在像平面上的高度是多少？

**涉及知识点**：针孔相机模型、透视投影

**提示**：使用相似三角形原理。

**解答**：

根据针孔相机模型的透视投影公式：

$$y = \frac{-fY}{Z}$$

其中：
- $f = 5$ cm（焦距/像平面距离）
- $Y = 2$ m = 200 cm（物体高度）
- $Z = 10$ m = 1000 cm（物体距离）

代入计算：

$$y = \frac{-5 \times 200}{1000} = -1 \text{ cm}$$

**答案**：物体在像平面上的高度为 **1 cm**（负号表示倒像）。

**考点解析**：
- 理解透视投影的几何关系
- 掌握相似三角形的应用
- 注意单位统一

### 习题 25.2 边缘检测

**题目**：高斯平滑滤波器的标准差 $\sigma$ 对边缘检测有什么影响？应该选择大 $\sigma$ 还是小 $\sigma$？

**涉及知识点**：高斯滤波、边缘检测

**提示**：考虑平滑程度和细节保留的权衡。

**解答**：

**$\sigma$ 的影响**：

| $\sigma$ 值 | 效果 | 适用场景 |
|------------|------|----------|
| 小 ($\sigma = 1$) | 保留细节，但噪声敏感 | 高信噪比图像 |
| 中 ($\sigma = 2$) | 平衡平滑和细节 | 一般情况 |
| 大 ($\sigma \geq 3$) | 强平滑，丢失细节 | 噪声严重的图像 |

**可视化**：

```
原始边缘：        小σ平滑：       大σ平滑：
   ▓▓░░░            ▒▓▓░░░          ░▓▓▓░░
   ▓▓░░░            ▒▓▓▓░░░         ░▓▓▓▓░
   ▓▓░░░            ▒▓▓░░░          ░▓▓▓░░
   ↑锯齿            保留细节        边缘变宽
```

**结论**：
- 噪声多 → 选择较大的 $\sigma$
- 细节重要 → 选择较小的 $\sigma$
- 通常 $\sigma \in [1, 2]$ 是较好的折中

**考点解析**：
- 理解高斯平滑的作用
- 掌握参数对结果的影响
- 理解噪声与细节的权衡

### 习题 25.3 卷积神经网络

**题目**：在CNN中，为什么第一层卷积核通常检测边缘，而深层检测更复杂的模式？

**涉及知识点**：CNN特征层次、感受野

**提示**：考虑感受野随层数的变化。

**解答**：

**感受野扩大机制**：

```
第1层：3×3卷积            第2层：3×3卷积
输入：5×5                  输入：第1层输出
感受野：3×3                感受野：5×5
检测：边缘                 检测：纹理元素
    ↓                         ↓
第3层：3×3卷积            第4层：3×3卷积
感受野：7×7                感受野：9×9
检测：部件(眼睛)           检测：物体(脸)
```

**解释**：

1. **第1层**：每个神经元只看3×3区域，只能检测局部边缘
2. **第2层**：接收第1层的边缘响应，组合成纹理模式
3. **第3层+**：组合更复杂的模式，最终形成物体级别的特征

**数学关系**：

若每层使用 $k \times k$ 卷积核，则第 $n$ 层的感受野大小为：

$$RF_n = 1 + n \times (k - 1)$$

**考点解析**：
- 理解感受野概念
- 理解CNN的分层特征学习
- 理解特征组合机制

### 习题 25.4 双目立体视觉

**题目**：一个双目立体视觉系统的基线为8cm，焦距为50mm。左图中某点在x=120像素处，右图中对应点在x=105像素处（假设图像平面与相机距离为f，像素大小为0.01mm）。计算该点的深度。

**涉及知识点**：双目立体视觉、视差与深度

**提示**：先计算像素视差，再转换为实际视差。

**解答**：

**步骤1：计算像素视差**

$$d_{pixel} = 120 - 105 = 15 \text{ 像素}$$

**步骤2：转换为实际视差**

像素大小为0.01mm，所以：

$$d = 15 \times 0.01 = 0.15 \text{ mm}$$

**步骤3：计算深度**

深度公式：

$$Z = \frac{fb}{d}$$

其中：
- $f = 50$ mm
- $b = 8$ cm = 80 mm
- $d = 0.15$ mm

代入：

$$Z = \frac{50 \times 80}{0.15} = \frac{4000}{0.15} \approx 26667 \text{ mm} = 26.67 \text{ m}$$

**答案**：该点的深度约为 **26.67米**。

**考点解析**：
- 理解视差概念
- 掌握深度计算公式
- 注意单位换算

### 习题 25.6 物体检测流程分析

**题目**：解释Fast R-CNN中为什么需要先使用RPN生成候选区域，而不是直接在原始图像的每个位置进行分类？

**涉及知识点**：物体检测、RPN、计算效率

**提示**：考虑计算复杂度和窗口数量。

**解答**：

**问题分析**：

对于 $n \times n$ 的图像，可能的矩形窗口数量为 $O(n^4)$。

例如，对于1000×1000的图像：
- 窗口数量约为 $10^{12}$ 量级
- 对每个窗口运行CNN分类是不可行的

**RPN的优势**：

```
传统方法：                    RPN方法：
┌─────────────┐              ┌─────────────┐
│ 所有可能    │              │ 特征图每个  │
│ 的窗口      │              │ 位置9个锚框 │
│ (10^12)     │              │ (~10^4)     │
└──────┬──────┘              └──────┬──────┘
       ↓                           ↓
  逐一分类                    预测前景分数
       ↓                           ↓
   不可行                    选择高分区域
                                   ↓
                              只分类~100个
```

**具体计算**：

假设：
- 图像大小：1000×1000
- 特征图大小：62×62（经过多层下采样）
- 每个位置9个锚框

候选框总数：

$$62 \times 62 \times 9 \approx 35000$$

经过RPN筛选后，通常只保留100-300个高质量候选框进行分类。

**效率提升**：

$$
\text{加速比} \approx \frac{10^{12}}{10^2} = 10^{10}$$

**考点解析**：
- 理解RPN的作用
- 理解计算复杂度问题
- 理解两阶段检测器的设计动机

### 习题 25.7 三维重建

**题目**：比较双目立体视觉和运动恢复结构（SfM）的异同。它们分别适用于什么场景？

**涉及知识点**：三维重建、立体视觉、SfM

**提示**：从相机配置、几何关系、应用场景考虑。

**解答**：

**比较表**：

| 特性 | 双目立体视觉 | 运动恢复结构(SfM) |
|------|------------|------------------|
| 相机配置 | 两个固定相机 | 单个移动相机 |
| 基线 | 固定已知 | 变化，需要估计 |
| 深度精度 | 高（基线固定） | 中等（基线变化） |
| 运动要求 | 无 | 需要相机移动 |
| 计算复杂度 | 较低 | 较高 |
| 适用场景 | 实时应用（机器人、自动驾驶） | 离线重建（建模、测绘） |

**几何关系**：

```
双目立体视觉：                  SfM：
    C₁    C₂                      C₁ → C₂ → C₃
    ●─────●                       ●────●────●
   ╱│╲   ╱│╲                     ╱│╲   ╱│╲   ╱│╲
  ╱ │ ╲ ╱ │ ╲                   ╱ │ ╲ ╱ │ ╲ ╱ │ ╲
 ╱  │  ●  │  ╲                 ╱  │  ●  │  ●  │  ╲
●───┼───┼───┼───●             ●───┼───┼───┼───┼───●
    │   P   │                      场景点P
    场景点P
```

**应用选择**：

**选择双目立体视觉**：
- 需要实时深度感知（自动驾驶、机器人导航）
- 相机可以固定安装
- 深度精度要求高

**选择SfM**：
- 从已有视频/照片重建
- 需要大场景重建
- 可以离线处理
- 相机轨迹也需要估计

**考点解析**：
- 理解两种方法的几何基础
- 理解各自的优缺点
- 能够根据应用场景选择合适方法

### 习题 25.8 对抗样本分析

**题目**：解释为什么深度神经网络容易受到对抗样本的攻击？提出至少两种防御策略。

**涉及知识点**：对抗样本、神经网络鲁棒性

**提示**：考虑决策边界和高维空间特性。

**解答**：

**对抗样本的脆弱性原因**：

```
高维空间的决策边界：

         正常样本          对抗样本
            ●                ●+
           ╱│╲              ╱│╲  ← 微小扰动
          ╱ │ ╲            ╱ │ ╲
    ─────┼──┼──┼────  ────┼──┼──┼────
         ╲ │ ╱            ╲ │ ╱
          ╲│╱              ╲│╱
            ●                ●
           猫                 长臂猿
           
在高维空间中，决策边界附近有大量
对抗方向，微小扰动就能跨越类别
```

**原因分析**：

1. **高维几何**：在高维空间中，数据点之间的距离分布使得微小扰动可能跨越决策边界
2. **线性假设**：神经网络在局部近似线性，输入的微小变化导致输出的线性变化
3. **训练目标**：标准训练不强制要求鲁棒性

**防御策略**：

**策略1：对抗训练**

```
训练过程：
1. 对输入生成对抗样本
2. 用对抗样本和原始样本一起训练
3. 损失函数包含对抗损失

效果：使网络在对抗样本附近也能正确分类
```

**策略2：输入预处理**

```
方法：
• 图像压缩/去噪
• 随机变换（随机裁剪、缩放）
• 特征压缩

原理：破坏对抗扰动的精细结构
       同时保留正常分类所需信息
```

**策略3：防御蒸馏**

```
步骤：
1. 训练教师网络
2. 用教师网络的软标签训练学生网络
3. 学生网络对对抗样本更鲁棒

原理：软标签平滑了决策边界
```

**考点解析**：
- 理解对抗样本的本质
- 理解深度网络的脆弱性来源
- 掌握基本的防御方法

### 习题 25.9 深度学习 vs 传统方法

**题目**：在计算机视觉中，深度学习方法相比传统手工设计特征（如SIFT、HOG）有什么优势和劣势？

**涉及知识点**：特征工程、深度学习、方法对比

**提示**：从数据需求、计算资源、可解释性等方面分析。

**解答**：

**优势**：

```
1. 特征学习
   传统：手工设计SIFT、HOG等
   深度：从数据自动学习最优特征
   
2. 端到端优化
   传统：特征提取和分类器分开优化
   深度：联合优化，全局最优
   
3. 性能
   ImageNet上：深度学习错误率 < 传统方法
   
4. 可迁移性
   预训练模型可用于各种下游任务
```

**劣势**：

```
1. 数据需求
   传统：少量数据即可
   深度：需要大量标注数据
   
2. 计算资源
   传统：CPU即可运行
   深度：需要GPU加速
   
3. 可解释性
   传统：特征含义明确（边缘、梯度）
   深度：黑盒，难以解释
   
4. 理论基础
   传统：有坚实的数学基础
   深度：很多现象难以理论解释
```

**对比表**：

| 方面 | 传统方法 | 深度学习方法 |
|------|---------|------------|
| 数据需求 | 少 | 多 |
| 特征设计 | 手工 | 自动学习 |
| 计算需求 | 低 | 高 |
| 性能上限 | 中等 | 高 |
| 可解释性 | 好 | 差 |
| 调参难度 | 中等 | 高 |
| 领域迁移 | 困难 | 容易（微调） |

**考点解析**：
- 理解两种范式的本质区别
- 能够客观评价各自的优缺点
- 理解适用场景的差别

### 计算题（30分）

**题目**：一个CNN网络的输入为224×224的RGB图像。第一层是卷积层，使用64个7×7的卷积核，步长为2，padding为3。计算：

(1) 输出特征图的空间尺寸  
(2) 这一层的参数量  
(3) 相比全连接层，卷积层的主要优势是什么

**参考答案**：

**(1) 输出尺寸**：

$$H_{out} = \frac{224 - 7 + 2 \times 3}{2} + 1 = 112$$

输出尺寸：112 × 112 × 64

**(2) 参数量**：

$$(7 \times 7 \times 3 + 1) \times 64 = 148 \times 64 = 9472$$

（+1是偏置项）

**(3) 卷积层优势**：
- 局部连接：参数共享，减少参数量
- 平移不变性：同一特征在不同位置检测
- 保留空间结构：适合图像数据
