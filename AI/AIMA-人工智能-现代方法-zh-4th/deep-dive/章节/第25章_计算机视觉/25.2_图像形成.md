# 25.2 图像形成

## 成像的畸变效应

### 透视投影的视觉效果

成像过程会改变物体的外观：

1. **铁轨效应**：平行线在图像中似乎交汇
2. **遮挡效应**：近处物体可以遮挡远处更大的物体
3. **透视收缩**：物体倾斜时产生形变

**示例**：
```
真实世界：    图像中：
 ┃ ┃ ┃       ╲   ╱
 ┃ ┃ ┃    →   ╲ ╱
 ┃ ┃ ┃         ╳
```

## 针孔相机模型

### 无透镜成像

**针孔相机原理**：

```
        物体P
          \
           \
    O-------●-------图像平面
   针孔             P'
```

- 针孔足够小，只允许特定方向的光线通过
- 每个传感器接收来自物体上特定点的光线
- 形成清晰的（但较暗的）图像

### 透视投影方程

建立坐标系：
- 原点O：针孔位置
- Z轴：光轴方向
- 图像平面：距离针孔f（焦距）

**投影公式**：

$$x = -f\frac{X}{Z}, \quad y = -f\frac{Y}{Z}$$

**推导**（相似三角形）：

```
       P(X,Y,Z)
         |\
         | \
         |  \
    -----O---|---图像平面
        |   |P'(x,y)
        |  /
        | / f
        |/
```

由相似三角形：$\frac{-x}{f} = \frac{X}{Z}$

### 关键性质

1. **距离与大小成反比**：
   - 远处物体成像小
   - 近处物体成像大

2. **图像倒置**：
   - 负号表示上下左右颠倒
   - $x = -f\frac{X}{Z}$ 中，X>0 对应 x<0

3. **深度信息丢失**：
   - 只知道 $X/Z$ 和 $Y/Z$
   - 无法单独确定X, Y, Z

### 消失点

**定义**：空间中平行线在图像中的交点。

**计算**：

方向向量 $(U, V, W)$ 的平行线族，其消失点为：

$$P_\infty = \left(f\frac{U}{W}, f\frac{V}{W}\right)$$

**示例**：
- 水平线（方向 $(1, 0, 0)$）：消失点在水平方向无穷远
- 铁轨（方向 $(0, 0, 1)$）：消失点在图像中心（如果相机水平）

## 透镜系统

### 针孔相机的局限

**问题**：
- 针孔小 → 进光少 → 图像暗
- 针孔大 → 成像模糊

### 透镜的作用

**解决方案**：使用透镜收集更多光线并聚焦。

```
       物体
         \
          \
    [======透镜======]
            \
             \
              ● 聚焦点
```

**透镜公式**：

$$\frac{1}{f} = \frac{1}{u} + \frac{1}{v}$$

其中：
- $f$：焦距
- $u$：物距
- $v$：像距

### 景深

**定义**：能够清晰成像的物体深度范围。

**影响因素**：
- **光圈大小**：光圈越大，景深越小
- **焦距**：焦距越长，景深越小
- **拍摄距离**：距离越近，景深越小

**应用**：
- 大光圈（f/1.8）：背景虚化（人像摄影）
- 小光圈（f/16）：全景清晰（风景摄影）

## 缩放正交投影

### 动机

透视投影的完整计算有时过于复杂，当物体深度变化较小时可以简化。

### 近似条件

当物体上所有点的深度 $Z$ 满足：

$$Z = Z_0 \pm \Delta Z, \quad \Delta Z \ll Z_0$$

### 投影方程

令 $s = f/Z_0$，则：

$$x = sX, \quad y = sY$$

**特点**：
- 无透视畸变（平行线保持平行）
- 保留缩放和旋转效果
- 计算简单

**应用场景**：
- 航空摄影（地面起伏远小于飞行高度）
- 医学成像
- 工业检测

## 光线与明暗

### 图像亮度的来源

像素亮度取决于：
1. **环境光强度**
2. **表面朝向**（向光面 vs 背光面）
3. **表面反射特性**

### 反射类型

#### 漫反射（Diffuse Reflection）

**特点**：
- 光线均匀散射到各个方向
- 亮度与观察方向无关
- 例子：纸张、布料、粗糙表面

**兰伯特定律**：

$$I = \rho I_0 \cos\theta$$

其中：
- $I$：表面亮度
- $\rho$：漫反射系数（0.05 ~ 0.95）
- $I_0$：入射光强度
- $\theta$：光源方向与表面法向夹角

#### 镜面反射（Specular Reflection）

**特点**：
- 光线按特定角度反射
- 产生高光（亮斑）
- 例子：镜子、金属、光滑塑料

**镜面反射亮斑（Specularity）**：
- 小而亮的区域
- 位置随光源或观察者移动

### 光照模型

**远点光源假设**（如太阳）：

- 所有光线平行
- 方向已知

**亮度计算**：

$$I = \rho I_0 \max(0, \mathbf{n} \cdot \mathbf{l}) + I_{ambient}$$

其中：
- $\mathbf{n}$：表面法向
- $\mathbf{l}$：光源方向
- $I_{ambient}$：环境光

## 颜色

### 人类色觉

**三原色原理**（托马斯·扬，1802）：

人类通过三种视锥细胞感知颜色，适当混合三种原色可以产生任何颜色的视觉效果。

**RGB三原色**：
- R：红色（~700nm）
- G：绿色（~546nm）
- B：蓝色（~435nm）

### 颜色表示

**RGB模型**：
- 每个像素由3个数值表示
- 每个通道通常8位（0-255）
- 总共约1600万种颜色

**颜色恒常性（Color Constancy）**：

人类能够在不同光照条件下感知物体的"真实"颜色：
- 白天看白纸是白色
- 傍晚看同一张纸仍是白色（尽管光线偏黄）

### 计算机视觉中的颜色

**简化模型**：
- 表面有RGB三个漫反射系数
- 光源有RGB三个强度值
- 分别应用兰伯特定律

**颜色应用**：
- 图像分割
- 物体识别
- 果实成熟度判断

## 关键公式总结

| 概念 | 公式 | 说明 |
|------|------|------|
| 透视投影 | $x = -fX/Z$ | 3D到2D投影 |
| 消失点 | $P_\infty = (fU/W, fV/W)$ | 平行线交点 |
| 透镜公式 | $1/f = 1/u + 1/v$ | 成像关系 |
| 兰伯特定律 | $I = \rho I_0 \cos\theta$ | 漫反射亮度 |
| 缩放正交 | $x = sX$ | 简化投影 |

## 小结

图像形成是计算机视觉的物理基础：

1. **针孔相机**：
   - 最简单的成像模型
   - 透视投影产生几何畸变
   - 理解投影方程是3D视觉的基础

2. **透镜系统**：
   - 收集更多光线，形成清晰图像
   - 景深的概念在摄影中至关重要

3. **光照与反射**：
   - 漫反射和镜面反射是两种基本类型
   - 兰伯特定律描述漫反射亮度
   - 阴影提供形状信息

4. **颜色**：
   - RGB模型基于人类三原色视觉
   - 颜色恒常性是人类视觉的重要特性

5. **实际应用**：
   - 这些原理指导相机标定
   - 是3D重建的理论基础
   - 影响图像处理和计算机视觉算法设计
