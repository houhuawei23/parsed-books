# 25.6 三维世界

## 背景与动机

### 二维图像 vs 三维世界

图像是三维世界的二维投影，但包含了丰富的三维信息：

```
三维世界：                    二维图像：
    z                          ┌─────────────┐
    ↑                         │             │
    │    ● P(X,Y,Z)           │     ● p(x,y)│
    │   ╱                     │             │
    │  ╱                      └─────────────┘
    │ ╱  摄像机
    ●────────────────→ x
    \
     ↓ y

从一张或多张图像恢复三维信息是计算机视觉的核心问题
```

### 三维重建的意义

| 应用 | 需求 |
|------|------|
| 机器人导航 | 知道物体在哪里，距离多远 |
| 增强现实 | 将虚拟物体正确放置在真实场景中 |
| 自动驾驶 | 理解周围环境的3D结构 |
| 医学影像 | 重建器官的三维模型 |

## 多视图几何

### 为什么需要多个视图？

**单个视图的问题**：
- 丢失深度信息
- 无法确定物体的真实大小

**两个视图的优势**：

```
左视图：              右视图：
┌─────────┐         ┌─────────┐
│    ●    │         │    ●    │
│   ╱│╲   │         │   ╱│╲   │
│  ╱ │ ╲  │         │  ╱ │ ╲  │
│ ●──┼──● │         │ ●──┼──● │
│    │    │         │    │    │
└─────────┘         └─────────┘
     ↓                   ↓
    点P(xₗ,yₗ)        点P(xᵣ,yᵣ)
    
通过对应点 (xₗ,yₗ) ↔ (xᵣ,yᵣ) 可以恢复深度
```

### 基本几何关系

**本质矩阵（Essential Matrix）**：

$$E = [t]_\times R$$

其中：
- $R$：两相机间的旋转矩阵
- $t$：平移向量
- $[t]_\times$：平移向量的反对称矩阵

**极线约束**：

```
左图像：              右图像：
┌─────────┐         ┌─────────┐
│         │         │         │
│    ●    │────────→│─────────│
│         │  极线   │    ●    │
│         │         │         │
└─────────┘         └─────────┘

左图中的点P，在右图中必位于对应的极线上
```

数学表达：

$$p_r^T E p_l = 0$$

其中 $p_l, p_r$ 是对应点的齐次坐标。

## 双目立体视觉

### 视差与深度

**视差（Disparity）**：同一场景点在左右图像中的水平位移。

```
左视图：              右视图：
┌─────────┐         ┌─────────┐
│  ●      │         │      ●  │
│  Pₗ     │         │     Pᵣ  │
└─────────┘         └─────────┘
    xₗ = 100            xᵣ = 150
    
视差 d = xₗ - xᵣ = -50（像素）
```

**深度公式**：

$$Z = \frac{fb}{d}$$

其中：
- $Z$：深度（物体到相机的距离）
- $f$：焦距
- $b$：基线（两眼/两相机间距）
- $d$：视差

**重要性质**：视差与深度成反比

```
深度 vs 视差：

深度Z：  1m ────── 2m ────── 5m ────── ∞
         │         │         │         │
视差d：  大        中        小        零
```

### 人类立体视觉

**参数**：
- 基线 $b \approx 6$ cm
- 焦距 $f \approx 17$ mm
- 可分辨最小视差 $\delta\theta \approx 5$ 角秒

**深度分辨能力**：

$$\delta Z = \frac{Z^2 \delta\theta}{b}$$

```
在Z = 30cm处：δZ ≈ 0.036mm  （可穿针）
在Z = 1m处：  δZ ≈ 0.4mm
在Z = 10m处： δZ ≈ 40mm
```

### 对应问题

**挑战**：找到左图中每一点在右图中的对应点。

```
左图块：              右图候选块：
┌───────┐            ┌───────┐ ┌───────┐ ┌───────┐
│ ▓▓░░░ │            │ ░░░▓▓ │ │ ▓▓░░░ │ │ ░▓▓░░ │
│ ▓▓░░░ │     →      │ ░░░▓▓ │ │ ▓▓░░░ │ │ ░▓▓░░ │
│ ▓▓░░░ │            │ ░░░▓▓ │ │ ▓▓░░░ │ │ ░▓▓░░ │
└───────┘            └───────┘ └───────┘ └───────┘
                      (错误)    (正确)    (错误)
```

**SSD匹配**：

$$SSD(d) = \sum_{(x,y)}[I_L(x,y) - I_R(x+d, y)]^2$$

**约束条件**：
- 唯一性：左图一点对应右图一点
- 连续性：相邻点视差相近
- 顺序性：左右顺序一致

## 运动恢复结构

### 移动相机的三维重建

**思想**：相机移动相当于在不同位置拍摄多幅图像。

```
时刻t：            时刻t+1：
    C                 C'
    ●                 ●
   ╱│╲               ╱│╲
  ╱ │ ╲      →      ╱ │ ╲
 ╱  │  ╲           ╱  │  ╲
●───┼───●         ●───┼───●
   场景              场景
   
运动：平移T + 旋转R
```

### 光流与深度

**光流方程**：

$$v_x = \frac{-T_x + xT_z}{Z}, \quad v_y = \frac{-T_y + yT_z}{Z}$$

**延伸焦点（Focus of Expansion）**：

```
相机向前运动时：

        ●
       ╱│╲
      ╱ │ ╲
     ╱  │  ╲
    ●───●───●  ← 所有光流从中心点向外辐射
     \  │  ╱       (延伸焦点)
      \ │ ╱
       \│╱
```

**应用**：

1. **接触时间**：$\tau = Z/T_z$
   - 苍蝇着陆时使用

2. **深度比**：通过光流比确定
   $$\frac{Z_1}{Z_2} = \frac{v_2}{v_1}$$

## 单视图三维线索

### 从单张图像恢复三维

即使只有一张图像，也能获得三维信息：

```
单视图线索：
┌─────────────────────────────────────────┐
│  1. 遮挡     →  遮挡者在前面             │
│  2. 纹理梯度  →  远处纹理元素更小        │
│  3. 明暗     →  表面朝向信息             │
│  4. 透视     →  平行线汇聚               │
│  5. 熟悉物体  →  已知大小推断距离        │
│  6. 地平线   →  地面上的物体位置         │
└─────────────────────────────────────────┘
```

### 纹理梯度

```
均匀的鹅卵石地面：       图像中的样子：
俯视图：                透视图：
┌──┬──┬──┬──┐         ┌──┬──┬──┬──┐
│○ │○ │○ │○ │         │○ │○ │○ │○ │  ← 近处，大
├──┼──┼──┼──┤         ├──┼──┼──┼──┤
│○ │○ │○ │○ │    →    │ ●│ ●│ ●│ ●│
├──┼──┼──┼──┤         ├──┼──┼──┼──┤
│○ │○ │○ │○ │         │ ·│ ·│ ·│ ·│  ← 远处，小
└──┴──┴──┴──┘         └──┴──┴──┴──┘
```

### 明暗与形状

**兰伯特余弦定律**：

$$I = \rho I_0 \cos\theta$$

其中：
- $I$：观测亮度
- $\rho$：反射率
- $I_0$：光源强度
- $\theta$：表面法线与光源方向夹角

```
从明暗推断形状：

光源方向 ↗
   
   ╭─────╮
  ╱       ╲
 │    ●    │   ← 最亮（法线指向光源）
 │   ╱│╲   │
  ╲ ╱  ╲ ╱
   ╰─────╯
   
较暗区域：法线远离光源
```

### 物体位姿估计

**问题**：已知物体模型，从单张图像估计其位置和方向。

```
3D模型点：      2D图像点：
P₁(X₁,Y₁,Z₁)   →   p₁(x₁,y₁)
P₂(X₂,Y₂,Z₂)   →   p₂(x₂,y₂)
P₃(X₃,Y₃,Z₃)   →   p₃(x₃,y₃)
...

求解：旋转R和平移t，使得投影误差最小
```

**PnP问题**（Perspective-n-Point）：
- 3个点：最多4个解
- 4个共面点：唯一解
- 6个点以上：线性求解

## 深度学习三维重建

### 单视图深度估计

**方法**：训练CNN从图像预测深度图

```
输入图像：                输出深度图：
┌─────────────┐          ┌─────────────┐
│   🏠        │    →     │ ████░░░░░░░ │
│  ╱  ╲       │  CNN     │ ████████░░░ │
│ ╱    ╲      │          │ ███████████ │
│╱──────╲     │          │ ███████████ │
│        │    │          │ ███████████ │
└─────────────┘          └─────────────┘
                          (深色=近，浅色=远)
```

**损失函数**：

$$L = \sum_{ij}|\log D_{ij} - \log D^*_{ij}|$$

### 多视图立体（MVS）

```
多张图像 → 特征提取 → 匹配代价体 → 深度估计
             ↓
         三维点云
```

**应用**：
- 城市建模
- 文物数字化
- 自动驾驶高精地图

## 常见陷阱

### 陷阱1：尺度歧义

**问题**：从图像无法确定绝对尺度。

```
情况1：              情况2：
小物体 + 近距离      大物体 + 远距离
┌───────┐            ┌───────┐
│  🚗   │            │  🚗   │  ← 同样大小！
│       │            │       │
└───────┘            └───────┘
玩具车(1m)           真车(50m)
```

**解决方案**：需要额外信息（已知大小的物体、多视图等）。

### 陷阱2：纹理缺失区域

```
白墙：               天空：
┌─────────┐         ┌─────────┐
│ ░░░░░░░ │         │ ░░░░░░░ │
│ ░░░░░░░ │         │ ░░░░░░░ │
│ ░░░░░░░ │         │ ░░░░░░░ │
└─────────┘         └─────────┘

无法找到对应点！
```

### 陷阱3：反光和透明表面

```
镜子：               玻璃：
反射景象            透射背景
     ↓               ↓
错误的深度估计      缺失的检测
```

### 陷阱4：动态场景

```
静态假设下的错误：

实际：人移动了      假设：静态场景
┌─────────┐        ┌─────────┐
│    🚶   │        │    ?    │
│   →     │        │  错误   │
└─────────┘        └─────────┘
```

## 关键公式总结

| 概念 | 公式 | 说明 |
|------|------|------|
| 深度-视差 | $Z = fb/d$ | 双目立体视觉 |
| 光流 | $v_x = \frac{-T_x + xT_z}{Z}$ | 运动恢复结构 |
| 极线约束 | $p_r^T E p_l = 0$ | 多视图几何 |
| 兰伯特定律 | $I = \rho I_0 \cos\theta$ | 明暗与形状 |

## 小结

1. **多视图几何**：
   - 两个视图可以恢复三维结构
   - 关键是对应问题
   - 视差与深度成反比

2. **立体视觉**：
   - 人类基线约6cm
   - 近距离深度分辨率高
   - 受限于纹理丰富区域

3. **运动恢复结构**：
   - 移动相机 = 多视图
   - 光流包含深度信息
   - 可计算接触时间和深度比

4. **单视图线索**：
   - 遮挡、纹理、明暗、透视
   - 结合先验知识可推断3D
   - 深度估计有尺度歧义

5. **深度学习方法**：
   - 单视图深度估计
   - 多视图立体
   - 端到端三维重建
