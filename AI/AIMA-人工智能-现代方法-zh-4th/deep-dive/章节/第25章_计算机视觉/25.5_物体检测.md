# 25.5 物体检测

## 背景与动机

### 图像分类 vs 物体检测

```
图像分类：                        物体检测：
┌─────────────┐                  ┌─────────────┐
│    🐱       │  → "猫"          │  ┌───┐      │
│             │                  │  │🐱 │  🐶   │
│  背景       │                  │  └───┘ ┌──┐ │
│             │                  │        │🐶│ │
└─────────────┘                  │        └──┘ │
                                 └─────────────┘
                              → "猫在(100,150)，
                                 狗1在(300,200)，
                                 狗2在(400,350)"
```

**物体检测任务**：
- 找出图像中所有感兴趣的物体
- 确定每个物体的类别
- 定位每个物体的位置（边界框）

## 滑动窗口方法

### 基本思想

```
在图像上滑动不同大小的窗口，
对每个窗口进行分类：

┌─────────────────────────┐
│ □ □ □ □ □ □ □ □ □ □ □ □ │
│ □ □ □ ┌───┐ □ □ □ □ □ □ │
│ □ □ □ │🐱 │ □ □ □ □ □ □ │
│ □ □ □ └───┘ □ □ □ □ □ □ │
│ □ □ □ □ □ □ □ □ □ □ □ □ │
│ □ □ □ □ □ □ □ ┌───┐ □ □ │
│ □ □ □ □ □ □ □ │🐶 │ □ □ │
│ □ □ □ □ □ □ □ └───┘ □ □ │
└─────────────────────────┘

每个窗口 → CNN分类器 → 类别分数
```

### 关键问题

| 问题 | 描述 | 解决方案 |
|------|------|----------|
| 窗口形状 | 矩形还是掩码？ | 轴对齐矩形 |
| 窗口尺寸 | 不同物体大小不同 | 多尺度检测 |
| 窗口数量 | $O(n^4)$个可能窗口 | 区域候选网络 |
| 重叠窗口 | 多个窗口覆盖同一物体 | 非极大值抑制 |
| 精确定位 | 粗糙窗口不够精确 | 边框回归 |

## 区域候选网络（RPN）

### 锚框（Anchor Boxes）

**思想**：在每个位置预定义一组候选框

```
每个中心点有9个锚框：

        小              中              大
      ┌───┐         ┌─────┐         ┌───────┐
高    │   │         │     │         │       │
      └───┘         │     │         │       │
                  ┌─┴─────┴─┐       │       │
宽    ┌─────────┐ │         │       │       │
      │         │ └─────────┘       │       │
      └─────────┘               ┌───┴───────┴───┐
                                │               │
方    ┌─────┐   ┌───────┐       │               │
      │     │   │       │       │               │
      └─────┘   └───────┘       └───────────────┘
```

**Fast R-CNN使用的锚框**：
- 3种尺寸：小、中、大
- 3种纵横比：高、宽、正方形
- 共9个锚框

### RPN工作流程

```
输入图像 → CNN特征提取 → RPN → 感兴趣区域(ROI)
                              ↓
                        ┌──────────┐
                        │ 前景分数  │  ← 包含物体的概率
                        │ 边界框修正 │  ← 调整锚框
                        └──────────┘
```

**输出**：高分的候选框（感兴趣区域）

## 多任务学习框架

### Fast R-CNN架构

```
输入图像
    ↓
┌─────────────┐
│ CNN特征提取  │  ← VGG/ResNet等骨干网络
│   (共享)     │
└─────────────┘
    ↓
    ├──────────────┐
    ↓              ↓
┌─────────┐   ┌──────────┐
│   RPN   │   │  ROI池化  │
│ (区域提议)│   │ (统一尺寸)│
└─────────┘   └──────────┘
                   ↓
            ┌──────────────┐
            │  全连接层     │
            └──────────────┘
                   ↓
           ┌───────┴───────┐
           ↓               ↓
      ┌─────────┐     ┌─────────┐
      │ 类别分类 │     │ 边框回归 │
      │(是什么)  │     │(精确位置) │
      └─────────┘     └─────────┘
```

### 损失函数

**多任务损失**：

$$L = L_{cls} + \lambda L_{reg}$$

其中：
- $L_{cls}$：分类损失（交叉熵）
- $L_{reg}$：边框回归损失（Smooth L1）
- $\lambda$：平衡系数

## ROI池化

### 问题

CNN分类器需要固定尺寸输入，但候选框尺寸各异。

### 解决方案

```
不同尺寸的ROI → ROI池化 → 固定尺寸特征图

ROI 1 (100×200):    ROI 2 (50×80):
┌──────────┐        ┌──────┐
│          │        │      │
│          │        └──────┘
└──────────┘
       ↓ ROI池化
┌────────┐          ┌────────┐
│ ■ ■ ■ ■│          │ ■ ■ ■ ■│
│ ■ ■ ■ ■│          │ ■ ■ ■ ■│
│ ■ ■ ■ ■│          │ ■ ■ ■ ■│
│ ■ ■ ■ ■│          │ ■ ■ ■ ■│
└────────┘          └────────┘
(固定7×7输出)
```

**操作**：将ROI均匀划分为固定网格，对每个网格做最大池化。

## 非极大值抑制（NMS）

### 问题

多个重叠的窗口可能检测到同一物体。

```
原始检测：                期望结果：
┌─────────────┐          ┌─────────────┐
│ ┌───┐       │          │ ┌───┐       │
│ │🐱 │       │    →     │ │🐱 │       │
│ │┌─┐│       │          │ └───┘       │
│ │└─┘│       │          │             │
│ └───┘       │          │             │
└─────────────┘          └─────────────┘
(3个框检测到同一只猫)      (只保留最佳框)
```

### NMS算法

```
算法：非极大值抑制
─────────────────────────
输入：检测框集合B，分数S，阈值Nt
输出：保留的检测框D

1. D ← ∅
2. 按分数S排序B
3. while B ≠ ∅ do
4.   m ← argmax(S)     // 最高分框
5.   D ← D ∪ {m}
6.   B ← B - {m}
7.   for b ∈ B do
8.     if IoU(m,b) > Nt then
9.       B ← B - {b}   // 移除重叠框
10.      S ← S - {s_b}
11. return D
```

### IoU（交并比）

```
          交集区域
┌─────────────────────┐
│    ┌───────┐        │
│    │  ∩   │        │
│    │   ┌───┼────┐   │
│    └───┼───┘    │   │
│        └───────┘    │
│       并集区域       │
└─────────────────────┘

IoU = 交集面积 / 并集面积

阈值通常设为0.5或0.7
```

## 边框回归

### 问题

候选框可能不够精确，需要微调。

### 边框参数化

```
候选框(锚框)：            真实框：
┌────────────┐          ┌────────────┐
│  (ax,ay)   │          │  (gx,gy)   │
│     ●──────┼──┐       │     ●──────┼──┐
│     │      │  │   →   │     │      │  │
│     │   aw │  │       │     │   gw │  │
│     │   ×  │ah │       │     │   ×  │gh │
│     │      │  │       │     │      │  │
│     └──────┴──┘       │     └──────┴──┘
│                          │
└────────────┘          └────────────┘

需要学习：中心点偏移(tx,ty)和尺度变换(tw,th)
```

### 回归目标

$$t_x = \frac{g_x - a_x}{a_w}, \quad t_y = \frac{g_y - a_y}{a_h}$$

$$t_w = \log\frac{g_w}{a_w}, \quad t_h = \log\frac{g_h}{a_h}$$

## 评估指标

### 平均精度（AP）

```
精确率-召回率曲线：

精确率
  1.0 ┤     ┌──────
      │    ╱
  0.8 ┤   ╱
      │  ╱
  0.6 ┤─●────────
      │
  0.0 ┼────┬────┬──→ 召回率
      0   0.5  1.0

AP = 曲线下面积
```

**mAP（平均精度均值）**：
- 所有类别AP的平均值
- 常用IoU阈值：0.5（mAP@0.5）

### 检测结果评估

```
真阳性(TP)：检测框与真实框IoU > 阈值
假阳性(FP)：检测框与真实框IoU ≤ 阈值（或没有匹配）
假阴性(FN)：真实框没有被检测到

精确率 = TP / (TP + FP)
召回率 = TP / (TP + FN)
```

## 现代检测器演进

### 两阶段检测器

```
R-CNN系列演进：

R-CNN (2014):        Fast R-CNN (2015):
  选择性搜索 →       整张图像CNN特征 →
  CNN特征提取  →    ROI池化 →
  SVM分类           联合训练

Faster R-CNN (2015):
  RPN + Fast R-CNN 端到端训练
```

### 单阶段检测器

```
YOLO/SSD：

输入图像 → CNN特征 → 直接预测
                      ├─ 类别概率
                      ├─ 边界框坐标
                      └─ 置信度分数

特点：速度更快，精度稍低
```

## 常见陷阱

### 陷阱1：小物体检测困难

**原因**：
- 深层CNN特征图分辨率低
- 小物体可能只占几个像素

**解决方案**：
- 特征金字塔网络（FPN）
- 多尺度训练/测试

### 陷阱2：密集遮挡

```
拥挤场景：
┌─────────────────┐
│ 👤👤👤👤👤👤👤👤 │
│ 人群相互遮挡      │
│ 难以区分个体      │
└─────────────────┘
```

**解决方案**：
- 实例分割辅助
- 关系网络建模

### 陷阱3：类别不平衡

**问题**：背景框远多于物体框。

**解决方案**：
- 困难负样本挖掘
- Focal Loss

### 陷阱4：定位不准

```
定位误差类型：

边界偏移：          尺度错误：
┌─────┐            ┌───────┐
│🐱   │   vs       │ 🐱    │
│     │            └───────┘
└─────┘            
(框偏左)           (框太大)
```

## 关键公式总结

| 概念 | 公式 | 说明 |
|------|------|------|
| IoU | $\frac{A \cap B}{A \cup B}$ | 交并比 |
| 边框回归 | $t_x = \frac{g_x - a_x}{a_w}$ | 中心点偏移 |
| NMS | 贪心选择最高分框 | 去除重复检测 |
| 多任务损失 | $L_{cls} + \lambda L_{reg}$ | 分类+回归 |

## 小结

1. **物体检测流程**：
   - 生成候选区域（RPN）
   - 提取区域特征（ROI池化）
   - 分类与定位（多任务学习）
   - 后处理（NMS + 边框回归）

2. **关键组件**：
   - 锚框：预定义的多尺度候选框
   - RPN：高效生成候选区域
   - ROI池化：统一特征尺寸
   - NMS：去除重复检测

3. **性能权衡**：
   - 两阶段：精度高，速度慢
   - 单阶段：速度快，精度稍低

4. **实际挑战**：
   - 小物体检测
   - 密集遮挡
   - 实时性要求
