## 三、各节核心要点

### 17.1 序贯决策问题与MDP

**核心定义**: MDP = ⟨S, A, P, R, γ⟩

```
┌────────────────────────────────────────────────────────────┐
│                    MDP 核心要素                             │
├────────────────────────────────────────────────────────────┤
│  S: 状态集合 (如4×3世界的11个非终止状态)                    │
│  A(s): 状态s下的可用动作                                    │
│  P(s'|s,a): 转移概率 (如0.8概率成功，0.1向两侧)            │
│  R(s,a,s'): 奖励函数 (如非终止状态转移奖励-0.04)           │
│  γ: 折扣因子 (0 ≤ γ < 1)                                   │
├────────────────────────────────────────────────────────────┤
│  策略 π: S → A —— 从状态到动作的映射                        │
│  状态价值 U^π(s): 从s开始执行π的期望折扣奖励                │
│  最优策略 π*: 使U^π(s)最大化的策略                          │
└────────────────────────────────────────────────────────────┘
```

**关键方程**:

```
贝尔曼方程:
U(s) = max_a Σ_{s'} P(s'|s,a)[R(s,a,s') + γU(s')]

Q函数:
Q(s,a) = Σ_{s'} P(s'|s,a)[R(s,a,s') + γ max_{a'} Q(s',a')]

策略提取:
π*(s) = argmax_a Q(s,a)
```

**重要性质**:
- **平稳性**: 无限期问题中最优策略与时间无关
- **折扣奖励**: 保证无限期效用有界(当γ<1时)
- **适当策略**: 保证最终到达终止状态

### 17.2 MDP算法

#### 17.2.1 价值迭代

```
算法: Value-Iteration(mdp, ε)
─────────────────────────────────────────
初始化: U(s) = 0 for all s

重复:
    U' ← U
    对每个状态s:
        U'(s) ← max_a Q-Value(mdp, s, a, U)
    δ ← max_s |U'(s) - U(s)|
直到 δ ≤ ε(1-γ)/γ

返回 U
```

**收敛性**: 贝尔曼更新是压缩映射(因子γ)，保证收敛到唯一解。

**复杂度**: 
- 每次迭代: O(|S|²|A|)
- 迭代次数: O(log(1/ε))

#### 17.2.2 策略迭代

```
算法: Policy-Iteration(mdp)
─────────────────────────────────────────
初始化: π ← 随机策略

重复:
    // 策略评估
    U ← Policy-Evaluation(π, mdp)  // 解线性方程组
    
    // 策略改进
    unchanged ← true
    对每个状态s:
        a* ← argmax_a Q-Value(mdp, s, a, U)
        if Q(s,a*) > Q(s,π(s)):
            π(s) ← a*
            unchanged ← false
直到 unchanged

返回 π
```

**策略评估**: 解简化贝尔曼方程组(线性，无max算子)
- 精确解: O(|S|³)
- 迭代解(修正策略迭代): 更快收敛

#### 17.2.3 线性规划

```
变量: U(s) for each s
目标: 最小化 Σ_s U(s)
约束: U(s) ≥ Σ_{s'} P(s'|s,a)[R(s,a,s') + γU(s')] 
      for all s, a
```

**意义**: 证明MDP可在多项式时间内求解。

#### 17.2.4 在线算法

**实时动态规划(RTDP)**:
- 从当前状态向前搜索
- 使用启发式边界状态价值
- 类似于LRTA*

**UCT算法**:
- 结合蒙特卡罗模拟和树搜索
- 适用于大规模MDP(如俄罗斯方块)

### 17.3 老虎机问题

**核心权衡**: 探索(Exploration) vs 利用(Exploitation)

```
┌────────────────────────────────────────────────────────────┐
│                   老虎机问题类型                            │
├────────────────────────────────────────────────────────────┤
│  1. 伯努利老虎机                                           │
│     - 每个臂: 成功概率μᵢ未知                                │
│     - 状态: (sᵢ, fᵢ) = (成功次数, 失败次数)                │
│                                                             │
│  2. 上下文老虎机                                           │
│     - 奖励取决于上下文变量                                  │
│                                                             │
│  3. 老虎机超过程(BSP)                                      │
│     - 每个臂是完整的MDP                                    │
│     - 全局最优 ≠ 局部最优之和                               │
└────────────────────────────────────────────────────────────┘
```

**基廷斯指数**:
```
Gittins(Mᵢ, s) = (1-γ) × V*(Mᵢ^s)

其中Mᵢ^s是在每个状态添加"重启到s"动作的MDP

最优策略: 始终选择基廷斯指数最高的臂
```

**近似算法**:
| 算法 | 核心思想 | 遗憾界 |
|-----|---------|-------|
| UCB | 均值+置信区间上界 | O(√(n log n)) |
| Thompson采样 | 后验采样 | O(√(n log n)) |
| ε-贪心 | 以ε概率随机探索 | O(n^{2/3}) |

### 17.4 部分可观测MDP

**核心挑战**: 智能体不知道确切状态，只有信念b(s) = P(当前状态=s | 历史)

```
┌────────────────────────────────────────────────────────────┐
│                   POMDP 要素                                │
├────────────────────────────────────────────────────────────┤
│  与MDP相同: S, A, P(s'|s,a), R(s,a,s'), γ                  │
│  新增: E (观测集合), P(e|s) (传感器模型)                   │
├────────────────────────────────────────────────────────────┤
│  信念更新(贝叶斯滤波):                                      │
│  b'(s') = α P(e|s') Σ_s P(s'|s,a) b(s)                     │
│                                                           │
│  = α × Forward(b, a, e)                                   │
└────────────────────────────────────────────────────────────┘
```

**关键洞察**: POMDP可转化为信念空间MDP

```
信念状态b是连续空间(|S|-1维单纯形)中的点
转移: P(b'|b,a) = Σ_e P(b'|b,a,e) P(e|a,b)
奖励: ρ(b,a) = Σ_s b(s) Σ_{s'} P(s'|s,a) R(s,a,s')
```

### 17.5 POMDP算法

#### 17.5.1 价值迭代

**关键性质**: 有限期最优价值函数是分段线性和凸的

```
U(b) = max_p b · α_p

其中α_p是条件规划p的效用向量
规划p对每个物理状态s给出效用α_p(s)
```

**算法**:
```
从深度1规划开始(单动作)
迭代加深:
  对每个动作a和每个子规划组合:
    构建新规划 [a; if e₁ then p₁ else ...]
    计算α向量
  剔除被支配的规划
直到收敛
```

**挑战**: 规划数量双指数增长 |A|^|E|^d

#### 17.5.2 在线算法

**POMCP (部分可观测蒙特卡罗规划)**:
```
1. 从当前信念b(粒子集合)开始
2. 构建信念空间搜索树
3. 使用粒子滤波更新信念
4. UCT算法选择动作
5. 执行动作，接收观测，重复
```

**优势**:
- 适用于大状态空间
- 不需要精确信念更新
- 可结合深度学习进行近似

## 四、核心算法与公式

### 4.1 贝尔曼方程对比

| 问题 | 方程 | 变量 |
|-----|-----|-----|
| MDP | U(s) = max_a Σ P(s'|s,a)[R + γU(s')] | 状态价值 |
| MDP (策略评估) | U^π(s) = Σ P(s'|s,π(s))[R + γU^π(s')] | 给定策略价值 |
| POMDP | U(b) = max_a [ρ(b,a) + γ Σ P(b'|b,a)U(b')] | 信念价值 |

### 4.2 压缩映射与收敛

```
贝尔曼算子B: (BU)(s) = max_a Σ P(s'|s,a)[R + γU(s')]

压缩性质: ||BU - BU'|| ≤ γ||U - U'||

不动点: U* = BU* (唯一解)

收敛速率: ||U_t - U*|| ≤ γ^t ||U_0 - U*||
```

### 4.3 函数设计定理

```
奖励变换保持最优策略不变:
R'(s,a,s') = R(s,a,s') + γΦ(s') - Φ(s)

应用: 奖励塑造(Reward Shaping)
- 添加势函数引导学习
- 不改变最优策略
- 加速收敛
```

## 五、重要案例与应用

### 5.1 4×3世界

```
+---+---+---+---+      r = -0.04 (每步)
|   |   |   |+1 |      γ = 1
+---+---+---+---+      +1和-1是终止状态
|   |   |   |-1 |
+---+---+---+---+
|S  |   |   |   |
+---+---+---+---+
```

**最优策略特征**:
- 平衡风险(避免-1)与效率(尽快到达+1)
- 状态(3,1)的Left和Up都是最优的

### 5.2 俄罗斯方块

```
状态空间: 7 × 7 × 2^200 ≈ 10^62
动作: 放置当前方块(位置和旋转)
奖励: 清除行数
终止: 棋盘填满
```

**所有策略都是适当的**: 最终必然填满。

### 5.3 老虎机应用

| 应用 | 臂 | 奖励 |
|-----|---|-----|
| 临床试验 | 不同疗法 | 治疗成功 |
| 广告投放 | 不同广告 | 点击率 |
| 推荐系统 | 不同推荐 | 用户满意度 |
| 投资组合 | 不同资产 | 收益率 |

## 六、学习路径建议

### 6.1 渐进学习路径
```
阶段1: 理解MDP定义和贝尔曼方程
    ↓ 解值迭代和策略迭代
阶段2: 理解折扣因子和收敛性
    ↓ 压缩映射理论
阶段3: 理解POMDP和信念状态
    ↓ 信念空间MDP转化
阶段4: 理解老虎机问题
    ↓ 探索vs利用权衡
阶段5: 综合应用
    ↓ 实际系统实现
```

### 6.2 关键思考问题
1. 为什么价值迭代收敛？压缩映射的作用是什么？
2. 策略迭代和价值迭代各有什么优缺点？
3. POMDP为什么比MDP困难得多？
4. 基廷斯指数为什么能分解老虎机问题？
5. 探索和利用的权衡如何在实际中平衡？

### 6.3 常见误区
- ❌ MDP需要枚举所有可能的状态序列
- ✅ MDP通过价值函数递归求解
- ❌ POMDP可以通过维护最可能状态来近似
- ✅ 需要维护完整信念分布
- ❌ 老虎机问题中最佳臂最终会被确定选择
- ✅ 最优策略可能以正概率选择次优臂

## 八、小结

第17章展示了在不确定性下进行序贯决策的理论和算法：

1. **MDP**: 完全可观测随机环境的统一框架
   - 价值迭代和策略迭代提供精确解法
   - 贝尔曼方程是理论基础

2. **POMDP**: 部分可观测环境的扩展
   - 信念状态概念
   - 在线算法处理复杂性

3. **老虎机**: 探索vs利用的经典问题
   - 基廷斯指数提供最优解
   - UCB和Thompson采样提供实用近似

这些技术构成了现代强化学习(第22章)和自主智能系统的理论基础。
