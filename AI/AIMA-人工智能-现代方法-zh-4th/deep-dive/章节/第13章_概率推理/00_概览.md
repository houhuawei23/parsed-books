## 知识要点详解

### 13.1 贝叶斯网络的表示

#### 核心思想
贝叶斯网络用**有向无环图(DAG)** 表示变量间的**条件独立性关系**，将指数级的完全联合分布紧凑地表示为局部条件概率的乘积。

#### 三个组成部分

| 组件 | 含义 | 示例 |
|-----|------|------|
| **节点** | 随机变量 | Burglary, Alarm, JohnCalls |
| **有向边** | 直接影响关系 | Burglary → Alarm |
| **CPT** | 条件概率表 | P(Alarm \| Burglary, Earthquake) |

#### 入室盗窃网络示例

```
Burglary ──┐
           ├──► Alarm ──► JohnCalls
Earthquake─┘           │
                       └──► MaryCalls
```

**语义**：入室盗窃和地震直接影响警报，警报又影响约翰和玛丽是否打电话。

---

### 13.2 贝叶斯网络的语义

#### 联合分布分解（链式法则的简化）

**通用链式法则**：
```
P(X₁, X₂, ..., Xₙ) = P(X₁) × P(X₂|X₁) × P(X₃|X₁,X₂) × ... × P(Xₙ|X₁,...,Xₙ₋₁)
```

**贝叶斯网络简化**：
```
P(X₁, X₂, ..., Xₙ) = ∏ᵢ P(Xᵢ | Parents(Xᵢ))
```

**关键洞察**：给定父节点，每个节点条件独立于其他前驱节点。

#### 条件独立性性质

| 性质 | 描述 | 图示 |
|-----|------|------|
| **局部马尔可夫性** | 给定父节点，节点独立于非后代 | X ⊥ NonDescendants(X) \| Parents(X) |
| **马尔可夫毯** | 给定父节点+子节点+子节点父节点，节点独立于其他所有节点 | X ⊥ All \| MB(X) |

#### d-分离（有向分离）

**问题**：给定集合Z，集合X和Y是否条件独立？

**判断步骤**：
1. 构造祖先子图（包含X、Y、Z及其祖先）
2. 道德化：连接共享子节点的无连接父节点
3. 去方向：将有向边改为无向边
4. 检查：Z是否阻塞所有X-Y路径

**三种基本连接结构**：

```
链式：X → Y → Z    分叉：X ← Y → Z    对撞：X → Y ← Z
给定Y时X⊥Z         给定Y时X⊥Z         给定Y时X⧫Z（相关）
```

### 13.3 精确推断算法

#### 1. 枚举推断（Enumeration）

**思想**：利用联合分布分解，对所有隐变量求和。

**算法复杂度**：O(n × 2ⁿ) —— 对n个布尔变量

**优化**：将求和向内移动，利用因子化减少重复计算。

#### 2. 变量消元（Variable Elimination）

**核心思想**：动态规划 —— 计算并缓存中间结果（因子）。

**关键操作**：
- **逐点乘积（Pointwise Product）**：合并因子
- **求和消元（Summing Out）**：消去变量

**示例**：
```
P(B|j,m) = α f₁(B) × Σₑ f₂(E) × Σₐ f₃(A,B,E) × f₄(A) × f₅(A)
         = α f₁(B) × Σₑ f₂(E) × f₆(B,E)
         = α f₁(B) × f₇(B)
```

**变量排序的重要性**：
- 不同消元顺序产生不同大小的中间因子
- 最优排序是NP困难问题
- 常用启发式：选择使下一个因子最小的变量

#### 复杂性分析

| 网络类型 | 复杂性 | 说明 |
|---------|-------|------|
| **单连通（多重树）** | O(n) | 线性时间，存在高效算法 |
| **多连通（一般网络）** | NP困难 | 可归约到3-SAT |
| **精确推断** | #P困难 | 计数问题，严格难于NP |

### 13.4 近似推断算法

#### 1. 直接采样方法

**a) 先验采样（Prior Sampling）**
- 按拓扑顺序采样每个变量
- 从P(Xᵢ | Parents(Xᵢ))中采样
- 生成与联合分布一致的样本

**b) 拒绝采样（Rejection Sampling）**
- 生成样本，拒绝与证据不一致的
- 在剩余样本中计数
- 问题：证据概率低时效率极差

**c) 似然加权（Likelihood Weighting）**
- 固定证据变量，只采样非证据变量
- 每个样本带有权重 = 证据的似然
- 权重是证据变量条件概率的乘积

#### 2. 马尔可夫链蒙特卡罗（MCMC）

**思想**：通过随机游走生成相关样本序列。

**a) 吉布斯采样（Gibbs Sampling）**
- 从任意状态开始（证据固定）
- 反复：选择一个非证据变量，以其**马尔可夫毯**为条件采样
- 在状态空间中随机游走

**马尔可夫毯采样分布**：
```
P(xᵢ | mb(Xᵢ)) ∝ P(xᵢ | Parents(Xᵢ)) × ∏_{Yⱼ ∈ Children(Xᵢ)} P(yⱼ | Parents(Yⱼ))
```

**为什么有效**：
- 吉布斯采样满足细致平衡条件
- 平稳分布就是真实后验分布P(X | e)
- 遍历性保证最终收敛

**b) 米特罗波利斯-黑斯廷斯（Metropolis-Hastings）**
- 从提议分布生成候选状态
- 按接受概率概率性地接受/拒绝
- 更灵活，可以设计更好的混合策略

#### 近似推断比较

| 方法 | 收敛速度 | 适用场景 | 缺点 |
|-----|---------|---------|------|
| 拒绝采样 | 极慢 | 证据概率高 | 证据概率低时不可行 |
| 似然加权 | 中等 | 证据在上游 | 下游证据时性能差 |
| 吉布斯采样 | 快 | 一般网络 | 确定性关系时混合慢 |
| MH采样 | 可调 | 复杂网络 | 需要设计提议分布 |

### 13.5 因果网络

#### 因果 vs 关联

| 特征 | 贝叶斯网络 | 因果网络 |
|-----|-----------|---------|
| 箭头方向 | 表示关联 | 表示因果 |
| 结构方程 | 无 | 有：X = f(Parents, U) |
| 干预预测 | 不能直接处理 | 通过do-算子处理 |

#### do-算子（干预）

**观测**：P(Y | X = x) —— X恰好为x的情况

**干预**：P(Y | do(X = x)) —— 强制设置X为x

**区别**：干预切断X的因果输入，观测不切断。

**调整公式**：
```
P(Y | do(X = x)) = Σ_z P(Y | X = x, Parents(X) = z) × P(Parents(X) = z)
```

#### 后门准则

**问题**：识别混杂变量，正确估计因果效应。

**后门准则**：集合Z满足：
1. Z阻塞所有从X到Y的后门路径
2. Z不包含X的后代

如果Z满足后门准则：
```
P(Y | do(X = x)) = Σ_z P(Y | X = x, Z = z) × P(Z = z)
```

## 常见问题

**Q1：贝叶斯网络中的箭头方向重要吗？**

对于表示联合分布，任何无环方向都可以。但：
- 因果方向通常产生更稀疏的网络
- 因果方向更符合人类专家知识
- 只有因果网络可以进行干预预测

**Q2：为什么精确推断是NP困难的？**

可将3-SAT归约到贝叶斯网络推断：
- 布尔变量作为网络根节点
- 子句节点作为确定性OR
- 可满足性对应P(S=true) > 0

**Q3：近似推断如何保证正确性？**

- 一致性：样本足够多时收敛到真实值
- 但收敛速度可能很慢
- MCMC的混合时间是关键问题

**Q4：什么时候使用精确推断，什么时候使用近似推断？**

- 精确：网络小（< 50节点）、树宽小、需要精确答案
- 近似：网络大、实时要求、可接受近似答案

## 章节小结

贝叶斯网络是人工智能历史上最成功的知识表示形式之一，它：

1. **解决了表示问题**：紧凑表示指数级联合分布
2. **提供了推断框架**：从精确到近似的完整算法谱系
3. **连接了因果推理**：超越关联，支持干预预测
4. **广泛应用**：医疗诊断、自然语言处理、计算机视觉...

掌握贝叶斯网络是理解现代概率AI系统的基础。
