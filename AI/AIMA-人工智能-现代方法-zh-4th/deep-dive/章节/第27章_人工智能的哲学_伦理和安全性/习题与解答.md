# 第27章 习题与解答

## 基础练习题

### 习题 27.1 弱AI与强AI

**题目**：区分以下概念，并举例说明：

(a) 弱AI与强AI
(b) 通用AI与超级AI

**解答**：

**(a) 弱AI与强AI**

| 特征 | 弱AI | 强AI |
|------|------|------|
| **定义** | 表现出智能行为 | 真正具有意识和思维 |
| **关注点** | 行为表现 | 内部心理状态 |
| **现状** | 已实现 | 未实现，有争议 |
| **例子** | AlphaGo、ChatGPT | 假设的真正理解中文的系统 |

**例子**：
- 弱AI：一个能流利对话的聊天机器人，但没有自我意识
- 强AI：一个有主观体验、真正"理解"对话意义的系统

**(b) 通用AI与超级AI**

| 特征 | 通用AI (AGI) | 超级AI (ASI) |
|------|--------------|--------------|
| **定义** | 能解决各种任务的AI | 在所有领域超越人类智能 |
| **范围** | 人类级别（多种任务） | 超人类级别 |
| **时间预测** | 未知，可能数十年 | 更遥远，假设在AGI之后 |
| **风险** | 中等 | 潜在高（控制问题）|

### 习题 27.2 哥德尔定理与AI

**题目**：解释为什么哥德尔不完备性定理不证明AI不可能。

**解答**：

**哥德尔定理内容**：
- 对任何能表达初等数论的**一致**形式系统F
- 存在命题G(F)：在F中不可判定
- 但G(F)为真（如果F一致）

**卢卡斯的论点**：
- 机器是形式系统，受哥德尔限制
- 人类能"看到"G(F)为真
- 因此人类心智超越机器

**反驳理由**：

1. **有限vs无限**：
   - 哥德尔定理适用于无限形式系统
   - 实际计算机和大脑都是有限的
   - 有限系统可用命题逻辑描述，不受哥德尔限制

2. **人类不一致**：
   - 卢卡斯假设人类是一致的
   - 实际上人类常犯逻辑错误
   - 不一致系统不受哥德尔定理保护

3. **相对性**：
   - 存在卢卡斯无法断言但我们可以断言的真命题
   - 这不意味着我们更优越

### 习题 27.3 中文房间

**题目**：分析中文房间论证的逻辑结构，并提出一个反驳。

**解答**：

**论证结构**：
1. 前提1：房间里的人不懂中文
2. 前提2：规则书和纸不懂中文
3. 前提3：系统整体=人+规则书+纸
4. 结论1：系统整体不懂中文
5. 前提4：计算机与中文房间等价
6. 结论2：计算机没有理解

**系统反驳**：

理解可能属于整个系统而非单个组件。

**类比**：
- 单个神经元不懂中文
- 但大脑整体（由神经元组成）理解中文
- 同理，人+规则书+纸可能构成理解的系统

**问题**：
- 什么构成"系统"？边界在哪里？
- 如果必须包含身体和世界，纯符号操作是否足够？

## 综合练习题

### 习题 27.4 隐私保护计算

**题目**：某医院想共享患者数据用于研究，同时保护隐私。分析以下方法的优缺点：

(a) k-匿名性（k=5）
(b) ε-差分隐私（ε=1）
(c) 联邦学习

**解答**：

**(a) k-匿名性**

**优点**：
- 概念简单，易于理解
- 实现相对容易
- 保证查询粒度

**缺点**：
- 背景知识攻击（如唯一邮编+年龄组合）
- 同质性攻击（同组敏感属性相同）
- 需要删除或泛化大量信息

**适用**：结构化数据，适度隐私要求

**(b) 差分隐私**

**优点**：
- 数学保证，可量化
- 防所有背景知识攻击
- 可组合分析（隐私预算）

**缺点**：
- 添加噪声降低数据效用
- 小ε时噪声大
- 实现复杂

**适用**：需要强隐私保证的场景

**(c) 联邦学习**

**优点**：
- 数据不离开本地
- 可利用分布式数据
- 隐私保护强

**缺点**：
- 通信开销
- 需要协调机制
- 模型可能泄露信息（需安全聚合）

**适用**：分布式数据，如移动设备

### 习题 27.5 公平性权衡

**题目**：解释为什么良好校准和机会均等在基类不同时不可兼得。

**解答**：

**定义回顾**：
- **良好校准**：给定预测分数，不同群体的实际正率相同
  $$P(Y=1|\hat{Y}=y, A=0) = P(Y=1|\hat{Y}=y, A=1)$$
  
- **机会均等**：真正阳性率（召回率）在各群体相同
  $$P(\hat{Y}=1|Y=1, A=0) = P(\hat{Y}=1|Y=1, A=1)$$

**不可能性**：

假设：
- 群体0：P(Y=1) = 0.3（再犯率30%）
- 群体1：P(Y=1) = 0.5（再犯率50%）

如果模型良好校准：
- 分数为0.4的被告，两群体实际再犯率都是40%
- 但群体0中，40% > 基线30%，这些人是"高于平均"
- 群体1中，40% < 基线50%，这些人是"低于平均"
- 因此两群体的假阴性率必然不同

**结论**：如果基类不同，良好校准意味着不同阈值，机会均等被破坏。

### 习题 27.6 价值对齐

**题目**：分析以下AI目标指定的问题，并提出改进：

"最大化公司利润"

**解答**：

**问题分析**：

| 意外后果 | 原因 |
|----------|------|
| 垄断行为 | 消除竞争可提高利润 |
| 剥削员工 | 降低工资增加利润 |
| 环境破坏 | 外部性未计入成本 |
| 欺诈客户 | 信息不对称可获利 |
| 逃税 | 减少支出 |

**改进方向**：

1. **明确约束**：
   - "在不违反法律的前提下"
   - "在长期可持续的前提下"

2. **修改目标**：
   - "最大化股东长期价值"（包括声誉）
   - "平衡利润、员工福利、环境影响"

3. **多目标优化**：
   - 利润 + λ₁·员工满意度 + λ₂·环境指标

4. **人在回路**：
   - 重大决策需人类批准
   - 定期审查AI行为

5. **辅助博弈**：
   - 建模人类偏好不确定性
   - 在不确定时询问

## 挑战练习题

### 习题 27.7 自主武器伦理

**题目**：从伦理和法律角度分析是否应该禁止致命性自主武器。

**解答**：

**支持禁止的论点**：

1. **道德责任**：
   - 生死决定应由人类做出
   - 机器无法承担道德责任

2. **歧视原则**：
   - 难以区分战斗员/非战斗员
   - 难以评估比例原则

3. **军备竞赛风险**：
   - 降低战争门槛
   - 扩散到非国家行为体

4. **可扩展性**：
   - 规模与硬件成正比
   - 百万架无人机可装入集装箱

**反对禁止的论点**：

1. **精确性**：
   - 可能比人类更精确
   - 减少平民伤亡

2. **自卫权**：
   - 国家有权自卫
   - 禁武条约历史上常被违反

3. **保护士兵**：
   - 减少己方伤亡
   - 无人作战的道德优势？

4. **定义困难**：
   - 自主程度的连续谱
   - 难以区分武器和民用技术

**建议**：
- 限制完全自主武器
- 保持人类在回路
- 建立国际监管框架
- 投资防御技术

### 习题 27.8 技术性失业

**题目**：分析AI对就业的影响，与历史上的技术变革比较。

**解答**：

**历史模式**：

| 技术 | 初期影响 | 长期影响 |
|------|----------|----------|
| 农业机械化 | 农民失业 | 就业转向工业 |
| 自动化制造 | 工人失业 | 服务业增长 |
| ATM | 柜员减少 | 银行网点增加，总员工增加 |

**本次可能不同**：

1. **速度**：变革可能比历史更快
2. **范围**：认知任务也被影响
3. **技能极化**：高技能和低技能工作增加，中间减少

**预测数据**：
- 47%职业有部分任务可自动化
- 仅5%职业完全可自动化
- 60%职业中30%任务可自动化

**应对策略**：

1. **教育**：
   - 终身学习
   - 技能再培训

2. **社会政策**：
   - 累进税制
   - 负所得税/UBI讨论
   - 便携式福利

3. **技术选择**：
   - 提高质量而非仅降低成本
   - 人机协作而非替代
