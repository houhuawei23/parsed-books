# 第26章 习题与解答

## 基础练习题

### 习题 26.1 传感器选择

**题目**：为以下应用场景选择合适的传感器组合，并说明理由：

(a) 室内扫地机器人
(b) 自动驾驶汽车
(c) 工业装配机械臂

**解答**：

**(a) 室内扫地机器人**

**推荐配置**：
- **激光雷达**（低成本，如RPLidar）：2D环境扫描、定位、地图构建
- **碰撞传感器/保险杠**：检测碰撞，触发避障
- **轮编码器**：里程计，估算行进距离
- **摄像头（可选）**：物体识别、语义理解
- **跌落传感器（悬崖传感器）**：检测楼梯边缘

**理由**：室内环境相对简单，成本控制优先；激光雷达提供足够精度用于SLAM；碰撞传感器作为安全备份。

**(b) 自动驾驶汽车**

**推荐配置**：
- **激光雷达**：高精度3D环境感知、障碍物检测
- **摄像头（多路）**：交通信号识别、车道线检测、物体分类
- **毫米波雷达**：远距离检测、速度测量、全天候工作
- **GPS + IMU**：全局定位、姿态估计
- **超声波**：近距离停车辅助

**理由**：多传感器融合应对不同场景（天气、光照）；冗余设计确保安全性；各传感器优势互补。

**(c) 工业装配机械臂**

**推荐配置**：
- **关节编码器**：精确关节位置反馈
- **力/扭矩传感器**：检测接触力，实现柔顺控制
- **摄像头**：视觉伺服、物体定位
- **接近传感器**：检测物体存在

**理由**：精度要求极高；力控制对装配任务至关重要；编码器提供必要的反馈精度。

### 习题 26.2 C空间分析

**题目**：考虑一个2连杆平面机械臂，臂长分别为$l_1$和$l_2$。

(a) C空间的维度是多少？如何表示？
(b) 如果工作空间中有圆形障碍物，C空间障碍物是什么形状？
(c) 给出正向运动学映射。

**解答**：

**(a) C空间维度**

- **维度**：2
- **表示**：$q = (\theta_1, \theta_2)$，其中$\theta_1$是连杆1相对于基座的角度，$\theta_2$是连杆2相对于连杆1的角度

**(b) C空间障碍物形状**

工作空间中的圆形障碍物在C空间中映射为复杂曲线围成的区域：
- 不是圆形
- 可能是非凸的
- 边界由代数曲线组成

**直观理解**：
- 机械臂有多种构形使末端位于同一工作空间点
- 障碍物边界对应于机械臂某点与障碍物接触的所有构形

**(c) 正向运动学**

末端执行器位置$(x, y)$：
$$x = l_1 \cos\theta_1 + l_2 \cos(\theta_1 + \theta_2)$$
$$y = l_1 \sin\theta_1 + l_2 \sin(\theta_1 + \theta_2)$$

末端执行器角度：$\phi = \theta_1 + \theta_2$

### 习题 26.3 贝叶斯滤波

**题目**：解释为什么在机器人定位中需要使用递归贝叶斯滤波而非单次测量。

**解答**：

**关键原因**：

1. **传感器噪声**：单次测量不可靠，需要多测量融合降低不确定性
2. **多模态分布**：全局定位需要表示多个可能位置（如对称走廊）
3. **动态环境**：机器人运动改变位置，需要预测-更新循环
4. **不完全可观测**：某些状态下传感器信息不足，需要时间序列推断

**递归滤波的优势**：
- 逐步积累证据
- 在线更新（无需存储所有历史数据）
- 预测步骤补偿运动

## 综合练习题

### 习题 26.4 PID控制器调参

**题目**：机械臂轨迹跟踪实验中观察到以下现象，分析原因并给出调整方案：

(a) 机械臂围绕参考轨迹持续震荡
(b) 机械臂缓慢接近轨迹但始终有稳态误差
(c) 机械臂对噪声非常敏感

**解答**：

**(a) 持续震荡**

**原因**：比例增益$K_P$过大或微分增益$K_D$过小

**分析**：
- 过大$K_P$：系统欠阻尼，惯性导致超调
- 过小$K_D$：缺乏足够的阻尼力抑制震荡

**调整方案**：
- 减小$K_P$
- 增大$K_D$

**(b) 缓慢接近+稳态误差**

**原因**：$K_P$过小且缺少积分项

**分析**：
- 过小$K_P$：纠正力不足
- 无$K_I$：无法消除累积误差

**调整方案**：
- 增大$K_P$
- 添加积分项（增加$K_I$）

**(c) 对噪声敏感**

**原因**：微分增益$K_D$过大

**分析**：
- 微分项对高频噪声敏感
- 过大$K_D$放大噪声影响

**调整方案**：
- 减小$K_D$
- 或添加低通滤波器预处理误差信号

### 习题 26.5 运动规划算法选择

**题目**：为以下场景推荐合适的运动规划算法：

(a) 工厂AGV在同一路径上多次往返
(b) 7自由度机械臂在杂乱环境中抓取物体
(c) 无人机在未知环境中实时避障

**解答**：

**(a) 工厂AGV多路径往返**

**推荐**：PRM（概率路线图）

**理由**：
- 多查询场景（同一路径反复使用）
- 2D/3D低维空间
- 可预处理，查询时快速

**(b) 7自由度机械臂杂乱环境**

**推荐**：RRT-Connect或RRT*

**理由**：
- 高维C空间（7维）
- 单查询场景（每次抓取目标不同）
- RRT*可渐近优化路径

**(c) 无人机实时避障**

**推荐**：改进的RRT（如Informed-RRT*）或势场法+优化

**理由**：
- 实时性要求（需要快速重规划）
- 动态环境（障碍物移动）
- 可能需要在线重规划（MPC）

### 习题 26.6 SLAM理解

**题目**：解释为什么SLAM被称为"鸡生蛋"问题，以及EKF-SLAM如何解决这个问题。

**解答**：

**"鸡生蛋"问题**：
- 定位需要地图作为参考
- 地图构建需要知道机器人位置
- 两者相互依赖，似乎都依赖对方

**EKF-SLAM解决方案**：

1. **联合状态向量**：
   $$X = [x_R, y_R, \theta_R, x_1, y_1, ..., x_n, y_n]^T$$
   包含机器人位姿和所有地标位置

2. **联合协方差矩阵**：
   同时估计机器人位置和地标位置的不确定性，以及它们之间的相关性

3. **递归更新**：
   - 观测到地标时，同时更新机器人和地标估计
   - 地标观测为机器人定位提供约束
   - 机器人运动模型为地标位置提供预测

**关键**：通过概率框架同时估计两者，利用观测相关性逐步减少不确定性。

### 习题 26.7 轨迹优化推导

**题目**：推导轨迹优化中效率项$J_{eff} = \int_0^1 \frac{1}{2}\|\dot{\tau}(s)\|^2 ds$的梯度。

**解答**：

**步骤1**：识别被积函数
$$F(s, \tau, \dot{\tau}) = \frac{1}{2}\|\dot{\tau}(s)\|^2 = \frac{1}{2}\dot{\tau}^T\dot{\tau}$$

**步骤2**：计算偏导数
$$\frac{\partial F}{\partial \tau} = 0$$（F不直接依赖$\tau$）

$$\frac{\partial F}{\partial \dot{\tau}} = \dot{\tau}$$

**步骤3**：应用欧拉-拉格朗日方程
$$\nabla_\tau J(s) = \frac{\partial F}{\partial \tau} - \frac{d}{ds}\frac{\partial F}{\partial \dot{\tau}}$$

$$= 0 - \frac{d}{ds}\dot{\tau}(s)$$

$$= -\ddot{\tau}(s)$$

**结论**：效率项的梯度是轨迹的负加速度。设梯度为零：
$$-\ddot{\tau}(s) = 0 \Rightarrow \ddot{\tau}(s) = 0$$

积分两次：$\tau(s) = as + b$，即直线路径。

**物理解释**：在无约束情况下，直线是最短（最有效率）的路径。

### 习题 26.8 人机协调博弈

**题目**：在两车并线场景中，用博弈论分析为什么"预测-决策分离"方法可能导致次优解。

**解答**：

**场景设定**：
- 自动驾驶汽车C欲并入主路
- 人类驾驶汽车H在主路行驶

**预测-决策分离方法**：
1. 预测H的行为（假设H独立于C行动）
2. 基于预测优化C的动作

**问题分析**：

1. **忽视相互影响**：
   - C的激进并线会促使H减速
   - C的保守等待会促使H保持速度
   - 分离方法无法建模这种相互影响

2. **错失协调机会**：
   - 最优策略可能是C主动表明意图，引导H做出配合
   - 分离方法只能被动响应预测的行为

3. **次优均衡**：
   - 可能收敛到"等待-通过"的低效均衡
   - 而"并线-让行"的协调更高效

**博弈论视角**：

完全解需要求解**动态博弈**，考虑：
- C的动作影响H的后续动作
- H的动作影响C的后续动作
- 寻找纳什均衡或斯坦克尔伯格均衡

**计算困难**：
- 连续状态-动作空间
- 不完全信息（不知道对方确切目标）
- 实时性要求

**实际折中**：
- 显式协调信号（转向灯、车辆运动）
- 部分博弈建模（考虑对预测的影响）

### 习题 26.9 强化学习迁移

**题目**：设计一个从模拟到真实的域随机化方案，用于训练四足机器人行走。

**解答**：

**随机化参数空间**：

| 参数类别 | 具体参数 | 随机范围 | 理由 |
|----------|----------|----------|------|
| **质量属性** | 身体质量 | [0.9, 1.1]×标称 | 制造误差、负载变化 |
| | 质心位置 | ±2cm | 装配误差 |
| | 腿部惯量 | [0.8, 1.2]×标称 | 结构变异 |
| **接触动力学** | 摩擦系数 | [0.3, 1.0] | 地面材质变化 |
| | 恢复系数 | [0.1, 0.5] | 地面软硬 |
| **执行器** | 最大扭矩 | [0.9, 1.0]×标称 | 电机个体差异 |
| | 响应延迟 | [0, 20ms] | 通信延迟 |
| **传感器** | 噪声标准差 | [0, 0.1]rad | 编码器噪声 |
| | 偏置 | [-0.05, 0.05]rad | 系统误差 |
| **外部环境** | 坡度 | [-5°, 5°] | 地形不平 |
| | 地面高度扰动 | ±2cm | 地形起伏 |

**训练策略**：

1. **课程学习**：
   - 初期：窄范围随机化（易环境）
   - 后期：宽范围随机化（难环境）

2. **自适应随机化**：
   - 根据真实机器人表现调整随机范围
   - 如果某参数在真实中表现差，增大该参数随机范围

3. **系统识别**：
   - 用真实数据校准部分参数分布
   - 减少不必要的保守性

**评估方案**：
- 在模拟中测试极端参数组合
- 在真实机器人上验证稳定性
- 比较不同随机化策略的迁移效果
