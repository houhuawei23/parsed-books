# 26.4 机器人感知

## 背景与动机

感知是机器人将传感器数据映射到环境内部表示的过程。由于传感器噪声、环境动态性和部分可观测性，感知本质上是一个**概率状态估计**问题。

## 核心概念

### 机器人感知框架

```
传感器数据 z_t ──→ 感知算法 ──→ 信念状态 P(X_t | z_{1:t}, a_{1:t-1})
                              ↓
                    用于决策和控制
```

### 递归贝叶斯滤波

状态估计的核心公式（连续变量版本）：

$$P(X_{t+1}|z_{1:t+1},a_{1:t}) = \alpha P(z_{t+1}|X_{t+1}) \int P(X_{t+1}|x_t,a_t) P(x_t|z_{1:t},a_{1:t-1}) dx_t$$

**组成部分**：
- **预测项** $\int P(X_{t+1}|x_t,a_t) P(x_t|...)dx_t$：根据动作预测状态
- **更新项** $P(z_{t+1}|X_{t+1})$：根据观测修正预测
- **归一化** $\alpha$：确保概率和为1

### 定位与地图构建

#### 1. 机器人状态表示

对于平面移动机器人，状态通常表示为：
$$X_t = (x_t, y_t, \theta_t)^T$$

其中：
- $(x_t, y_t)$：笛卡儿坐标位置
- $\theta_t$：朝向角

#### 2. 运动模型

**确定性模型**（运动学近似）：
$$\hat{X}_{t+1} = X_t + \begin{pmatrix} v_t \Delta t \cos\theta_t \\ v_t \Delta t \sin\theta_t \\ \omega_t \Delta t \end{pmatrix}$$

**概率模型**（高斯噪声）：
$$P(X_{t+1}|X_t, v_t, \omega_t) = \mathcal{N}(\hat{X}_{t+1}, \Sigma_x)$$

#### 3. 传感器模型

**地标模型**（已知特征位置）：

对于位置为$(x_i, y_i)$的地标：
$$\hat{z}_t = h(x_t) = \begin{pmatrix} \sqrt{(x_t-x_i)^2 + (y_t-y_i)^2} \\ \arctan\frac{y_i-y_t}{x_i-x_t} - \theta_t \end{pmatrix}$$

带噪声的传感器模型：
$$P(z_t|x_t) = \mathcal{N}(\hat{z}_t, \Sigma_z)$$

**距离扫描模型**：

对于M个波束的距离扫描$(z_1, ..., z_M)$：
$$P(z_t|x_t) = \alpha \prod_{j=1}^{M} e^{-(z_j - \hat{z}_j)^2 / 2\sigma^2}$$

其中$\hat{z}_j$是第j个波束方向的最近障碍物距离。

#### 4. 蒙特卡罗定位（MCL）

MCL是粒子滤波在定位中的应用：

```
算法：MCL
输入：动作a，观测z，粒子数N，运动模型，传感器模型，地图
输出：更新的粒子集

1. 初始化：从先验P(X_0)采样N个粒子
2. 对于每个时间步：
   a. 对每个粒子：
      - 根据运动模型采样新位置
      - 根据传感器模型计算权重
   b. 根据权重重采样N个粒子
```

**可视化**：
```mermaid
flowchart LR
    A[全局随机分布<br/>粒子] -->|观测1| B[聚焦到走廊<br/>多峰分布]
    B -->|观测2| C[收敛到<br/>单峰分布]
```

#### 5. 扩展卡尔曼滤波（EKF）定位

**核心思想**：线性化非线性模型

**线性化过程**：
- 运动模型在$\mu_t$处泰勒展开
- 传感器模型在预测位置处泰勒展开

**EKF算法**：
1. **预测**：
   - $\bar{\mu}_{t+1} = f(\mu_t, a_t)$
   - $\bar{\Sigma}_{t+1} = F_t \Sigma_t F_t^T + R_t$

2. **更新**：
   - $K_{t+1} = \bar{\Sigma}_{t+1} H_{t+1}^T (H_{t+1}\bar{\Sigma}_{t+1}H_{t+1}^T + Q_{t+1})^{-1}$
   - $\mu_{t+1} = \bar{\mu}_{t+1} + K_{t+1}(z_{t+1} - h(\bar{\mu}_{t+1}))$
   - $\Sigma_{t+1} = (I - K_{t+1}H_{t+1})\bar{\Sigma}_{t+1}$

**MCL vs EKF对比**：

| 特性 | MCL（粒子滤波） | EKF |
|------|----------------|-----|
| 表示 | 粒子集（非参数） | 高斯分布（参数） |
| 多峰性 | 支持 | 不支持 |
| 非线性 | 适用 | 需要线性化 |
| 计算 | 可并行 | 矩阵运算 |
| 适用场景 | 全局定位、绑架恢复 | 局部跟踪 |

### SLAM（同时定位与地图构建）

**问题定义**：同时估计机器人轨迹和未知环境地图

**方法**：
- **EKF-SLAM**：扩展状态向量包含地标位置
- **图优化**：将SLAM建模为图优化问题
- **FastSLAM**：结合粒子滤波和EKF

**挑战**：
- 维度诅咒：状态空间随地标数增长
- 数据关联：确定观测对应哪个地标

### 机器学习在感知中的应用

#### 1. 低维嵌入

使用无监督学习将高维传感器数据映射到低维空间。

#### 2. 适应性感知

**示例**：自动驾驶中的"可行驶表面"分类

```
初始状态：只识别路面为可行驶
         ↓
被迫驶离路面
         ↓
开始将草地分类为可行驶
         ↓
更新模型，将草地纳入可行驶表面
```

**自监督学习**：使用激光雷达作为监督信号训练视觉分类器

## 关键公式总结

| 公式 | 名称 | 作用 |
|------|------|------|
| $P(X_{t+1}\|...) = \alpha P(z\|X) \int P(X'\|x,a)P(x\|...)dx$ | 贝叶斯滤波 | 递归状态估计 |
| $\hat{X} = f(X,v,\omega)$ | 运动学模型 | 预测机器人运动 |
| $\hat{z} = h(x)$ | 观测模型 | 预测传感器读数 |
| $P(z\|x) = \mathcal{N}(\hat{z}, \Sigma_z)$ | 高斯传感器模型 | 建模测量噪声 |

## 常见陷阱

1. **忽视数据关联**
   - 观测到地标时需要确定是哪个地标
   - 错误关联会导致发散

2. **粒子退化**
   - MCL中粒子集中在少数位置
   - 需要定期重采样和添加随机粒子

3. **EKF线性化误差**
   - 高度非线性系统线性化误差大
   - 可能导致估计偏差

4. **过拟合环境**
   - 在特定环境训练的感知模型泛化差
   - 需要域适应技术
