# 第6章 约束满足问题 - 概览

## 章节定位

本章介绍**约束满足问题（CSP, Constraint Satisfaction Problem）**，这是一种通过因子化表示（factored representation）来求解问题的方法。与将状态视为原子黑盒的经典搜索不同，CSP显式地表示状态的内部结构——变量及其约束，从而可以使用通用启发式高效求解复杂问题。

## 核心思想

### 从原子状态到因子化表示

| 经典搜索 | 约束满足问题 |
|---------|-------------|
| 状态是原子黑盒 | 状态 = 变量赋值集合 |
| 领域特定转移函数 | 通用约束推理 |
| 只能问"是否目标？" | 可识别"为何不是解" |
| 启发式需人工设计 | 通用启发式自动推导 |

### CSP的三大优势

1. **自然表示**：许多问题可以直观地表示为变量和约束
2. **高效求解器**：多年研究产生快速通用的求解算法
3. **剪枝能力强**：通过约束传播一次性消除大量搜索空间

## 关键概念详解

### 1. CSP形式化

**定义**：CSP由三元组 $\langle X, D, C \rangle$ 定义
- $X = \{X_1, ..., X_n\}$：变量集合
- $D = \{D_1, ..., D_n\}$：域集合
- $C = \{C_1, ..., C_m\}$：约束集合

**解**：一致的完整赋值（所有变量赋值，满足所有约束）

### 2. 约束传播算法对比

| 算法 | 一致性级别 | 时间复杂度 | 应用场景 |
|-----|-----------|-----------|---------|
| 节点一致性 | 1一致性 | $O(nd)$ | 处理一元约束 |
| AC-3 | 弧一致性（2一致性） | $O(cd^3)$ | 通用预处理 |
| PC-2 | 路径一致性（3一致性） | $O(n^3d^5)$ | 困难问题 |
| MAC | 维护弧一致性 | 与搜索结合 | 搜索过程中 |

### 3. 回溯搜索优化策略

**变量排序**：
- **MRV（最少剩余值）**：选择"最受限"变量，失败优先
- **度启发式**：选择约束最多的变量，降低分支因子

**值排序**：
- **最少约束值**：为后续变量保留最多选择，失败延后

**推断机制**：
- **前向检验**：赋值后立即检查相邻变量
- **MAC**：维护完整弧一致性

**回溯优化**：
- **回跳**：跳过无关变量
- **约束学习**：记录无用赋值避免重复

### 4. 问题结构利用

| 结构类型 | 求解方法 | 复杂度 |
|---------|---------|--------|
| 独立子问题 | 分解求解 | $O(d^c \cdot n/c)$ |
| 树状结构 | Tree-CSP-Solver | $O(nd^2)$ |
| 近似树结构 | 割集调整 | $O(d^c \cdot (n-c)d^2)$ |
| 一般结构 | 树分解 | $O(nd^{w+1})$ |

## 关键公式速查

**弧一致性（AC-3）**：
```
对于弧(Xi, Xj)：
    对于Di中的每个值v：
        若Dj中没有值与v满足约束：
            从Di中删除v
```

**MRV启发式**：
```
选择具有最少剩余合法值的变量
即：失败优先原则
```

**树分解求解时间**：
```
O(nd^(w+1))
其中w是树宽
```

## 总结

约束满足问题提供了一种强大的问题建模和求解框架，其核心优势在于：

1. **声明式建模**：只需描述问题，无需指定如何求解
2. **通用推理**：约束传播技术适用于所有CSP
3. **结构利用**：问题结构（树宽、割集）直接决定求解效率

CSP技术已广泛应用于调度、配置、规划、生物信息学等领域，是人工智能的核心工具之一。
