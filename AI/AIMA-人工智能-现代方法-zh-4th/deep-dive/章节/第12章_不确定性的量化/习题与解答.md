### 习题12.1 【难度：⭐】概率基本公理

**知识点**：概率公理、样本空间、概率分布

**题目**：
考虑掷一个公平骰子的实验。

**问题**：
1. 样本空间Ω是什么？
2. 根据概率公理，每个可能结果的概率是多少？
3. 计算事件"点数大于4"的概率。
4. 如果骰子是非公平的，P(6)=0.5，其他点数概率相等，求P(1)。

**逐步提示**：
- 样本空间是所有可能结果的集合
- 概率公理要求所有概率之和为1
- 公平骰子每个结果概率相等

**详细解答**：

1. **样本空间**：
   Ω = {1, 2, 3, 4, 5, 6}

2. **公平骰子的概率**：
   根据公理：Σ P(ω) = 1，且每个结果等可能
   P(1) = P(2) = P(3) = P(4) = P(5) = P(6) = 1/6

3. **点数大于4的概率**：
   事件E = {5, 6}
   P(E) = P(5) + P(6) = 1/6 + 1/6 = 1/3

4. **非公平骰子**：
   P(6) = 0.5
   其他5个结果概率相等：P(1) = P(2) = P(3) = P(4) = P(5) = x
   
   由概率和为1：
   5x + 0.5 = 1
   5x = 0.5
   x = 0.1
   
   所以 **P(1) = 0.1**

**考点解析**：
- 概率公理：① 0 ≤ P(ω) ≤ 1 ② ΣP(ω) = 1
- 事件概率是其包含结果概率之和
- 概率分布是赋予每个可能值的概率

### 习题12.2 【难度：⭐】条件概率与乘积法则

**知识点**：条件概率、乘积法则、联合概率

**题目**：
某人群中：
- 患蛀牙(Cavity)的先验概率 P(C) = 0.2
- 牙痛(Toothache)的先验概率 P(T) = 0.1
- 已知患蛀牙时牙痛的条件概率 P(T|C) = 0.6

**问题**：
1. 计算P(T ∧ C)（既蛀牙又牙痛的概率）。
2. 计算P(C|T)（牙痛患者有蛀牙的概率）。
3. 解释为什么P(C|T) ≠ P(T|C)。

**逐步提示**：
- 使用乘积法则：P(a ∧ b) = P(a|b)P(b)
- 使用贝叶斯规则：P(a|b) = P(b|a)P(a)/P(b)

**详细解答**：

1. **计算P(T ∧ C)**：
   使用乘积法则：
   P(T ∧ C) = P(T|C) × P(C) = 0.6 × 0.2 = **0.12**

2. **计算P(C|T)**：
   使用贝叶斯规则：
   P(C|T) = P(T|C) × P(C) / P(T) = 0.6 × 0.2 / 0.1 = 0.12 / 0.1 = **1.2**
   
   等等，概率不能大于1！这说明题目给的数据不一致。
   
   实际上，如果P(T) = 0.1且P(T ∧ C) = 0.12，则不可能（因为P(T ∧ C) ≤ P(T)）。
   
   假设P(T) = 0.3（更合理的值）：
   P(C|T) = 0.12 / 0.3 = **0.4**

3. **P(C|T) ≠ P(T|C)的原因**：
   - P(T|C) = 0.6：在蛀牙患者中，60%会牙痛
   - P(C|T) = 0.4：在牙痛患者中，40%有蛀牙
   - 两者基数不同：蛀牙人群 vs 牙痛人群
   - 这是典型的"医生知道P(症状|疾病)，想知道P(疾病|症状)"问题
   - 贝叶斯法则允许我们从因果方向推断诊断方向

**考点解析**：
- 条件概率P(a|b)与P(b|a)通常不相等
- 乘积法则：P(a ∧ b) = P(a|b)P(b) = P(b|a)P(a)
- 贝叶斯规则连接了两个方向的条件概率

### 习题12.3 【难度：⭐⭐】边缘化与推断

**知识点**：边缘化、完全联合分布、概率推断

**题目**：
考虑 Toothache、Cavity、Catch（探针卡住）三变量的完全联合分布：

| | toothache, catch | toothache, ¬catch | ¬toothache, catch | ¬toothache, ¬catch |
|---|:---:|:---:|:---:|:---:|
| **cavity** | 0.108 | 0.012 | 0.072 | 0.008 |
| **¬cavity** | 0.016 | 0.064 | 0.144 | 0.576 |

**问题**：
1. 验证概率之和为1。
2. 计算P(Cavity)（通过边缘化）。
3. 计算P(Cavity | toothache)。
4. 计算P(Cavity | toothache ∧ catch)。

**逐步提示**：
- 边缘化：对不关心变量求和消元
- 条件概率：P(X|e) = P(X ∧ e) / P(e)
- 可以使用归一化常数α简化计算

**详细解答**：

1. **验证概率之和**：
   0.108 + 0.012 + 0.072 + 0.008 + 0.016 + 0.064 + 0.144 + 0.576 = **1.0** ✓

2. **计算P(Cavity)**：
   P(cavity) = 0.108 + 0.012 + 0.072 + 0.008 = **0.2**
   P(¬cavity) = 0.016 + 0.064 + 0.144 + 0.576 = **0.8**
   
   所以 **P(Cavity) = ⟨0.2, 0.8⟩**

3. **计算P(Cavity | toothache)**：
   P(toothache) = 0.108 + 0.012 + 0.016 + 0.064 = 0.2
   
   P(cavity ∧ toothache) = 0.108 + 0.012 = 0.12
   P(¬cavity ∧ toothache) = 0.016 + 0.064 = 0.08
   
   P(Cavity | toothache) = α⟨0.12, 0.08⟩ = ⟨0.12/0.2, 0.08/0.2⟩ = **⟨0.6, 0.4⟩**

4. **计算P(Cavity | toothache ∧ catch)**：
   P(toothache ∧ catch) = 0.108 + 0.016 = 0.124
   
   P(cavity ∧ toothache ∧ catch) = 0.108
   P(¬cavity ∧ toothache ∧ catch) = 0.016
   
   P(Cavity | toothache ∧ catch) = α⟨0.108, 0.016⟩ = ⟨0.108/0.124, 0.016/0.124⟩ ≈ **⟨0.87, 0.13⟩**

**考点解析**：
- 边缘化：P(Y) = Σz P(Y, z)
- 归一化：先计算非归一化值，再除以总和
- 条件概率的计算可以从联合分布直接得到

### 习题12.4 【难度：⭐⭐】独立性与条件独立性

**知识点**：独立性、条件独立性、联合分布分解

**题目**：
考虑以下两个场景：

**场景A**：Weather（天气）和 Toothache（牙痛）
- 常识告诉我们天气与牙痛无关

**场景B**：Toothache（牙痛）、Cavity（蛀牙）、Catch（探针卡住）
- 给定Cavity时，Toothache和Catch独立

**问题**：
1. 写出场景A的独立性断言。
2. 场景A的独立性如何简化P(Weather, Toothache, Cavity)的表示？
3. 写出场景B的条件独立性断言。
4. 场景B的条件独立性如何简化P(Toothache, Catch, Cavity)的表示？

**逐步提示**：
- 独立性：P(X|Y) = P(X) 或 P(X,Y) = P(X)P(Y)
- 条件独立性：P(X,Y|Z) = P(X|Z)P(Y|Z)
- 独立性减少需要指定的参数数量

**详细解答**：

1. **场景A的独立性**：
   P(Weather | Toothache) = P(Weather)
   或等价地：P(Weather, Toothache) = P(Weather) × P(Toothache)

2. **场景A的简化**：
   原始：P(Weather, Toothache, Cavity) 需要 4×2×2 - 1 = 15个独立参数
   
   使用独立性：
   P(Weather, Toothache, Cavity) = P(Weather) × P(Toothache, Cavity)
   
   需要参数：
   - P(Weather)：3个（假设4种天气）
   - P(Toothache, Cavity)：3个（2×2-1）
   总计：6个参数，大大简化

3. **场景B的条件独立性**：
   P(Toothache, Catch | Cavity) = P(Toothache | Cavity) × P(Catch | Cavity)
   或等价地：P(Toothache | Catch, Cavity) = P(Toothache | Cavity)

4. **场景B的简化**：
   原始：P(Toothache, Catch, Cavity) 需要 2×2×2 - 1 = 7个参数
   
   使用条件独立性：
   P(Toothache, Catch, Cavity) = P(Toothache | Cavity) × P(Catch | Cavity) × P(Cavity)
   
   需要参数：
   - P(Cavity)：1个
   - P(Toothache | Cavity)：2个
   - P(Catch | Cavity)：2个
   总计：5个参数
   
   对于n个症状，条件独立性使参数从O(2ⁿ)降到O(n)！

**考点解析**：
- 条件独立性是贝叶斯网络的基础
- 给定原因，不同症状条件独立
- 这种结构反映了因果关系的独立性模式

### 习题12.5 【难度：⭐⭐】贝叶斯法则应用

**知识点**：贝叶斯法则、因果推理、诊断推理

**题目**：
医学诊断场景：
- 某种罕见疾病M的先验概率 P(M) = 0.001
- 该疾病引起症状S的概率 P(S|M) = 0.95
- 症状S的总体发生率 P(S) = 0.02

**问题**：
1. 使用贝叶斯法则计算P(M|S)（有症状S的人患病的概率）。
2. 解释为什么尽管P(S|M)很高，P(M|S)却很低。
3. 如果P(S|¬M) = 0.01，验证P(S) = 0.02并重新计算P(M|S)。

**逐步提示**：
- 贝叶斯法则：P(M|S) = P(S|M)P(M)/P(S)
- 注意区分因果概率P(S|M)和诊断概率P(M|S)
- 使用条件化计算P(S)

**详细解答**：

1. **计算P(M|S)**：
   P(M|S) = P(S|M) × P(M) / P(S)
          = 0.95 × 0.001 / 0.02
          = 0.00095 / 0.02
          = **0.0475** 或约4.75%

2. **解释**：
   - P(S|M) = 0.95：患病者中95%有症状（高）
   - 但疾病本身罕见：P(M) = 0.001
   - 症状S更常见：P(S) = 0.02（可能是其他原因引起）
   - 基数效应：在大量人群中，虽然患病者几乎都有症状，但有症状的人中绝大多数未患病
   - 假阳性远多于真阳性

3. **使用P(S|¬M)验证**：
   P(S) = P(S|M)P(M) + P(S|¬M)P(¬M)
        = 0.95 × 0.001 + 0.01 × 0.999
        = 0.00095 + 0.00999
        = 0.01094 ≈ 0.011
   
   重新计算：
   P(M|S) = 0.95 × 0.001 / 0.01094 ≈ **0.087** 或约8.7%
   
   （注：原题P(S)=0.02与P(S|¬M)=0.01不完全一致，这里展示了如何正确计算）

**考点解析**：
- 贝叶斯法则连接因果和诊断方向
- 基础比率忽视：人们常高估P(M|S)而忽略P(M)
- 医学筛查中假阳性问题是普遍现象

### 习题12.6 【难度：⭐⭐⭐】联合概率分布与分解

**知识点**：联合分布、链式法则、条件概率分解

**题目**：
考虑四个布尔变量：Cloudy（多云）、Sprinkler（洒水器）、Rain（下雨）、WetGrass（草地湿）。

**问题**：
1. 写出链式法则，将P(C, S, R, W)分解为条件概率的乘积。
2. 如果已知因果关系：Cloudy影响Sprinkler和Rain；Sprinkler和Rain影响WetGrass，写出利用这些条件独立性的简化分解。
3. 计算两种表示方式所需的参数数量。

**逐步提示**：
- 链式法则：P(A,B,C,D) = P(A)P(B|A)P(C|A,B)P(D|A,B,C)
- 利用条件独立性移除条件集合中的变量
- 每个条件分布P(X|Parents)需要2^|Parents|个参数

**详细解答**：

1. **链式法则分解**：
   P(C, S, R, W) = P(C) × P(S|C) × P(R|C,S) × P(W|C,S,R)

2. **利用因果结构简化**：
   - Cloudy影响Sprinkler：P(S|C)（保留C，去掉S前的其他变量）
   - Cloudy影响Rain：P(R|C)（Rain不直接依赖Sprinkler）
   - Sprinkler和Rain影响WetGrass：P(W|S,R)（WetGrass不直接依赖Cloudy）
   
   简化分解：
   **P(C, S, R, W) = P(C) × P(S|C) × P(R|C) × P(W|S,R)**

3. **参数数量比较**：
   
   **完全联合分布**：
   2⁴ - 1 = **15个参数**
   
   **链式法则（无简化）**：
   - P(C)：1个
   - P(S|C)：2个
   - P(R|C,S)：4个
   - P(W|C,S,R)：8个
   总计：15个（与联合分布相同）
   
   **因果简化**：
   - P(C)：1个
   - P(S|C)：2个
   - P(R|C)：2个
   - P(W|S,R)：4个
   总计：**9个参数**
   
   节省：15 - 9 = 6个参数（40%减少）

**考点解析**：
- 链式法则提供任意分解方式
- 条件独立性基于领域知识
- 稀疏因果关系显著减少参数数量
- 这是贝叶斯网络的核心思想

### 习题12.7 【难度：⭐⭐⭐】朴素贝叶斯模型

**知识点**：朴素贝叶斯、条件独立性假设、分类

**题目**：
考虑文本分类的朴素贝叶斯模型：
- 类别变量：Category ∈ {sports, politics, technology}
- 特征变量：Word₁, Word₂, ..., Wordₙ（每个表示某关键词是否出现）

给定训练数据：
- 60%是sports文章，其中80%包含"score"，70%包含"team"
- 30%是politics文章，其中20%包含"score"，30%包含"team"
- 10%是technology文章，其中10%包含"score"，40%包含"team"

**问题**：
1. 写出朴素贝叶斯模型的联合概率分解公式。
2. 给定一篇包含"score"但不包含"team"的文章，预测其类别。
3. 解释朴素贝叶斯中"朴素"的含义及其潜在问题。

**逐步提示**：
- 朴素贝叶斯假设给定类别时特征条件独立
- 使用对数概率避免下溢
- 预测选择后验概率最大的类别

**详细解答**：

1. **联合概率分解**：
   P(Category, Word₁, ..., Wordₙ) = P(Category) × Πᵢ P(Wordᵢ | Category)

2. **类别预测**：
   
   证据e = {score=yes, team=no}
   
   计算P(Category | e) ∝ P(Category) × P(score|Category) × P(team=no|Category)
   
   **sports**：
   P(sports) × P(score|sports) × P(¬team|sports)
   = 0.6 × 0.8 × 0.3 = 0.144
   
   **politics**：
   P(politics) × P(score|politics) × P(¬team|politics)
   = 0.3 × 0.2 × 0.7 = 0.042
   
   **technology**：
   P(technology) × P(score|technology) × P(¬team|technology)
   = 0.1 × 0.1 × 0.6 = 0.006
   
   归一化：
   总和 = 0.144 + 0.042 + 0.006 = 0.192
   
   P(sports|e) = 0.144/0.192 = **0.75**
   P(politics|e) = 0.042/0.192 = **0.219**
   P(technology|e) = 0.006/0.192 = **0.031**
   
   **预测类别：sports**

3. **"朴素"的含义**：
   
   **含义**：模型假设给定类别时，所有特征相互独立。这意味着：
   P(Word₁, Word₂ | Category) = P(Word₁ | Category) × P(Word₂ | Category)
   
   **潜在问题**：
   - 现实中特征通常相关（如"first"和"quarter"在商业文章中共同出现）
   - 独立性假设过于简化
   - 后果：
     * 后验概率过于极端（接近0或1）
     * 模型对自己的预测过于自信
     * 但类别排序通常仍然准确
   
   **为什么仍有效**：
   - 即使概率估计不准确，决策边界（哪个类别概率最高）常常正确
   - 参数少，不易过拟合
   - 计算效率高

**考点解析**：
- 朴素贝叶斯是实践中最成功的分类器之一
- 条件独立性假设是"朴素"的来源
- 尽管假设不成立，但在许多任务上表现优异
- 文本分类、垃圾邮件过滤是经典应用

### 习题12.8 【难度：⭐⭐⭐】Wumpus世界概率推理

**知识点**：概率建模、条件独立性、边界变量

**题目**：
考虑4×4的Wumpus世界（简化版，忽略Wumpus和金币），智能体位于[1,1]，已访问[1,2]和[2,1]并检测到微风。

egin{array}{|c|c|c|c|}
\hline
[1,4] & [2,4] & [3,4] & [4,4] \\
\hline
[1,3] & [2,3] & [3,3] & [4,3] \\
\hline
[1,2]^* & [2,2] & [3,2] & [4,2] \\
\hline
[1,1] & [2,1]^* & [3,1] & [4,1] \\
\hline
\end{array}

*表示检测到微风的位置

假设：
- 每个非[1,1]的方格有0.2的概率包含无底洞(Pit)
- 微风(Breeze)当且仅当相邻方格有无底洞时出现

**问题**：
1. 定义相关的随机变量。
2. 识别边界变量(frontier)和其他变量(other)。
3. 解释为什么P(P₁,₃ | known, breeze)的计算只需要考虑边界变量。
4. 列出与证据一致的所有边界变量赋值。

**逐步提示**：
- 边界变量是已访问方格的相邻未知方格
- 利用条件独立性：给定边界变量，其他变量与证据独立
- 枚举所有一致的边界赋值

**详细解答**：

1. **随机变量**：
   - Pᵢⱼ：布尔变量，表示[i,j]有无底洞
   - Bᵢⱼ：布尔变量，表示[i,j]是否有微风（只在已访问方格观测）

2. **变量分类**：
   - **known**：P₁,₁（已知无洞），P₁,₂（已知无洞），P₂,₁（已知无洞）
   - **frontier（边界）**：与已访问方格相邻的未知方格
     * P₂,₂（相邻[1,2]和[2,1]）
     * P₃,₁（相邻[2,1]）
     * P₁,₃（相邻[1,2]）
   - **other（其他）**：其余10个方格

3. **条件独立性**：
   给定边界变量（P₂,₂, P₃,₁, P₁,₃），微风在[1,2]和[2,1]完全由这些边界变量决定。
   
   更远的变量（如P₄,₄）不直接影响观测到的微风。
   
   因此：
   P(P₁,₃ | known, breeze) = α P(P₁,₃) Σ_{frontier\{P₁,₃}} P(breeze | frontier) P(frontier)
   
   other变量被边缘化消去。

4. **一致的边界赋值**：
   
   [1,2]有微风 ⇒ P₂,₂ ∨ P₁,₃ = true（至少一个有洞）
   [2,1]有微风 ⇒ P₂,₂ ∨ P₃,₁ = true（至少一个有洞）
   
   一致赋值（f=false, t=true）：
   - (P₂,₂=t, P₃,₁=t, P₁,₃=t)
   - (P₂,₂=t, P₃,₁=t, P₁,₃=f)
   - (P₂,₂=t, P₃,₁=f, P₁,₃=t)
   - (P₂,₂=f, P₃,₁=t, P₁,₃=t)
   - (P₂,₂=t, P₃,₁=f, P₁,₃=f)
   - (P₂,₂=f, P₃,₁=t, P₁,₃=t)
   
   计算P(P₁,₃=t | evidence)：
   P₁,₃=t出现在4个赋值中，计算加权概率...
   （详细计算见教材12.7节）
   
   结果：P(P₁,₃) ≈ 0.31，P(P₂,₂) ≈ 0.86

**考点解析**：
- 这是概率推理的经典示例
- 条件独立性使得推理可处理
- 边界变量方法将指数级问题降为多项式级
- 概率智能体优于纯逻辑智能体

### 习题12.9 【难度：⭐⭐⭐】决策论基础

**知识点**：效用理论、期望效用、理性决策

**题目**：
考虑去机场的决策问题：
- 飞机起飞时间：10:00
- 从家到机场通常需要30分钟
- 不同出发时间的选项：

| 出发时间 | 及时到达概率 | 提前等待时间 |
|---------|------------|-------------|
| 8:30 | 0.99 | 60分钟 |
| 9:00 | 0.95 | 30分钟 |
| 9:30 | 0.70 | 0分钟 |

效用函数：
- 错过飞机：效用 = -100
- 等待每分钟：效用 = -1

**问题**：
1. 计算每个选项的期望效用。
2. 根据最大期望效用(MEU)原则，理性智能体应选择哪个选项？
3. 如果效用函数改为U(错过)=-50，决策会如何变化？

**逐步提示**：
- EU(action) = Σ P(outcome) × U(outcome)
- 每个选项有两个可能结果：及时到达或错过
- MEU选择期望效用最大的动作

**详细解答**：

1. **期望效用计算**：
   
   **8:30出发**：
   - 及时到达（概率0.99）：效用 = -60（等待60分钟）
   - 错过飞机（概率0.01）：效用 = -100
   - EU = 0.99 × (-60) + 0.01 × (-100) = -59.4 - 1 = **-60.4**
   
   **9:00出发**：
   - 及时到达（概率0.95）：效用 = -30
   - 错过飞机（概率0.05）：效用 = -100
   - EU = 0.95 × (-30) + 0.05 × (-100) = -28.5 - 5 = **-33.5**
   
   **9:30出发**：
   - 及时到达（概率0.70）：效用 = 0
   - 错过飞机（概率0.30）：效用 = -100
   - EU = 0.70 × 0 + 0.30 × (-100) = **-30**

2. **MEU决策**：
   比较期望效用：-60.4 < -33.5 < -30
   
   最大期望效用是 **-30（9:30出发）**
   
   但注意：虽然9:30的EU最高，但有30%概率错过飞机！
   这取决于效用函数的准确设定。

3. **效用函数变化后**：
   
   如果U(错过) = -50：
   
   **8:30**：0.99 × (-60) + 0.01 × (-50) = -59.4 - 0.5 = -59.9
   **9:00**：0.95 × (-30) + 0.05 × (-50) = -28.5 - 2.5 = -31.0
   **9:30**：0.70 × 0 + 0.30 × (-50) = -15
   
   现在 **9:30出发**（EU = -15）更明显地是最优选择。
   
   如果U(错过) = -200（非常严重的后果）：
   **9:00**：0.95 × (-30) + 0.05 × (-200) = -28.5 - 10 = -38.5
   **9:30**：0.70 × 0 + 0.30 × (-200) = -60
   
   此时 **9:00出发** 成为最优选择。

**考点解析**：
- 决策论 = 概率论 + 效用理论
- MEU原则是理性决策的基础
- 效用函数编码了决策者的偏好和风险态度
- 同样的概率，不同效用导致不同决策

### 习题12.10 【难度：⭐⭐⭐⭐】概率公理的合理性

**知识点**：概率公理、德菲内蒂定理、荷兰赌

**题目**：
德菲内蒂定理表明，违反概率公理的信念会导致"荷兰赌"（Dutch Book）——对手可以设计一组赌注使智能体必然亏损。

假设智能体持有以下信念：
- P(A) = 0.4
- P(B) = 0.3
- P(A ∧ B) = 0.0（认为A和B互斥）
- P(A ∨ B) = 0.8

**问题**：
1. 验证这些信念是否违反概率公理（特别是容斥原理）。
2. 说明对手如何设计荷兰赌。
3. 解释为什么一致性要求信念必须服从概率公理。

**逐步提示**：
- 容斥原理：P(A ∨ B) = P(A) + P(B) - P(A ∧ B)
- 荷兰赌涉及一组赌注，智能体无法全部获胜
- 一致性是理性的基本要求

**详细解答**：

1. **验证违反公理**：
   根据容斥原理：
   P(A ∨ B) = P(A) + P(B) - P(A ∧ B)
             = 0.4 + 0.3 - 0.0
             = 0.7
   
   但智能体认为 P(A ∨ B) = 0.8 ≠ 0.7
   
   **违反概率公理！**

2. **荷兰赌设计**：
   
   对手设计以下三笔赌注（金额单位：美元）：
   
   | 命题 | 智能体信念 | 对手赌注 | 智能体赌注 | A,B | A,¬B | ¬A,B | ¬A,¬B |
   |-----|-----------|---------|-----------|-----|------|------|-------|
   | A | 0.4 | 买A，4美元 | 卖A，6美元 | -6 | -6 | 4 | 4 |
   | B | 0.3 | 买B，3美元 | 卖B，7美元 | -7 | 3 | -7 | 3 |
   | A∨B | 0.8 | 卖A∨B，8美元 | 买A∨B，2美元 | 2 | 2 | 2 | -8 |
   | **合计** | | | | **-11** | **-1** | **-1** | **-1** |
   
   无论A和B结果如何，智能体**必然亏损**！

3. **一致性解释**：
   - 概率公理确保信念之间逻辑一致
   - 违反公理意味着对逻辑相关命题持有矛盾信念
   - 荷兰赌揭示了这种矛盾的代价
   - 理性要求避免可预见的必然损失
   - 因此理性智能体的信念必须服从概率公理

**考点解析**：
- 德菲内蒂定理为概率论提供行为主义基础
- 荷兰赌论证是概率主观解释的核心
- 概率公理不是任意的数学约定，而是理性的必要条件

### 习题12.11 【难度：⭐⭐⭐⭐】多证据的贝叶斯推理

**知识点**：多证据更新、条件独立性、诊断推理

**题目**：
考虑医疗诊断场景，疾病D与三个症状S₁、S₂、S₃：
- P(D) = 0.01
- P(S₁|D) = 0.9, P(S₁|¬D) = 0.1
- P(S₂|D) = 0.8, P(S₂|¬D) = 0.2
- P(S₃|D) = 0.7, P(S₃|¬D) = 0.1

假设给定D时，S₁、S₂、S₃条件独立。

**问题**：
1. 写出利用条件独立性简化P(D|S₁,S₂,S₃)的公式。
2. 计算P(D|S₁∧S₂∧¬S₃)。
3. 分析随着更多症状观测，对D的信念如何变化。

**逐步提示**：
- 朴素贝叶斯推理适用于多证据情况
- 使用对数变换避免数值下溢
- 注意归一化常数

**详细解答**：

1. **简化公式**：
   根据朴素贝叶斯模型：
   
   P(D | S₁, S₂, S₃) ∝ P(D) × P(S₁|D) × P(S₂|D) × P(S₃|D)
   
   P(¬D | S₁, S₂, S₃) ∝ P(¬D) × P(S₁|¬D) × P(S₂|¬D) × P(S₃|¬D)

2. **计算P(D|S₁∧S₂∧¬S₃)**：
   
   **对于D**：
   P(D) × P(S₁|D) × P(S₂|D) × P(¬S₃|D)
   = 0.01 × 0.9 × 0.8 × 0.3
   = 0.01 × 0.9 × 0.8 × 0.3
   = **0.00216**
   
   **对于¬D**：
   P(¬D) × P(S₁|¬D) × P(S₂|¬D) × P(¬S₃|¬D)
   = 0.99 × 0.1 × 0.2 × 0.9
   = **0.01782**
   
   **归一化**：
   P(D | S₁, S₂, ¬S₃) = 0.00216 / (0.00216 + 0.01782)
                      = 0.00216 / 0.01998
                      ≈ **0.108** 或约10.8%

3. **信念变化分析**：
   
   | 证据 | P(D|证据) |
   |-----|---------|
   | 无 | 1% |
   | S₁ | 8.3% |
   | S₁∧S₂ | 26.5% |
   | S₁∧S₂∧S₃ | 74.4% |
   | S₁∧S₂∧¬S₃ | 10.8% |
   
   观测：
   - 每个正症状都增加疾病概率
   - 三个正症状使概率从1%升至74%
   - S₃为假显著降低概率（从26.5%到10.8%）
   - 这就是诊断推理：从症状推断疾病

**考点解析**：
- 多证据推理是朴素贝叶斯的核心
- 条件独立性使计算可行
- 症状的组合模式对诊断至关重要
- 医学诊断系统基于此原理

### 习题12.12 【难度：⭐⭐⭐⭐】连续变量的概率

**知识点**：概率密度函数、均匀分布、高斯分布

**题目**：
考虑连续随机变量X表示 tomorrow's noon temperature。

**问题**：
1. 如果X ~ Uniform(18°C, 26°C)，计算P(20°C ≤ X ≤ 22°C)。
2. 解释为什么P(X = 20°C) = 0。
3. 如果X ~ N(22°C, 4°C²)，计算P(20°C ≤ X ≤ 24°C)。
4. 比较高斯分布和均匀分布的适用场景。

**逐步提示**：
- 连续变量概率是密度函数下的面积
- 单点概率为0（区间宽度为0）
- 高斯分布表或数值积分计算概率

**详细解答**：

1. **均匀分布概率**：
   Uniform(18, 26)的密度函数：
   f(x) = 1/(26-18) = 1/8 = 0.125，对于18 ≤ x ≤ 26
   
   P(20 ≤ X ≤ 22) = ∫₂₀²² f(x) dx
                  = (22-20) × 0.125
                  = 2 × 0.125
                  = **0.25** 或25%

2. **单点概率为0**：
   P(X = 20) = ∫₂₀²⁰ f(x) dx = 0
   
   解释：
   - 连续变量有无限多个可能值
   - 每个具体值的"概率质量"无限小
   - 只有区间才有非零概率
   - 概率密度f(x)不是概率，而是"概率密度"
   - 实际意义：温度精确等于20.000...°C的可能性为0

3. **高斯分布概率**：
   X ~ N(μ=22, σ²=4)，所以σ=2
   
   P(20 ≤ X ≤ 24) = P((20-22)/2 ≤ Z ≤ (24-22)/2)
                  = P(-1 ≤ Z ≤ 1)
                  
   查标准正态表：
   Φ(1) = 0.8413
   Φ(-1) = 0.1587
   
   P(-1 ≤ Z ≤ 1) = 0.8413 - 0.1587 = **0.6826** 或约68%
   
   （这是著名的"68-95-99.7"规则中的68%）

4. **分布比较**：
   
   **均匀分布适用**：
   - 完全无知（除范围外）
   - 物理对称性（如公平骰子）
   - 截断均匀：只知变量在某个区间
   
   **高斯分布适用**：
   - 中心极限定理（许多小因素之和）
   - 测量误差（通常假设高斯）
   - 自然现象（身高、体重等）
   - 有更丰富的结构（均值、方差）

**考点解析**：
- 连续变量需要概率密度函数
- 概率是密度下的面积，不是单点值
- 高斯分布在统计和AI中无处不在
- 选择合适的分布是建模的关键决策
