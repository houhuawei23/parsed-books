### 1.1 理论与实践的结合

前两节建立了稳定性理论的数学框架：
- 5.1节：定义了均匀稳定性和假设稳定性
- 5.2节：证明了稳定性可以推出泛化界

然而，理论的真正价值在于**应用**。本节将通过三个具体的算法实例，展示如何：
1. **验证算法的稳定性**：计算稳定性系数 $\beta$ 或 $\gamma$
2. **推导泛化误差界**：将稳定性系数代入定理5.1或5.2
3. **理解正则化的作用**：分析正则化如何增强算法稳定性

### 1.2 选取实例的考量

本节选取的三个算法具有代表性：

| 算法 | 损失函数 | 正则化 | 稳定性类型 | 代表意义 |
|-----|---------|-------|----------|---------|
| **岭回归** | 平方损失 | L2正则 | 均匀稳定性 | 经典正则化回归 |
| **支持向量回归** | $\epsilon$-不敏感损失 | L2正则 | 均匀稳定性 | 稀疏性+正则化 |
| **k-近邻** | 0/1损失 | 无 | 假设稳定性 | 无限VC维算法 |

### 1.3 核心方法论

分析一个算法的稳定性通常遵循以下步骤：

**步骤1：设定问题**
- 明确样本空间、标记空间、损失函数
- 写出算法的优化目标

**步骤2：考虑数据扰动**
- 定义原始数据集 $D$ 和扰动数据集 $D'$（移除或替换一个样本）
- 设 $w_D$ 和 $w_{D'}$ 分别为在两个数据集上学习得到的模型参数

**步骤3：利用强凸性**
- 证明目标函数是强凸的（通常是正则化的结果）
- 利用强凸性得到 $\|w_D - w_{D'}\|$ 的上界

**步骤4：推导稳定性**
- 利用参数差距控制损失差距：$|\ell(w_D, z) - \ell(w_{D'}, z)|$
- 得到稳定性系数 $\beta$

**步骤5：应用泛化定理**
- 将 $\beta$ 代入定理5.1或5.2
- 得到具体的泛化误差界

### 3.1 岭回归（Ridge Regression）

#### 3.1.1 问题设定

**样本空间**：$\mathcal{X} = \{x \in \mathbb{R}^d : \|x\| \leq r\}$（有界）

**标记空间**：$\mathcal{Y} \subset \mathbb{R}$

**损失函数**：平方损失（Square Loss）
$$\ell_2(w, (x, y)) = (w^\top x - y)^2 \tag{5.71}$$

**优化目标**：
$$F_D(w) = \frac{1}{m}\sum_{i=1}^m (w^\top x_i - y_i)^2 + \lambda\|w\|^2 \tag{5.72}$$

其中 $\lambda > 0$ 是正则化参数。

#### 3.1.2 稳定性分析

**定理5.7**：给定常数 $r > 0$，考虑样本空间 $\mathcal{X} = \{x \in \mathbb{R}^d : \|x\| \leq r\}$，平方函数 $\ell_2(\cdot, \cdot) \in [0, M]$，以及优化目标函数(5.72)，则岭回归具有替换样本$\beta$-均匀稳定性。

**证明详解**：

**步骤1：设定符号**

令 $D = \{(x_1, y_1), \ldots, (x_m, y_m)\}$ 为原始数据集，$D' = D^{k, z_k'}$ 为将第 $k$ 个样本替换为 $z_k' = (x_k', y_k')$ 后的数据集。

令 $w_D$ 和 $w_{D'}$ 分别为优化 $F_D(w)$ 和 $F_{D'}(w)$ 得到的最优解。

**步骤2：控制损失差距**

对任意样本 $(x, y)$，利用平方差公式：
$$\begin{aligned}
&|(w_D^\top x - y)^2 - (w_{D'}^\top x - y)^2| \\
&= |(w_D - w_{D'})^\top x| \cdot |w_D^\top x - y + w_{D'}^\top x - y| \\
&\leq \|w_D - w_{D'}\| \cdot \|x\| \cdot (|w_D^\top x - y| + |w_{D'}^\top x - y|) \\
&\leq \|w_D - w_{D'}\| \cdot r \cdot 2\sqrt{M} \\
&= 2r\sqrt{M} \|w_D - w_{D'}\|
\end{aligned}$$

其中利用了：
- Cauchy-Schwarz不等式：$|a^\top b| \leq \|a\| \|b\|$
- 有界性假设：$|w^\top x - y| \leq \sqrt{M}$
- 样本有界：$\|x\| \leq r$

**步骤3：利用强凸性**

关键观察：目标函数 $F_D(w)$ 是**$2\lambda$-强凸**的。

**强凸性定义**：函数 $f$ 是 $\mu$-强凸的，如果 $f(y) \geq f(x) + \nabla f(x)^\top(y-x) + \frac{\mu}{2}\|y-x\|^2$。

对于岭回归，由于存在正则项 $\lambda\|w\|^2$，其Hessian矩阵满足：
$$\nabla^2 F_D(w) = \frac{2}{m}\sum_{i=1}^m x_i x_i^\top + 2\lambda I \succeq 2\lambda I$$

因此 $F_D$ 是 $2\lambda$-强凸的。

利用强凸性的一阶条件（$w_D$ 是最优点，$\nabla F_D(w_D) = 0$）：
$$F_D(w_{D'}) \geq F_D(w_D) + \lambda\|w_D - w_{D'}\|^2 \tag{5.74}$$

同理：
$$F_{D'}(w_D) \geq F_{D'}(w_{D'}) + \lambda\|w_D - w_{D'}\|^2 \tag{5.75}$$

**步骤4：合并不等式**

将(5.74)和(5.75)相加：
$$\begin{aligned}
2\lambda\|w_D - w_{D'}\|^2 &\leq F_D(w_{D'}) - F_D(w_D) + F_{D'}(w_D) - F_{D'}(w_{D'}) \\
&= \frac{1}{m}[\ell_2(w_{D'}, (x_k, y_k)) - \ell_2(w_D, (x_k, y_k)) \\
&\quad + \ell_2(w_D, (x_k', y_k')) - \ell_2(w_{D'}, (x_k', y_k'))]
\end{aligned}$$

利用步骤2的结果（损失差距上界）：
$$2\lambda\|w_D - w_{D'}\|^2 \leq \frac{1}{m} \cdot 2r\sqrt{M} \|w_D - w_{D'}\| + \frac{1}{m} \cdot 2r\sqrt{M} \|w_D - w_{D'}\| = \frac{4r\sqrt{M}}{m}\|w_D - w_{D'}\|$$

解得：
$$\|w_D - w_{D'}\| \leq \frac{2r\sqrt{M}}{m\lambda} \tag{5.78}$$

**步骤5：得到稳定性系数**

将(5.78)代回步骤2：
$$|(w_D^\top x - y)^2 - (w_{D'}^\top x - y)^2| \leq 2r\sqrt{M} \cdot \frac{2r\sqrt{M}}{m\lambda} = \frac{4r^2M}{m\lambda}$$

因此，岭回归具有替换样本$\beta$-均匀稳定性，其中：
$$\beta = \frac{4r^2M}{m\lambda}$$

#### 3.1.3 泛化误差界

**推论5.3**：给定常数 $r > 0$，考虑样本空间 $\mathcal{X} = \{x \in \mathbb{R}^d : \|x\| \leq r\}$ 和平方函数 $\ell_2(\cdot, \cdot) \in [0, M]$，令 $w_D$ 表示优化目标函数 $F_D(w)$ 所得的最优解。对任意 $\delta \in (0, 1)$，以至少 $1 - \delta$ 的概率有：

$$R(w_D) \leq \widehat{R}(w_D) + \frac{4Mr^2}{\lambda m} + \left(\frac{8Mr^2}{\lambda} + M\right)\sqrt{\frac{\ln(1/\delta)}{2m}} \tag{5.80}$$

**证明**：将 $\beta = \frac{4r^2M}{m\lambda}$ 代入定理5.1的(5.17)。

**结果解读**：
- 泛化界为 $O(1/\sqrt{m})$，与样本量的平方根成反比
- 正则化参数 $\lambda$ 越大，稳定性越好（$\beta$ 越小），但可能增加偏差
- 样本界 $r$ 越大，稳定性越差（$\beta$ 越大）

### 3.2 支持向量回归（Support Vector Regression）

#### 3.2.1 问题设定

**损失函数**：$\epsilon$-不敏感损失（$\epsilon$-insensitive Loss）
$$\ell_\epsilon(w, (x, y)) = \begin{cases} 0 & \text{如果 } |w^\top x - y| \leq \epsilon \\ |w^\top x - y| - \epsilon & \text{如果 } |w^\top x - y| > \epsilon \end{cases} \tag{5.55}$$

**直观理解**：
- 当预测误差小于 $\epsilon$ 时，损失为0（"不敏感"区域）
- 当预测误差大于 $\epsilon$ 时，损失为线性增长
- 这使得SVR具有稀疏性（类似于SVM的支持向量）

**优化目标**：
$$F_D(w) = \frac{1}{m}\sum_{i=1}^m \ell_\epsilon(w, (x_i, y_i)) + \lambda\|w\|^2 \tag{5.56}$$

#### 3.2.2 稳定性分析

**定理5.6**：给定常数 $r > 0$，考虑样本空间 $\mathcal{X} = \{x \in \mathbb{R}^d : \|x\| \leq r\}$，以及优化目标函数(5.56)，则支持向量回归具有替换样本$\beta$-均匀稳定性。

**证明详解**：

**步骤1：分析损失函数的Lipschitz性质**

$\epsilon$-不敏感损失的一个重要性质是**1-Lipschitz**：
$$|\ell_\epsilon(w, (x, y)) - \ell_\epsilon(w', (x, y))| \leq |w^\top x - w'^\top x| = |(w - w')^\top x| \leq \|w - w'\| \|x\| \leq r\|w - w'\|$$

**分情况验证**：

设 $f = w^\top x - y$，$f' = w'^\top x - y$。

**情况1**：$|f| \leq \epsilon$ 且 $|f'| \leq \epsilon$
- 损失都为0，差距为0

**情况2**：$|f| > \epsilon$ 且 $|f'| > \epsilon$
- $|\ell_\epsilon(w) - \ell_\epsilon(w')| = ||f| - \epsilon - (|f'| - \epsilon)| = ||f| - |f'|| \leq |f - f'|$

**情况3**：$|f| > \epsilon$ 且 $|f'| \leq \epsilon$
- $|\ell_\epsilon(w) - \ell_\epsilon(w')| = ||f| - \epsilon| = |f| - \epsilon \leq |f| - |f'| \leq |f - f'|$

综上，总有：
$$|\ell_\epsilon(w, (x, y)) - \ell_\epsilon(w', (x, y))| \leq r\|w - w'\| \tag{5.61}$$

**步骤2：利用强凸性**

与岭回归类似，$F_D(w)$ 是 $2\lambda$-强凸的。

利用强凸性：
$$\lambda\|w_D - w_{D'}\|^2 \leq F_D(w_{D'}) - F_D(w_D)$$

展开 $F_D(w_{D'}) - F_D(w_D)$：
$$\begin{aligned}
&F_D(w_{D'}) - F_D(w_D) \\
&= \frac{1}{m}[\ell_\epsilon(w_{D'}, (x_k, y_k)) - \ell_\epsilon(w_D, (x_k, y_k))] \\
&\quad + \frac{1}{m}[\ell_\epsilon(w_D, (x_k', y_k')) - \ell_\epsilon(w_{D'}, (x_k', y_k'))] \\
&\quad + F_{D'}(w_{D'}) - F_{D'}(w_D) \\
&\leq \frac{r\|w_D - w_{D'}\|}{m} + \frac{r\|w_D - w_{D'}\|}{m} + 0 \\
&= \frac{2r\|w_D - w_{D'}\|}{m}
\end{aligned}$$

其中利用了：
- (5.61)的Lipschitz性质
- $F_{D'}(w_{D'}) - F_{D'}(w_D) \leq 0$（因为 $w_{D'}$ 是 $F_{D'}$ 的最小值点）

因此：
$$\lambda\|w_D - w_{D'}\|^2 \leq \frac{2r\|w_D - w_{D'}\|}{m}$$

解得：
$$\|w_D - w_{D'}\| \leq \frac{2r}{m\lambda} \tag{5.68}$$

**步骤3：得到稳定性系数**

将(5.68)代入(5.61)：
$$|\ell_\epsilon(w_D, (x, y)) - \ell_\epsilon(w_{D'}, (x, y))| \leq r \cdot \frac{2r}{m\lambda} = \frac{2r^2}{m\lambda}$$

因此，支持向量回归具有替换样本$\beta$-均匀稳定性，其中：
$$\beta = \frac{2r^2}{m\lambda}$$

#### 3.2.3 泛化误差界

**推论5.2**：给定常数 $r > 0$，考虑样本空间 $\mathcal{X} = \{x \in \mathbb{R}^d : \|x\| \leq r\}$ 和 $\epsilon$-不敏感函数 $\ell_\epsilon(\cdot, \cdot) \in [0, M]$，令 $w_D$ 表示优化目标函数 $F_D(w)$ 所得的最优解。对任意 $\delta \in (0, 1)$，以至少 $1 - \delta$ 的概率有：

$$R(w_D) \leq \widehat{R}(w_D) + \frac{2r^2}{m\lambda} + \left(\frac{4r^2}{\lambda} + M\right)\sqrt{\frac{\ln(1/\delta)}{2m}} \tag{5.70}$$

**与岭回归的比较**：

| 特性 | 岭回归 | 支持向量回归 |
|-----|-------|------------|
| 稳定性系数 | $\beta = \frac{4r^2M}{m\lambda}$ | $\beta = \frac{2r^2}{m\lambda}$ |
| 依赖损失界 $M$ | 是 | 否 |
| 稀疏性 | 无 | 有（支持向量） |
| 对异常值敏感 | 是（平方损失） | 否（线性损失） |

注意：SVR的稳定性系数不直接依赖于 $M$，这是因为 $\epsilon$-不敏感损失的Lipschitz常数是固定的（为1），而平方损失的Lipschitz常数依赖于损失值的大小。

### 3.3 k-近邻（k-Nearest Neighbor）

#### 3.3.1 问题设定

**样本空间**：$\mathcal{X} \subseteq \mathbb{R}^d$

**标记空间**：$\mathcal{Y} = \{-1, +1\}$（二分类）

**算法定义**：

给定样本 $x \in \mathcal{X}$，令 $\pi_1(x), \pi_2(x), \ldots, \pi_m(x)$ 表示根据欧氏距离排序的训练样本索引：
$$\|x - x_{\pi_i(x)}\| \leq \|x - x_{\pi_{i+1}(x)}\|$$

k-NN的输出函数为：
$$\mathfrak{L}_D^k(x) = \begin{cases} +1 & \text{如果 } \sum_{i=1}^k y_{\pi_i(x)} \geq 0 \\ -1 & \text{如果 } \sum_{i=1}^k y_{\pi_i(x)} < 0 \end{cases} \tag{5.81}$$

**损失函数**：0/1损失
$$\ell(\mathfrak{L}_D^k, (x, y)) = \mathbb{I}(\mathfrak{L}_D^k(x) \neq y) \tag{5.82}$$

#### 3.3.2 为什么不具有均匀稳定性

**观察**：k-NN不具有替换样本或移除样本的均匀稳定性。

**反例**：
考虑一个测试点 $x$，其第 $k$ 近邻和第 $(k+1)$ 近邻距离相等，且标签相反。替换其中一个样本可能完全改变预测结果。

形式化地，无论 $m$ 多大，总存在数据集 $D$ 和样本 $z$ 使得：
$$|\ell(\mathfrak{L}_D^k, z) - \ell(\mathfrak{L}_{D^{i,z'}}, z)| = 1$$

因此，k-NN无法满足均匀稳定性的定义。

#### 3.3.3 假设稳定性分析

**定理5.8**：考虑样本空间 $\mathcal{X} \subseteq \mathbb{R}^d$ 和标记空间 $\mathcal{Y} = \{-1, +1\}$，则k-近邻算法具有替换样本$\beta$-假设稳定性。

**证明详解**：

**步骤1：分析预测改变的概率**

给定训练集 $D$，考虑替换第 $i$ 个样本为 $z_i'$。对于随机抽取的测试点 $x \sim \mathcal{D}_{\mathcal{X}}$，分析 $\mathfrak{L}_D^k(x) \neq \mathfrak{L}_{D^{i,z_i'}}^k(x)$ 的概率。

**关键观察**：预测改变需要同时满足：
1. 被替换的样本 $z_i$ 是 $x$ 的 $k$ 个近邻之一
2. $x$ 的 $k$ 个近邻中正负样本个数相差不超过1（即决策边界附近）

**步骤2：计算概率**

条件于 $x$ 和其第 $(k+1)$ 近邻 $(x_{\pi_{k+1}(x)}, y_{\pi_{k+1}(x)})$：

- 被替换样本是 $k$ 近邻之一的概率：$\frac{k}{m}$
- 给定 $k$ 近邻中有 $N_+$ 个正样本，决策边界附近的概率：$P(|N_+ - k/2| \leq 1/2)$

利用**引理5.2** [Devroye and Wagner, 1979a]：

若 $X \sim \text{Binomial}(k, 1/2)$，则：
$$P(|X - k/2| \leq a/2) \leq \frac{2\sqrt{2}a}{\sqrt{\pi k}}$$

取 $a = 1$：
$$P(|N_+ - k/2| \leq 1/2) \leq \frac{2\sqrt{2}}{\sqrt{\pi k}}$$

**步骤3：合并结果**

$$\begin{aligned}
&P_{x \sim \mathcal{D}_{\mathcal{X}}}(\mathfrak{L}_D^k(x) \neq \mathfrak{L}_{D^{i,z_i'}}^k(x)) \\
&\leq \frac{k}{m} \cdot \frac{2\sqrt{2}}{\sqrt{\pi k}} = \frac{2\sqrt{2k}}{m\sqrt{\pi}}
\end{aligned}$$

**步骤4：得到稳定性系数**

对于0/1损失：
$$\begin{aligned}
&\mathbb{E}_{(x,y) \sim \mathcal{D}}[|\mathbb{I}(\mathfrak{L}_D^k(x) \neq y) - \mathbb{I}(\mathfrak{L}_{D^{i,z_i'}}^k(x) \neq y)|] \\
&= P_{x \sim \mathcal{D}_{\mathcal{X}}}(\mathfrak{L}_D^k(x) \neq \mathfrak{L}_{D^{i,z_i'}}^k(x)) \\
&\leq \frac{2\sqrt{2k}}{m\sqrt{\pi}}
\end{aligned}$$

因此，k-NN具有替换样本$\beta$-假设稳定性，其中：
$$\beta = \frac{2\sqrt{2k}}{m\sqrt{\pi}}$$

#### 3.3.4 泛化误差界

**推论5.4**：考虑样本空间 $\mathcal{X} \subseteq \mathbb{R}^d$ 和标记空间 $\mathcal{Y} = \{-1, +1\}$，令 $\mathfrak{L}_D$ 表示k-近邻算法基于数据集 $D$ 学习所得的输出函数，则有：

$$\mathbb{E}_{D \sim \mathcal{D}^m}[(R(\mathfrak{L}_D) - \widehat{R}(\mathfrak{L}_D))^2] \leq \frac{8M\sqrt{2k}}{m\sqrt{\pi}} + \frac{M^2}{m} \tag{5.88}$$

**证明**：将 $\beta = \frac{2\sqrt{2k}}{m\sqrt{\pi}}$ 代入定理5.2的(5.27)。

**意义**：
- 这是**期望意义下**的泛化界（而非高概率界）
- 稳定性系数 $\beta = O(\sqrt{k}/m)$，与 $k$ 的平方根成正比
- 当 $k$ 固定、$m \to \infty$ 时，泛化差距趋于0

### 4.1 强凸性的核心作用

在岭回归和SVR的分析中，**强凸性**是推导稳定性的关键。让我们深入理解这一点。

**强凸性的几何意义**：

一个$\mu$-强凸函数在任何点的曲率都至少为$\mu$。这意味着：
1. 函数有唯一的最小值点
2. 偏离最小值时，函数值增长至少为二次速度

**强凸性如何带来稳定性**：

设 $w_D$ 和 $w_{D'}$ 分别是在 $D$ 和 $D'$ 上的最优解。强凸性意味着：
$$F_D(w_{D'}) \geq F_D(w_D) + \frac{\mu}{2}\|w_D - w_{D'}\|^2$$

如果 $w_{D'}$ 离 $w_D$ 很远，则 $F_D(w_{D'})$ 会比 $F_D(w_D)$ 大很多。但由于 $D$ 和 $D'$ 只相差一个样本，$F_D$ 和 $F_{D'}$ 的值不会相差太大，因此 $w_{D'}$ 不能离 $w_D$ 太远。

**正则化与强凸性**：

对于经验风险 $R_{emp}(w) = \frac{1}{m}\sum_{i=1}^m \ell(w, z_i)$，加入L2正则化：
$$F_D(w) = R_{emp}(w) + \lambda\|w\|^2$$

即使 $R_{emp}$ 只是凸的（而非强凸的），$F_D$ 也是 $2\lambda$-强凸的。这就是为什么**正则化带来稳定性**。

### 4.2 Lipschitz连续性的作用

在SVR的分析中，我们利用了$\epsilon$-不敏感损失的**Lipschitz连续性**：
$$|\ell_\epsilon(w, z) - \ell_\epsilon(w', z)| \leq L\|w - w'\|$$

这使得我们可以将**参数空间的差距**转化为**损失值的差距**：
$$|\ell(w_D, z) - \ell(w_{D'}, z)| \leq L\|w_D - w_{D'}\|$$

**比较平方损失和$\epsilon$-不敏感损失**：

| 损失函数 | 局部Lipschitz常数 | 稳定性系数依赖 |
|---------|-----------------|--------------|
| 平方损失 | $2|w^\top x - y| \cdot \|x\|$ | 依赖于损失界 $M$ |
| $\epsilon$-不敏感 | $\|x\|$ | 不依赖于 $M$ |

这说明$\epsilon$-不敏感损失在大误差区域更"稳健"，不会导致稳定性系数随误差增大而恶化。

### 5.1 正则化作为"稳定锚"

**类比：船与锚**

想象经验风险 $R_{emp}(w)$ 是一艘船，正则化项 $\lambda\|w\|^2$ 是锚：

- **无正则化**：船随风浪（数据扰动）自由漂移，位置不稳定
- **有正则化**：锚将船固定在原点附近，风浪只能造成小幅摆动

锚越重（$\lambda$ 越大），船越稳定，但活动范围也越小（偏差增大）。这就是**稳定性-偏差权衡**。

### 5.2 k-NN的稳定性直觉

**为什么k-NN具有假设稳定性而非均匀稳定性？**

**均匀稳定性**要求：对于**所有**可能的数据集，替换一个样本都不会显著改变预测。

**反例**：考虑一个数据集，其中所有样本都位于同一点。此时k-NN的预测完全由这些样本的标签决定，替换其中一个样本可能完全改变预测。

**假设稳定性**只要求：对于**随机抽取**的数据集，替换一个样本不会显著改变预测。

在随机数据集中，样本不太可能都聚集在同一点，因此单个样本的影响有限。

### 5.3 三种算法的稳定性对比

| 算法 | 正则化 | 稳定性类型 | 稳定性系数 | 关键机制 |
|-----|-------|----------|----------|---------|
| 岭回归 | L2 | 均匀 | $O(M/m)$ | 强凸性+有界损失 |
| SVR | L2 | 均匀 | $O(1/m)$ | 强凸性+Lipschitz损失 |
| k-NN | 无 | 假设 | $O(\sqrt{k}/m)$ | 概率分析+投票机制 |

**观察**：
1. 正则化算法通常具有均匀稳定性
2. 无正则化算法可能只有假设稳定性
3. 稳定性系数都与 $1/m$ 成正比，保证样本量增加时稳定性增强

## 6. 一句话核心要义

> **正则化稳定，稳定泛化**：正则化通过引入强凸性使学习算法对训练数据的扰动不敏感（稳定），而稳定性保证了算法的泛化性能——无论假设空间多么复杂，只要算法是稳定的，就能给出有意义的泛化误差界。

### 7.1 核心知识点回顾

1. **岭回归**：
   - 稳定性系数：$\beta = \frac{4r^2M}{m\lambda}$
   - 关键：平方损失的Lipschitz常数依赖于损失值

2. **支持向量回归**：
   - 稳定性系数：$\beta = \frac{2r^2}{m\lambda}$
   - 关键：$\epsilon$-不敏感损失具有固定的Lipschitz常数

3. **k-近邻**：
   - 稳定性系数：$\beta = \frac{2\sqrt{2k}}{m\sqrt{\pi}}$
   - 关键：只有假设稳定性，通过概率分析得到

### 7.2 方法论总结

**稳定性分析的标准流程**：

```
定义扰动数据集 D' ──▶ 比较最优解 w_D 和 w_D'
        │                      │
        │                      ▼
        │         利用强凸性控制 ||w_D - w_D'||
        │                      │
        ▼                      ▼
   利用Lipschitz性质    代入稳定性定义
   控制损失差距         得到β
        │                      │
        └──────────┬───────────┘
                   ▼
           应用定理5.1或5.2
           得到泛化界
```
