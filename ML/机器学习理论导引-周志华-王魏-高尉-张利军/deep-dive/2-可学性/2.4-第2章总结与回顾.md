### 本章在全书中的地位

第2章"可学性"是《机器学习理论导引》的理论基石。本章从最基本的概念出发，建立了PAC（Probably Approximately Correct）学习理论框架，为后续章节讨论复杂度、泛化界、稳定性等更深入的主题奠定了基础。

### 本章的核心问题

本章围绕一个根本性问题展开：**什么样的问题是可学习的？**

具体而言：
1. **可学习的形式化定义是什么？**
2. **需要多少样本才能保证学习效果？**
3. **计算效率如何影响可学性？**
4. **目标概念不在假设空间中时怎么办？**

### 3.1 误差度量体系

| 术语（Term） | 符号 | 定义 | 含义 |
|--------------|------|------|------|
| **泛化误差**（Generalization Error） | $E(h)$ | $P_{(\mathbf{x},y)\sim\mathcal{D}}(h(\mathbf{x}) \neq y)$ | 假设在新数据上的期望错误率 |
| **经验误差**（Empirical Error） | $\widehat{E}(h)$ | $\frac{1}{m}\sum_{i=1}^m \mathbb{I}(h(\mathbf{x}_i) \neq y_i)$ | 假设在训练集上的错误率 |
| **关键性质** | | $\mathbb{E}_{D\sim\mathcal{D}^m}[\widehat{E}(h)] = E(h)$ | 经验误差的期望等于泛化误差 |

### 3.2 PAC学习核心定义

#### 定义2.1：PAC辨识（PAC Identify）

对 $0 < \epsilon, \delta < 1$，所有 $c \in \mathcal{C}$ 和分布 $\mathcal{D}$，若存在学习算法 $\mathfrak{L}$，其输出假设 $h \in \mathcal{H}$ 满足：

$$P(E(h) \leq \epsilon) \geq 1 - \delta$$

则称**学习算法 $\mathfrak{L}$ 能从假设空间 $\mathcal{H}$ 中PAC辨识概念类 $\mathcal{C}$**。

**核心解读**：
- **Probably**：以至少 $1-\delta$ 的概率
- **Approximately Correct**：误差不超过 $\epsilon$

#### 定义2.2：PAC可学（PAC Learnable）

若存在学习算法 $\mathfrak{L}$ 和多项式函数 $\mathrm{poly}(\cdot, \cdot, \cdot, \cdot)$，使得对于任何：

$$m \geq \mathrm{poly}(1/\epsilon, 1/\delta, \mathrm{size}(\mathbf{x}), \mathrm{size}(c))$$

$\mathfrak{L}$ 能从假设空间 $\mathcal{H}$ 中PAC辨识概念类 $\mathcal{C}$，则称**概念类 $\mathcal{C}$ 对假设空间 $\mathcal{H}$ 而言是PAC可学的**。

#### 定义2.3：PAC学习算法（PAC Learning Algorithm）

若学习算法 $\mathfrak{L}$ 使概念类 $\mathcal{C}$ 为PAC可学，且$\mathfrak{L}$的运行时间也是多项式函数，则称**概念类 $\mathcal{C}$ 是高效PAC可学的**。

#### 定义2.5：不可知PAC可学（Agnostic PAC Learnable）

令 $h^* = \arg\min_{h' \in \mathcal{H}} E(h')$ 为假设空间中最优假设。若学习算法 $\mathfrak{L}$ 输出的假设 $h$ 满足：

$$P\left(E(h) - \min_{h' \in \mathcal{H}} E(h') \leq \epsilon\right) \geq 1 - \delta$$

则称假设空间 $\mathcal{H}$ 是**不可知PAC可学的**。

### 3.3 易混淆概念辨析

#### 辨析1：PAC辨识 vs PAC可学

| 维度 | PAC辨识 | PAC可学 |
|------|---------|---------|
| **关注点** | 单次学习的效果保证 | 样本效率（多项式样本复杂度） |
| **核心问题** | 给定 $m$ 个样本，能否以高概率学得低误差假设？ | 需要多少样本才能保证PAC辨识？ |
| **关系** | 基础概念 | 在PAC辨识基础上增加样本效率要求 |

**类比**：
- PAC辨识就像问"这个学生能学会吗？"
- PAC可学就像问"这个学生需要学多久才能学会？"

#### 辨析2：恰PAC可学 vs 不可知PAC可学

| 维度 | 恰PAC可学 | 不可知PAC可学 |
|------|-----------|---------------|
| **假设空间** | $\mathcal{H} = \mathcal{C}$ | $\mathcal{H}$ 任意 |
| **目标概念** | $c \in \mathcal{H}$（可分） | $c \notin \mathcal{H}$ 可能（不可分） |
| **学习目标** | $E(h) \leq \epsilon$ | $E(h) - E(h^*) \leq \epsilon$ |
| **现实性** | 较理想化 | 更贴近实际 |

**关键洞察**：
- 恰PAC可学假设学习算法"恰好知道"概念类是什么
- 不可知PAC可学承认我们对真实概念一无所知，只能追求假设空间中的最优

#### 辨析3：可分 vs 不可分

| 维度 | 可分（Separable） | 不可分（Non-separable） |
|------|-------------------|-------------------------|
| **定义** | $c \in \mathcal{H}$ | $c \notin \mathcal{H}$ |
| **存在完美假设** | 是 | 否 |
| **最优泛化误差** | 0 | $\min_{h \in \mathcal{H}} E(h) > 0$ |
| **适用理论** | 标准PAC学习 | 不可知PAC学习 |

**现实情况**：
- 理论上，可分情形更简单
- 实践中，不可分情形更常见（我们几乎不可能知道真实的概念类）

### 4.1 核心定理回顾

#### 定理2.1：经验误差与泛化误差的关系

若训练集 $D$ 包含 $m$ 个从分布 $\mathcal{D}$ 上独立同分布采样而得的样本，则对任意 $h \in \mathcal{H}$，以至少 $1-\delta$ 的概率成立：

$$\widehat{E}(h) - \sqrt{\frac{1}{2m}\ln\frac{2}{\delta}} < E(h) < \widehat{E}(h) + \sqrt{\frac{1}{2m}\ln\frac{2}{\delta}}$$

**证明思路**：
1. 基于Hoeffding不等式（引理2.1）
2. 令 $\delta = 2\exp(-2m\epsilon^2)$，解出 $\epsilon = \sqrt{\frac{1}{2m}\ln\frac{2}{\delta}}$
3. 代入即得结论

**核心意义**：样本数目 $m$ 较大时，经验误差可以作为泛化误差的很好近似。

### 4.2 样本复杂度公式汇总

| 概念类 | 假设空间类型 | 样本复杂度 | 备注 |
|--------|--------------|------------|------|
| 一般有限假设空间 | 有限 | $m \geq \frac{1}{\epsilon}(\ln|\mathcal{H}| + \ln\frac{1}{\delta})$ | 通用上界 |
| 布尔合取式 | 有限 | $m \geq \frac{2n}{\epsilon}\ln\frac{2n}{\delta}$ | $n$为变量数 |
| 轴平行矩形 | 无限 | $m \geq \frac{4}{\epsilon}\ln\frac{4}{\delta}$ | 与维度无关 |
| 3-CNF | 有限 | 多项式样本复杂度 | 通过变量替换转化 |
| 3-DNF | 有限 | 非高效PAC可学 | 除非$\mathcal{RP}=\mathcal{NP}$ |

### 4.3 证明技巧总结

**技巧1：联合界（Union Bound）**

对于任意事件 $A_1, A_2, \ldots, A_n$：

$$P\left(\bigcup_{i=1}^n A_i\right) \leq \sum_{i=1}^n P(A_i)$$

**应用场景**：控制多个"坏"事件中至少一个发生的概率。

**技巧2：指数不等式**

$$(1-x)^m \leq e^{-mx}, \quad \text{对于 } x \in (0,1)$$

**应用场景**：简化概率计算，便于求解样本复杂度。

**技巧3：坏元素分析**

1. 定义对误差贡献大的"坏"元素
2. 证明无坏元素时误差有界
3. 用联合界控制所有坏元素被消除的概率

**应用场景**：布尔合取式的坏字分析。

**技巧4：概率质量划分**

1. 将误差区域划分为等概率的小区域
2. 证明若假设与所有区域相交则误差有界
3. 用联合界控制假设避开某个区域的概率

**应用场景**：轴平行矩形的分析。

### 示例 2.8：PAC学习的完整流程

**场景**：学习识别"苹果"

**问题设定**：
- 样本空间：所有水果的图片
- 概念类：所有"是苹果"的概念
- 假设空间：由颜色、形状、大小等特征定义的决策规则集合

**PAC学习过程**：

1. **采样**：从水果分布中随机抽取 $m$ 个样本（带标签）
2. **学习**：算法从假设空间中选择与训练集一致的规则
3. **输出**：得到一个假设 $h$
4. **保证**：以至少 $1-\delta$ 的概率，$h$ 的错误率不超过 $\epsilon$

**参数设置**：
- 希望错误率 $\epsilon \leq 5\%$
- 希望成功率 $1-\delta \geq 95\%$
- 假设空间大小 $|\mathcal{H}| = 10^6$

**计算样本量**：

$$m \geq \frac{1}{0.05}(\ln(10^6) + \ln(20)) = 20 \times (13.8 + 3.0) \approx 336$$

只需要约336个样本！

### 示例 2.9：可分 vs 不可分的类比

**类比：寻找完美的另一半**

| 机器学习 | 寻找伴侣类比 |
|----------|--------------|
| 可分情形 | 你心中有一个明确的"理想型"，且这个理想型在你的交友圈（假设空间）中存在 |
| 不可分情形 | 你的"理想型"过于理想化，交友圈中不存在完美匹配，只能找最接近的 |
| 恰PAC可学 | 坚持只考虑符合"理想型"标准的人 |
| 不可知PAC可学 | 接受现实，在交友圈中找到最合适的人 |

### 示例 2.10：样本复杂度的直观理解

**类比：考试复习**

| PAC学习 | 考试复习类比 |
|----------|--------------|
| 训练样本 | 练习题 |
| 假设空间 | 可能的解题方法 |
| 泛化误差 | 正式考试的错误率 |
| 经验误差 | 练习题的错误率 |
| 样本复杂度 | 需要做多少练习题才能考好 |
| $\epsilon$ | 可接受的考试错误率 |
| $\delta$ | 考好的把握 |

**洞察**：
- 假设空间越大（解题方法越多），需要做的练习题越多（但仅对数增长）
- 精度要求越高（错误率要求越低），需要做的练习题越多（线性增长）

## 6. 一句话核心要义

**PAC学习理论为机器学习提供了严格的数学基础：只要样本量与假设空间复杂度的对数、精度要求的倒数、置信度要求的对数呈多项式关系，就能以高概率学到近似最优的假设；而当目标概念不在假设空间中时，不可知PAC学习框架仍能保证逼近假设空间中的最优解。**

### 核心知识点串联

**从基本概念到PAC理论**：

1. **误差度量**：泛化误差 vs 经验误差 → 经验误差是泛化误差的无偏估计
2. **学习框架**：概念类 vs 假设空间 → 学习算法在假设空间中搜索目标概念
3. **可分性**：可分情形下最优误差为0，不可分情形下追求逼近最优
4. **PAC理论**：概率近似正确 → 以高概率学到近似最优假设
5. **样本复杂度**：多项式样本复杂度保证学习效率
6. **实例分析**：从有限到无限假设空间，从简单到复杂概念类

### 核心公式汇总

| 公式 | 名称 | 含义 |
|------|------|------|
| $E(h) = P(h(\mathbf{x}) \neq y)$ | 泛化误差 | 假设的期望错误率 |
| $\widehat{E}(h) = \frac{1}{m}\sum_{i=1}^m \mathbb{I}(h(\mathbf{x}_i) \neq y_i)$ | 经验误差 | 假设在训练集上的错误率 |
| $P(E(h) \leq \epsilon) \geq 1-\delta$ | PAC辨识条件 | 以高概率达到低误差 |
| $m \geq \frac{1}{\epsilon}(\ln|\mathcal{H}| + \ln\frac{1}{\delta})$ | 有限假设空间样本复杂度 | 通用上界 |
| $\widehat{E}(h) - \sqrt{\frac{1}{2m}\ln\frac{2}{\delta}} < E(h) < \widehat{E}(h) + \sqrt{\frac{1}{2m}\ln\frac{2}{\delta}}$ | 经验-泛化误差界 | 定理2.1 |
| $P(E(h) - \min_{h'}E(h') \leq \epsilon) \geq 1-\delta$ | 不可知PAC条件 | 逼近最优假设 |

### 常见误区总结

| 误区 | 正确理解 |
|------|----------|
| "PAC保证学到最优假设" | PAC保证学到"近似"最优（误差$\leq\epsilon$），而非精确最优 |
| "PAC是确定性保证" | PAC是概率性保证，有$\delta$的失败概率 |
| "可分假设总是成立" | 不可分情形更常见，不可知学习更实用 |
| "样本越多越好" | PAC给出充分条件，过多样本可能造成浪费；关键是样本效率 |
| "假设空间越大越好" | 大假设空间需要更多样本，存在偏差-方差权衡 |
| "PAC只适用于分类" | PAC框架可推广到回归、聚类等其他学习问题 |

### 延伸思考

1. **分布无关性的代价与收益**：
   PAC理论要求对**任意**分布都成立，这是"最坏情况分析"。这种保守性保证了理论的普适性，但也可能导致样本复杂度估计过于悲观。如果我们知道数据分布的某些性质（如高斯分布），能否得到更紧的界？这引出了**分布依赖的泛化界**研究。

2. **主动学习的样本效率**：
   本章假设训练样本是被动接收的（i.i.d.采样）。如果学习算法可以主动选择要标注的样本，能否显著降低样本复杂度？这引出**主动学习（Active Learning）**领域，在某些情况下可以将样本复杂度从$O(1/\epsilon)$降低到$O(\ln(1/\epsilon))$。

3. **在线学习与PAC学习的联系**：
   本章讨论的是批量学习（Batch Learning），即一次性获得所有训练样本。在**在线学习（Online Learning）**设置中，样本逐个到达，学习算法需要实时更新。这两种框架有何联系？第8章将讨论在线学习与遗憾界。

4. **表示学习的理论基础**：
   3-DNF与3-CNF的例子揭示了表示形式对可学性的关键影响。这启发了**表示学习（Representation Learning）**的研究：如何自动学习数据的良好表示，使下游学习任务变得更简单？这是深度学习成功的关键之一。
