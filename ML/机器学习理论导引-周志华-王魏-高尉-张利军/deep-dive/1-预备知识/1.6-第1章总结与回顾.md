### 2.1 函数的性质（1.1节）

**核心概念**：

| 概念 | 定义/条件 | 几何意义 |
|------|-----------|----------|
| 凸集 | $\theta \boldsymbol{x} + (1-\theta)\boldsymbol{z} \in C$ | 任意两点连线在集合内 |
| 凸函数 | $f(\theta \boldsymbol{x} + (1-\theta)\boldsymbol{z}) \leqslant \theta f(\boldsymbol{x}) + (1-\theta)f(\boldsymbol{z})$ | 函数在弦下方 |
| 一阶条件 | $f(\boldsymbol{z}) \geqslant f(\boldsymbol{x}) + \nabla f(\boldsymbol{x})^{\mathrm{T}}(\boldsymbol{z}-\boldsymbol{x})$ | 切线在下 |
| 二阶条件 | $\nabla^2 f(\boldsymbol{x}) \succeq 0$ | 函数"向上弯曲" |
| 强凸性 | 加二次间隙 $\frac{\lambda}{2}\|\boldsymbol{z}-\boldsymbol{x}\|^2$ | 保证唯一最小值 |
| Lipschitz连续 | $|f(\boldsymbol{z}) - f(\boldsymbol{x})| \leqslant l\|\boldsymbol{z}-\boldsymbol{x}\|$ | 变化率有界 |
| 光滑性 | 梯度Lipschitz连续 | 曲率有界 |

**关键定理**：
- 凸函数一阶条件：可微函数凸当且仅当切线在下
- 凸函数二阶条件：二阶可微函数凸当且仅当Hessian半正定

### 2.2 重要不等式（1.2节）

**不等式谱系**：

```
Markov不等式（基础）
    │
    ├──→ Chebyshev不等式（方差信息）
    │
    ├──→ Chernoff界（矩母函数）
    │       │
    │       ├──→ Hoeffding不等式（有界变量和）
    │       │       └──→ 样本复杂度：m = O(ln(1/δ)/ε²)
    │       │
    │       └──→ Bernstein不等式（方差+有界）
    │
    └──→ McDiarmid不等式（有界差分函数）
            └──→ 稳定性分析的基础
```

**重点掌握**：

**Hoeffding不等式**：
- 概率形式：$P(\bar{X} - \mathbb{E}[\bar{X}] \geqslant \epsilon) \leqslant e^{-2m\epsilon^2}$
- 置信形式：以至少$1-\delta$概率，$\bar{X} \leqslant \mathbb{E}[\bar{X}] + \sqrt{\frac{\ln(1/\delta)}{2m}}$

**McDiarmid不等式**：
- 适用于满足有界差分条件的任意函数
- 是Hoeffding的推广，稳定性分析的核心工具

### 2.3 最优化基础（1.3节）

**核心框架**：

```
原问题（Primal）                    对偶问题（Dual）
min f(x)                           max Γ(λ, μ)
s.t. h_i(x) ≤ 0                    s.t. λ ≥ 0
     g_j(x) = 0
        │                                │
        └──────── 弱对偶性：d* ≤ p* ───────┘
        │
        └──────── 强对偶性：d* = p*（凸+Slater条件）
```

**KKT条件**（强对偶性成立时）：

1. **原始可行性**：$h_i(\boldsymbol{x}^*) \leqslant 0$，$g_j(\boldsymbol{x}^*) = 0$
2. **对偶可行性**：$\boldsymbol{\lambda}^* \succeq 0$
3. **互补松弛**：$\lambda_i^* h_i(\boldsymbol{x}^*) = 0$
4. **梯度为零**：$\nabla f(\boldsymbol{x}^*) + \sum \lambda_i^* \nabla h_i(\boldsymbol{x}^*) + \sum \mu_j^* \nabla g_j(\boldsymbol{x}^*) = 0$

**关键理解**：
- 弱对偶性总是成立
- 凸优化+Slater条件 $\Rightarrow$ 强对偶性
- KKT条件对凸优化是充要条件

### 2.4 支持向量机（1.4节）

**SVM理论框架**：

```
硬间隔SVM                    核技巧扩展                    软间隔SVM
    │                            │                            │
    ▼                            ▼                            ▼
max 2/||w||                   特征映射φ(x)                 引入松弛变量ξ
s.t. y_i(w^T x_i + b) ≥ 1     κ(x_i, x_j) = φ(x_i)^T φ(x_j)  s.t. y_i(w^T φ(x_i) + b) ≥ 1 - ξ_i
    │                            │                            │
    └──────────────┬─────────────┴─────────────┬──────────────┘
                   │                           │
                   ▼                           ▼
              对偶问题（二次规划）          Hinge损失
              min_α 1/2 ΣΣ α_i α_j y_i y_j κ(x_i, x_j) - Σ α_i
              s.t. Σ α_i y_i = 0, 0 ≤ α_i ≤ β
```

**核心要点**：
- **最大间隔原则**：几何直观与理论保证的统一
- **对偶问题**：内积形式便于引入核函数
- **支持向量**：稀疏性来源于互补松弛条件
- **核技巧**：隐式高维映射，避免维度灾难
- **软间隔**：Hinge损失处理噪声和不可分情况

### 2.5 理论的作用（1.5节）

**三方面作用**：

1. **可行性判定**：某件事能做吗？
   - PAC学习框架
   - VC维与可学性

2. **关键因素识别**：什么因素重要？
   - 样本量$m$与复杂度$d$的权衡
   - 泛化界$O(\sqrt{d/m})$的指导意义

3. **改进空间评估**：能否做得更好？
   - 收敛率比较
   - 最优性证明

**理论界的解读**：
- 定性关系重于定量数值
- 紧致界揭示本质难度
- 概率解释：以高概率保证性能

4. **形成理论视角**：学习理论如何指导算法设计和分析

带着这些基础进入第2章，你将开始探索机器学习理论的核心问题：**可学性**。
