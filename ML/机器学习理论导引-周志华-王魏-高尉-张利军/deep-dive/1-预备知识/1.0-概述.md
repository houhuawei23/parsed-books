## 一、本章定位

《机器学习理论导引》全书共8章，第1章"预备知识"是后续7章的数学基石。本章不追求理论的深度拓展，而是系统梳理机器学习理论研究中最常用的数学工具，为读者构建一个扎实的知识底座。

你可以将本章理解为进入机器学习理论殿堂的"工具箱"——后续章节中反复出现的凸分析、概率不等式、优化理论，都在这里预先备好。

## 二、学习目标

完成本章学习后，你应当能够：

1. **判断函数的凸性**：运用定义、一阶条件、二阶条件判断给定函数是否为凸函数
2. **理解光滑性与强凸性**：明确Lipschitz连续、光滑函数、强凸函数的几何意义及相互关系
3. **熟练运用集中不等式**：特别是Hoeffding不等式和McDiarmid不等式的应用场景与推导逻辑
4. **掌握对偶理论与KKT条件**：理解拉格朗日对偶的构造方法，以及KKT条件在最优性刻画中的作用
5. **建立SVM的理论框架**：理解最大间隔、核方法、软间隔的数学本质
6. **认识学习理论的价值**：明确理论研究在机器学习中的三重作用

### 1.1 函数的性质 → 1.3 最优化基础
凸函数的定义与性质是理解凸优化的前提。没有凸性概念，就无法理解为什么凸优化问题"容易"求解。

### 1.2 重要不等式 → 全书各章
集中不等式（Concentration Inequalities）是机器学习理论的核心武器。从第2章的PAC学习框架到第4章的泛化界推导，Hoeffding不等式和McDiarmid不等式无处不在。

### 1.3 最优化基础 → 1.4 支持向量机
拉格朗日对偶和KKT条件是理解SVM对偶问题的钥匙。最大间隔分类器的推导完全建立在这些优化工具之上。

### 1.4 支持向量机 → 全书分析实例
SVM是本书后续章节反复使用的分析对象。掌握其理论框架，有助于理解各种学习理论概念的具体含义。

### 1.5 理论的作用 → 建立学习动机
这一节回答"为什么要学理论"的问题，为全书学习建立正确的认知框架。

### 凸分析部分

| 名称 | 公式 | 编号 |
|------|------|------|
| 凸集定义 | $\theta \boldsymbol{x}_1 + (1-\theta)\boldsymbol{x}_2 \in C, \forall \theta \in [0,1]$ | (1.1) |
| 凸函数定义 | $f(\theta \boldsymbol{x} + (1-\theta)\boldsymbol{z}) \leqslant \theta f(\boldsymbol{x}) + (1-\theta)f(\boldsymbol{z})$ | (1.2) |
| 一阶条件 | $f(\boldsymbol{z}) \geqslant f(\boldsymbol{x}) + \nabla f(\boldsymbol{x})^{\mathrm{T}}(\boldsymbol{z}-\boldsymbol{x})$ | (1.4) |
| 二阶条件 | $\nabla^2 f(\boldsymbol{x}) \succeq 0$ | - |
| 强凸函数 | $f(\boldsymbol{z}) \geqslant f(\boldsymbol{x}) + \nabla f(\boldsymbol{x})^{\mathrm{T}}(\boldsymbol{z}-\boldsymbol{x}) + \frac{\lambda}{2}\mid\boldsymbol{z}-\boldsymbol{x}\mid^2$ | (1.6) |
| Lipschitz连续 | $\mid f(\boldsymbol{z}) - f(\boldsymbol{x})\mid \leqslant l\mid\boldsymbol{z}-\boldsymbol{x}\mid$ | (1.7) |

### 重要不等式部分

| 名称 | 公式 | 编号 | 应用场景 |
|------|------|------|----------|
| Jensen不等式 | $f(\mathbb{E}[X]) \leqslant \mathbb{E}[f(X)]$ | (1.11) | 期望的函数 vs 函数的期望 |
| Hoeffding不等式 | $P(\bar{X} - \mathbb{E}[\bar{X}] \geqslant \epsilon) \leqslant e^{-2m\epsilon^2}$ | (1.28) | 有界随机变量和 |
| McDiarmid不等式 | $P(f - \mathbb{E}[f] \geqslant \epsilon) \leqslant e^{-2\epsilon^2/\sum c_i^2}$ | (1.32) | 有界差分函数 |
| Bernstein不等式 | $P(\bar{X} \geqslant \mathbb{E}[\bar{X}] + \epsilon) \leqslant \exp\left(\frac{-m\epsilon^2}{2\mathbb{V}[X_1] + 2b\epsilon}\right)$ | (1.37) | 方差信息可用时 |

### 优化理论部分

| 名称 | 公式 | 编号 |
|------|------|------|
| 拉格朗日函数 | $L(\boldsymbol{x}, \boldsymbol{\lambda}, \boldsymbol{\mu}) = f(\boldsymbol{x}) + \sum \lambda_i h_i(\boldsymbol{x}) + \sum \mu_j g_j(\boldsymbol{x})$ | (1.46) |
| 弱对偶性 | $d^* \leqslant p^*$ | (1.52) |
| 强对偶性 | $d^* = p^*$ | (1.53) |
| KKT条件 | 互补松弛 + 梯度为零 + 原/对偶可行 | - |

### SVM部分

| 名称 | 公式 | 编号 |
|------|------|------|
| 最大间隔 | $\min_{\boldsymbol{w},b} \frac{1}{2}\|\boldsymbol{w}\|^2$ s.t. $y_i(\boldsymbol{w}^{\mathrm{T}}\boldsymbol{x}_i + b) \geqslant 1$ | (1.59) |
| 对偶问题 | $\min_{\boldsymbol{\alpha}} \frac{1}{2}\sum\sum \alpha_i\alpha_j y_i y_j \kappa(\boldsymbol{x}_i, \boldsymbol{x}_j) - \sum \alpha_i$ | (1.68) |
| Hinge损失 | $\ell_{\text{hinge}}(x) = \max(0, 1-x)$ | (1.72) |

### 重点内容（必须掌握）

1. **凸函数的判断方法**
   - 定义法：验证不等式(1.2)
   - 一阶条件：验证不等式(1.4)
   - 二阶条件：验证Hessian矩阵半正定

2. **Hoeffding不等式的两种形式**
   - 概率形式：$P(\bar{X} - \mathbb{E}[\bar{X}] \geqslant \epsilon) \leqslant e^{-2m\epsilon^2}$
   - 置信形式：以至少$1-\delta$概率，$\bar{X} \leqslant \mathbb{E}[\bar{X}] + \sqrt{\frac{1}{2m}\ln\frac{1}{\delta}}$

3. **KKT条件的四个组成部分**
   - 原始可行性
   - 对偶可行性
   - 互补松弛条件
   - 梯度为零条件

4. **SVM的核技巧**
   - 理解为什么可以隐式地在高维空间计算
   - 掌握Mercer定理的判断标准

### 难点内容（需要反复思考）

1. **共轭函数的几何意义**
   - 初学者常困惑于$f_*(\boldsymbol{z}) = \sup_{\boldsymbol{x}}(\boldsymbol{z}^{\mathrm{T}}\boldsymbol{x} - f(\boldsymbol{x}))$的定义
   - 关键理解：线性函数$\boldsymbol{z}^{\mathrm{T}}\boldsymbol{x}$与原函数$f(\boldsymbol{x})$的最大差距

2. **强对偶性的成立条件**
   - Slater条件的技术细节
   - 为什么凸优化+严格可行点能保证强对偶性

3. **McDiarmid不等式的有界差分条件**
   - $|f(x_1,\ldots,x_i,\ldots,x_m) - f(x_1,\ldots,x_i',\ldots,x_m)| \leqslant c_i$
   - 理解"改变一个变量对函数值影响有限"的含义

4. **核函数与特征空间的对应关系**
   - 同一个核函数可能对应不同的特征映射
   - 特征空间的维数可能无穷

### 常见理解误区

| 误区 | 正确理解 |
|------|----------|
| "凸函数就是开口向上的函数" | 凸函数的定义依赖于定义域的凸性，且高维凸函数的形状远比"开口向上"复杂 |
| "Hoeffding不等式要求随机变量独立同分布" | Hoeffding不等式只要求独立性，不要求同分布 |
| "对偶问题的最优值总是等于原问题" | 只有强对偶性成立时才相等，一般情况下只有弱对偶性 |
| "SVM的核函数就是特征映射" | 核函数是特征映射的内积，不是映射本身 |

### 阅读顺序

1. **第一遍**：快速通读，建立整体框架，不必纠结于证明细节
2. **第二遍**：精读每个定理，动手推导关键证明（如凸函数一阶条件）
3. **第三遍**：结合后续章节，理解这些工具的实际应用场景

### 配套练习

- 完成章末习题1.1-1.8，特别是1.1（凸性分析）、1.6（损失函数凸性）、1.7（对偶推导）
- 尝试用Python绘制凸函数图像，验证一阶条件的几何意义
- 编写代码模拟Hoeffding不等式，观察收敛行为

### 延伸阅读

- 凸优化深度理解：Boyd & Vandenberghe《Convex Optimization》
- 集中不等式专题：Boucheron et al. 《Concentration Inequalities》
- SVM完整理论：Schölkopf & Smola《Learning with Kernels》
